随机变量的数字特征

随机变量的分布函数能够完整描述其统计特性，但在实际问题中，分布函数往往不易直接获得；许多任务也并不需要掌握完整分布，而只需要把握随机变量某些关键而稳定的统计特征。用于刻画这些特征的量称为随机变量的数字特征。数字特征本身是数，不再是随机变量。就本讲的内容范围而言，核心对象是数学期望与方差；协方差、相关系数与矩属于同一主题中的后续内容。

1. 数学期望

1.1 数学期望的基本含义

数学期望刻画随机变量取值的平均水平，其本质是概率意义下的加权平均。对于随机变量，不能简单把“所有可能取值”作普通算术平均，而必须按各取值出现的概率或频率加权。

以大学生年龄为例，若某班 200 人中，20 岁 60 人，19 岁 110 人，18 岁、17 岁、16 岁各 10 人，则平均年龄不是把 20、19、18、17、16 五个数直接相加后除以 5，而应按人数占比加权：

20020×60+19×110+18×10+17×10+16×10​=19

也可写成

20×0.3+19×0.55+18×0.05+17×0.05+16×0.05=19

因此，数学期望反映的是随机变量取值在概率意义下的中心位置，而不是“不同取值的简单平均”。它不一定等于最可能出现的值，也不一定恰好是随机变量能够取到的值。

1.2 离散型随机变量的数学期望

设离散型随机变量 X 的分布律为

P(X=xk​)=pk​,k=1,2,…

若级数

k=1∑∞​∣xk​∣pk​

收敛，则称 X 的数学期望存在，并定义为

E(X)=k=1∑∞​xk​pk​

绝对收敛条件的作用在于保证该加权和有确定意义，不会因求和次序的变化而产生歧义。对有限取值的离散随机变量，这一条件自动满足。

打靶成绩的例子能够直接说明数学期望的比较意义。若甲、乙两人的命中环数分别记为 X 与 Y，则按照分布表计算可得

E(X)=0×0.05+5×0.05+6×0.05+7×0.15+8×0.3+9×0.4+10×0=7.6 E(Y)=0×0.25+5×0.2+6×0.2+7×0.05+8×0.05+9×0.05+10×0.2=5.4

因此，从平均命中环数看，甲的射击水平高于乙。数学期望给出的是长期平均水平，而不是单次结果。

1.3 常见离散分布的数学期望

1.3.1 0-1 分布

若 X∼Bernoulli(p)，即

P(X=1)=p,P(X=0)=1−p

则

E(X)=0⋅(1−p)+1⋅p=p

这表明 0-1 随机变量的期望等于“成功”事件发生的概率。

1.3.2 二项分布

若 X∼B(n,p)，记 q=1−p，则

P(X=k)=(kn​)pkqn−k,k=0,1,…,n

由定义

E(X)=k=0∑n​k(kn​)pkqn−k

利用恒等式

k(kn​)=n(k−1n−1​)

得

E(X)=npk=1∑n​(k−1n−1​)pk−1q(n−1)−(k−1)

令 i=k−1，则

E(X)=npi=0∑n−1​(in−1​)piq(n−1)−i=np

因此，二项分布的期望等于试验次数与单次成功概率的乘积。

1.3.3 泊松分布

若 X∼P(λ)，则

P(X=k)=e−λk!λk​,k=0,1,2,…

于是

E(X)=k=0∑∞​ke−λk!λk​

将 k=0 项去掉并整理得

E(X)=λk=1∑∞​e−λ(k−1)!λk−1​

令 i=k−1，则

E(X)=λi=0∑∞​e−λi!λi​=λ

因此，泊松分布的期望等于其参数 λ。

1.3.4 超几何分布

若 X∼H(N,M,n)，表示总体容量为 N、其中“成功类”个体数为 M、不放回抽取 n 次时成功个数的分布，则

E(X)=NnM​

这一结果表明，超几何分布的平均成功次数等于样本容量乘以总体中的成功比例。尽管抽样过程中各次抽取并不独立，其平均水平仍然服从这一线性比例关系。

1.3.5 几何分布

若 X 服从几何分布，取值为第一次成功出现的试验次数，且

P(X=k)=qk−1p,k=1,2,…,q=1−p

则

E(X)=k=1∑∞​kqk−1p=p1​

因此，成功概率越大，第一次成功所需等待的平均试验次数越少。

1.4 一般随机变量的数学期望

为了统一离散型与连续型情形，可先用离散近似来定义一般随机变量的期望。对任意随机变量 X，给定 ε>0，把数轴划分为长度为 ε 的小区间，并定义近似随机变量 Xε∗​ 为（总是取左端点）

Xε∗​=kε,X∈[kε,(k+1)ε)

则有

0≤X−Xε∗​<ε

当 ε 很小时，Xε∗​ 与 X 很接近。若 E(Xε∗​) 存在，且当 ε→0 时极限存在，则定义

E(X)=ε→0lim​E(Xε∗​)

这一构造表明，数学期望可以被理解为对随机变量进行越来越细的离散化之后，其加权平均的极限（类似于分箱机制）。

1.5 连续型随机变量的数学期望

设连续型随机变量 X 的概率密度为 f(x)。若广义积分

∫−∞∞​∣x∣f(x)dx

收敛，则称 X 的数学期望存在，并定义为

E(X)=∫−∞∞​xf(x)dx

这一公式可由一般随机变量的离散近似定义推出。对连续型随机变量而言，期望仍然是概率加权平均，只是求和形式被积分形式取代。

1.6 常见连续分布的数学期望

1.6.1 均匀分布

若 X∼U[a,b]，其密度为

f(x)=⎩⎨⎧​b−a1​,0,​a<x<b其他​

则

E(X)=∫ab​x⋅b−a1​dx=2a+b​

均匀分布的期望就是区间中点。

1.6.2 指数分布

若 X∼E(λ)，其密度为

f(x)={λe−λx,0,​x≥0x<0​

则

E(X)=∫0∞​xλe−λxdx

令 t=λx，得

E(X)=λ1​∫0∞​te−tdt=λ1​

因此，指数分布的平均等待时间等于 λ1​。

1.6.3 正态分布

若 X∼N(μ,σ2)，其密度为

f(x)=2π​σ1​e−2σ2(x−μ)2​,−∞<x<∞

则

E(X)=2π​σ1​∫−∞∞​xe−2σ2(x−μ)2​dx

令

u=σx−μ​

则 x=σu+μ，从而

E(X)=2π​1​∫−∞∞​(σu+μ)e−2u2​du

将积分拆开后，含 u 的部分是奇函数在对称区间上的积分，结果为 0；剩余部分等于 μ。因此

E(X)=μ

正态分布的期望就是其位置参数。

1.6.4 Gamma 分布

若 X∼Γ(α,β)，采用 rate 参数化，其密度为

f(x)=⎩⎨⎧​Γ(α)βα​xα−1e−βx,0,​x>0x≤0​

其中

Γ(α)=∫0∞​tα−1e−tdt

并满足递推关系

Γ(α+1)=αΓ(α)

于是

E(X)=∫0∞​x⋅Γ(α)βα​xα−1e−βxdx

令 t=βx，得

E(X)=βΓ(α)1​∫0∞​tαe−tdt

即

E(X)=βΓ(α)Γ(α+1)​=βα​

因此，Gamma 分布的期望等于形状参数与 rate 参数之比。

2. 随机变量函数的数学期望

2.1 基本思想

设 Y=g(X)。求 Y 的数学期望有两条思路。其一是先由 X 的分布推出 Y 的分布，再按期望定义计算；其二是直接利用均值公式，通过 X 的分布与函数关系 g 计算 E(Y)。对多维随机变量函数也有完全对应的处理方法。均值公式的意义在于，它把“先求函数变量的分布”这一中间步骤省去，从而大大简化计算。

2.2 一维随机变量函数的均值公式

若离散型随机变量 X 的分布为

P(X=xi​)=pi​,i=1,2,…

且级数

i=1∑∞​∣g(xi​)∣pi​

收敛，则

E[g(X)]=i=1∑∞​g(xi​)pi​

若连续型随机变量 X 的概率密度为 f(x)，且广义积分

∫−∞∞​∣g(x)∣f(x)dx

收敛，则

E[g(X)]=∫−∞∞​g(x)f(x)dx

这里的关键不是函数值本身是否有界，而是 g(X) 是否可积。

均值公式实质上说明：只要随机变量函数的期望存在，就可以直接对“函数值乘以原随机变量的概率结构”求和或积分。

例设 X∼U(0,π)，Y=sinX。由于

fX​(x)=⎩⎨⎧​π1​,0,​0<x<π其他​

于是

E(Y)=E(sinX)=∫0π​sinx⋅π1​dx=π2​

该结果说明，函数随机变量的期望未必需要先求出 Y=sinX 的密度函数。

2.3 二维与多维随机变量函数的均值公式

若离散型随机变量 (X,Y) 的联合分布为

P(X=xi​,Y=yj​)=pij​

且级数绝对收敛，则对 Z=g(X,Y) 有

E(Z)=i∑​j∑​g(xi​,yj​)pij​

若连续型随机变量 (X,Y) 的联合密度为 f(x,y)，且广义积分绝对收敛，则

E(Z)=∫−∞∞​∫−∞∞​g(x,y)f(x,y)dxdy

同理可推广到 n 维情形。若 Z=φ(X1​,…,Xn​)，则在可积条件下，

E(Z)=∫⋯∫φ(x1​,…,xn​)f(x1​,…,xn​)dx1​⋯dxn​

多维均值公式把函数的期望直接还原为对联合分布的加权平均。

例设 X∼U(0,1)，Y∼E(1)，且 X 与 Y 相互独立。则

fX​(x)={1,0,​0<x<1其他​fY​(y)={e−y,0,​y>0其他​

独立性给出联合密度

f(x,y)={e−y,0,​0<x<1, y>0其他​

于是

E(X+Y)=∫0∞​∫01​(x+y)e−ydxdy

先对 x 积分，得

E(X+Y)=∫0∞​(21​+y)e−ydy

从而

E(X+Y)=21​∫0∞​e−ydy+∫0∞​ye−ydy=21​+1=23​

这一计算与后续的线性性质相一致。事实上，和的期望具有更强的一般性，并不要求独立。

3. 数学期望的性质

3.1 常数与线性组合

设 a,ai​,C 为常数，则

E(C)=C E(aX)=aE(X) E(X+Y)=E(X)+E(Y)

进一步，

E(i=1∑n​ai​Xi​+C)=i=1∑n​ai​E(Xi​)+C

这说明线性变换或线性组合的期望，等于期望的同样线性变换或线性组合。和的期望不要求随机变量独立，这是期望最重要的结构性质之一。

3.2 独立随机变量乘积的期望

若 X 与 Y 独立，则

E(XY)=E(X)E(Y)

更一般地，若 X1​,…,Xn​ 相互独立，则

E(X1​⋯Xn​)=E(X1​)⋯E(Xn​)

其根本原因在于独立性使联合分布分解为边缘分布的乘积，因此关于乘积函数的联合积分或联合求和也随之分解。需要区分的是，E(X+Y)=E(X)+E(Y) 总成立，而 E(XY)=E(X)E(Y) 只在独立条件下成立；

==反过来，E(XY)=E(X)E(Y) 也不能推出 X 与 Y 必然独立==。

3.3 保序性

若存在常数 a 使

P(X≥a)=1

则

E(X)≥a

若存在常数 b 使

P(X≤b)=1

则

E(X)≤b

这一性质说明数学期望保留随机变量几乎处处成立的大小关系。期望位于随机变量取值整体分布所允许的范围之内，不会偏离几乎必然成立的上下界。

3.4 Schwarz 不等式

期望还满足

[E(XY)]2≤E(X2)E(Y2)

这是 Schwarz 不等式在随机变量情形下的表达。证明思路来自平方非负性：对任意实数 λ，

E[(λX+Y)2]≥0

展开得

λ2E(X2)+2λE(XY)+E(Y2)≥0

该二次式对任意 λ 都非负，因此判别式不大于零，即

[2E(XY)]2−4E(X2)E(Y2)≤0

从而得到所需不等式。它给出了混合项 E(XY) 的一个普遍上界，在后续的协方差与相关系数理论中具有基础作用。

3.5 期望线性性质的一个典型应用

二项分布期望还可以通过指标随机变量分解得到更有解释力的推导。设 X∼B(n,p)，将 X 理解为 n 次 Bernoulli 试验中事件 A 发生的次数。对第 i 次试验，定义

Xi​={1,0,​A发生A不发生​i=1,2,…,n

则

X=X1​+X2​+⋯+Xn​

且每个 Xi​ 都服从参数为 p 的 0-1 分布，因此

E(Xi​)=p

由线性性质立刻得到

E(X)=E(X1​)+E(X2​)+⋯+E(Xn​)=np

这一推导揭示了 np 的含义：二项分布的平均成功次数等于每次试验平均成功值 p 在 n 次试验上的累加。这里不需要先对 (kn​)pkqn−k 进行求和变形。

4. 方差

4.1 方差与标准差的定义

若

E[(X−E(X))2]

存在，则称其为随机变量 X 的方差，记为 D(X) 或 Var(X)，即

D(X)=E[(X−E(X))2]

标准差定义为

σ=D(X)​

方差描述随机变量取值偏离其平均水平的平均平方偏离程度，用来刻画随机变量取值的分散程度。方差越大，随机变量围绕均值的波动通常越强。标准差与原随机变量具有相同量纲，而方差的量纲是原变量量纲的平方。

4.2 离散型随机变量的方差

若离散型随机变量 X 的分布律为

P(X=xk​)=pk​,k=1,2,…

则在方差存在时，

D(X)=k=1∑∞​(xk​−E(X))2pk​

这一公式直接由均值公式得到。它说明离散型随机变量的方差仍然是对“偏离均值的平方”作概率加权平均。使用平方而不是绝对值，一方面保证非负性，另一方面使大偏差得到更强惩罚。

4.3 连续型随机变量的方差

若连续型随机变量 X 的概率密度为 f(x)，则在方差存在时，

D(X)=∫−∞∞​(x−E(X))2f(x)dx

这与离散型情形完全对应，只是由求和变为积分。连续型方差仍然是偏离均值平方的概率加权平均。

4.4 计算方差的常用公式

直接使用定义计算方差时，往往需要先求均值，再把平方偏差积分或求和，步骤较长。更常用的恒等式是

D(X)=E(X2)−[E(X)]2

其推导为

D(X)​=E[(X−E(X))2]=E[X2−2XE(X)+[E(X)]2]=E(X2)−2E(X)E(X)+[E(X)]2=E(X2)−[E(X)]2​

这一公式的价值在于，它把方差计算转化为两个期望的计算：二阶原点矩 E(X2) 与一阶矩 E(X)。在实际计算中，只要求出这两个量即可得到方差。该公式同时表明，方差存在至少需要二阶矩存在。
