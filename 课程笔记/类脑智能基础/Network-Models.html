Network Models

Network Models

Apr 25, 202613 min read

课程核心概述

本讲的核心探讨了大脑如何利用神经网络的复杂动态来实现计算。主要围绕两个极为经典的模型展开：

  1. 吸引子网络 (Attractor Networks)：解决大脑如何稳健地存储和提取信息（记忆）。它通过构建一个能量地貌（Energy Landscape），让记忆成为系统状态的“谷底”（吸引子）。

  2. 兴奋-抑制平衡网络 (E-I Balanced Networks)：解决大脑计算为什么能这么快以及为什么神经元在活体中表现出不规则放电。通过 E（兴奋）和 I（抑制）信号的抵消，让涨落（噪声）主导放电，从而打破了单神经元物理时间常数的限制，实现对刺激的瞬时响应。

神经电路模型：吸引子网络与平衡网络

第一部分：吸引子网络 (Attractor Networks) 与记忆存储

1. 直觉理解：什么是吸引子网络？

大脑极其复杂（约 1011 个神经元，每个有 103∼104 个连接）。连接主义（Connectionism）认为：单个神经元的计算很简单，丰富复杂的网络动态（Dynamics）才实现了高级的大脑功能。

直觉上的“记忆”

想象一张有很多坑坑洼洼的弹性床单。如果你往上面扔一个小球，小球最终会滚落到离它最近的某个坑底停下来。

在神经网络中，网络的“状态”就是那个小球，而设定的“坑底”就是吸引子（Attractors）。信息（记忆）被编码为这些稳定的定态（Stationary states）。当我们给网络一个不完整的提示（把小球放在坑的边缘），网络会自动演化到最近的稳定状态（滚入坑底），这就实现了稳健的记忆检索（内容寻址记忆）。

八卦插曲：2024年诺贝尔物理学奖颁给了 Hopfield 和 Hinton，表彰他们对人工神经网络的贡献。但实际上，Amari 在 1972 年（早 Hopfield 10年）就提出了完全相同的数学模型。由于 Hopfield 的影响力极大，后来大家都叫它 Hopfield 网络（或 Amari-Hopfield 模型）。

2. 核心数学表达与动态规则

网络由 N 个神经元组成，状态只有两种：激活或抑制。

  • 神经元状态：Si​=±1

  • 突触连接权重：wij​

  • 更新规则：神经元 i 会整合来自其他神经元的输入，如果总输入大于阈值 θ，则激活为 1，否则为 −1。

Si​(t+1)=sign(j∑​wij​Sj​(t)−θ)

这里 sign(x)=1（当 x>0 时），否则为 −1。

3. 为什么网络最终会停下来？（能量函数）

为了证明这个网络确实有“坑底”，Hopfield 引入了物理学中的能量函数 (Energy Function)：

E=−21​i,j∑​wij​Si​Sj​+θi∑​Si​

推导：为什么能量总是单调递减的？

假设我们在 t 时刻更新神经元 i：

Si​(t+1)=sign[∑j​wij​Sj​(t)−θ]

计算能量的变化量 ΔE=E(t+1)−E(t)：

ΔE=−[Si​(t+1)−Si​(t)]j∑​wij​Sj​(t)+θ[Si​(t+1)−Si​(t)] ΔE=−[Si​(t+1)−Si​(t)][j∑​wij​Sj​(t)−θ]

观察这个式子：

  • 如果 Si​ 从 −1 变到 1（即中括号第一项为正），说明 ∑j​wij​Sj​(t)−θ>0（中括号第二项也为正），乘积前加负号，ΔE≤0。

  • 反之亦然。

    结论：不管网络怎么演化，能量 E 永远不会增加。因为状态空间是有限的，它最终一定会收敛到能量的局部极小值（也就是我们的记忆模式）。

4. 记忆的存储容量与串扰

我们要如何设置权重 wij​，才能让特定的模式 ξ 成为坑底（吸引子）呢？

(1) 存储单一模式

我们模仿赫布法则（Hebb rule，同时激活则连接增强）：

wij​=N1​ξi​ξj​

这样设置后，网络只要处于 ξi​ 状态，算出来的信号就正好支撑它继续保持 ξi​，记忆非常稳定。

(2) 存储多个模式（容量危机）

如果我们要存储 p 个模式（记为 ξiμ​，μ=1,...,p）：

wij​=N1​μ=1∑p​ξiμ​ξjμ​

此时，验证某个目标记忆（比如第 ν 个模式）是否稳定时，输入信号 hiν​ 会变成：

hiν​=j∑​wij​ξjν​=ξiν​+N1​j∑​μ=ν∑​ξiμ​ξjμ​ξjν​

注意看后面那一大坨——这就是串扰项（Cross-talk term）。它代表了其他记忆模式对当前提取记忆的干扰。

(3) 记忆会算错的概率 (Perror​)

定义错误项 Ciν​=−ξiν​N1​∑j​∑μ=ν​ξiμ​ξjμ​ξjν​。

只有当串扰大到彻底掩盖了真实信号，也就是 Ciν​>1 时，网络才会回忆出错。

在 N（神经元数量）和 p（记忆模式数量）都很大时，Ciν​ 服从正态分布，其均值为 0，方差 σ2=p/N。

回忆错误率即为处于正态分布尾部的概率：

Perror​=2π​σ1​∫1∞​e−x2/2σ2dx

关键结论：网络能存多少记忆，取决于我们能容忍多大的错误率。如果要保证绝大多数回忆都是对的，最大容量大约是 pmax​≈0.138N。也就是说，1000 个神经元的网络，大约能记住 138 个正交模式。

第二部分：E-I 平衡网络 (E-I Balanced Networks) 与计算速度

1. 神经编码的矛盾：活体中的不规则放电

大脑是如何编码信息的？

  • 频率编码 (Rate coding)：用单位时间内的脉冲（Spikes）数量来传递信息。

  • 时间编码 (Temporal coding)：靠脉冲发生的精确时间点来传递信息（比如猫头鹰听声辨位，靠的是双耳接收声音 0.1 毫秒的时间差，这就需要神经元做“重合检测器”）。

这里出现了一个经典的悖论：

当我们在体外（in vitro）给单个神经元注入恒定电流时，它会非常规则地打出脉冲（符合 Hodgkin-Huxley 模型）。

但是，在活体大脑（in vivo）中，皮层神经元的放电是**极度不规则（Irregular）**的，表现出泊松过程（Poisson Process）的特征，即方差等于均值，Fano Factor ≈1。

中心极限定理悖论

一个皮层神经元会接收来自 1000 到 10000 个其他神经元的输入。根据统计学中的中心极限定理（Central Limit Theorem），大量独立随机信号叠加后，其相对波动应该趋近于零（也就是输入电流会变成一个平滑的恒定值）。

如果输入是平滑恒定的，神经元应该规则放电才对。为什么活体中实际观测到的却是极度不规则放电？

2. 破局机制：兴奋-抑制平衡 (E-I Balance)

Van Vreeswijk 和 Sompolinsky (1996) 提出了 E-I 平衡网络来解释这个现象。

网络中有两类神经元：兴奋性 (Excitatory, E) 和 抑制性 (Inhibitory, I)，它们之间稀疏随机连接。

单个神经元接收到的动态方程大致如下：

τdtduiE​​=−uiE​+∑JEE​rjE​−∑JEI​rjI​+IiE​

平衡的奥秘：

  • 网络会自发演化到一个状态，使得兴奋性输入的均值和抑制性输入的均值相互抵消。
KE​JEE​μ−KI​JEI​μ≈0

  • 均值虽然抵消了，但波动的方差并没有抵消，反而叠加了！

  • 结果就是：驱动神经元放电的不再是一股平稳的水流（均值），而是水面的剧烈浪花（方差/波动）。这种由输入涨落（Fluctuations）驱动的放电，自然就表现出了混沌和不规则性。

3. 为什么大脑要保持平衡与不规则？（核心意义）

答案是：为了极致的速度 (Speeding up computation)。

如果在一个没有噪声的环境里，神经元的膜电位从静息状态慢慢爬升到阈值，这个反应时间 T 受限于神经元细胞膜的物理属性——膜时间常数 τ（通常是 10-20 毫秒）。这意味着大脑反应会很慢。

噪声如何加速反应？

想象一条起跑线（发射阈值 θ）。

如果没有噪声，所有运动员（神经元膜电位）都在起点线后 100 米处慢慢走，听到枪响（外部刺激）后也要走很久才能过线。

在 E-I 平衡带来的理想噪声环境中，运动员的位置是被随机打散的，总有相当一部分人恰好就站在起跑线边缘。当哪怕极微小的刺激（发令枪）到来时，这批刚好在边缘的人瞬间就会越线放电！

  • 宏观表现：虽然单个神经元的反应受限于 τ，但由大量神经元组成的 E-I 平衡网络群体，可以对外部输入的剧烈变化做出近乎瞬时（无延迟）的响应，响应时间只受到神经元数量 M 的限制（误差随 M 线性下降）。

  • 线性编码：在这种状态下，网络整体的平均放电率可以与外部输入强度保持优美的线性关系。

4. 落地应用：脉冲相机 (Spike Camera)

借鉴了 E-I 平衡网络快速响应的特性，学者们开发了超高速脉冲相机 (Spike Camera)，可以达到 40,000 fps 的帧率，能清晰捕捉高速旋转的扇叶。

工作机制 (非泄漏积分放电模型)：

  1. 累加器 (Accumulator)：持续积分光电流 I(t)。

  2. 比较器 (Comparator)：当积分值 ∫I(t)dt>Φ（固定阈值）时，发放一个脉冲（Spike = 1）。

  3. 重置 (Reset)：放电后立刻清空积分电容上的电荷。

  4. 高频读取：每 25微秒 读取一次所有像素的状态。

通过在脉冲相机后续处理中引入局部 E-I 平衡网络层，既能维持局部连接带来的空间结构信息，又能利用平衡态特性实现对视觉信号的极速捕捉和重建。
