Theoretical Explanation of Learning Mechanism

The Rescorla–Wagner Model

Rescorla–Wagner 模型（简称 R-W 模型）由 Robert Rescorla 和 Allan Wagner 于 1972 年提出，是心理学和计算神经科学中最具影响力的经典条件反射模型之一。它第一次将 ” 学习 ” 这一抽象的心理过程用简洁的数学公式进行了量化，并提出了一个核心观点：学习取决于 ” 惊讶程度 “（Surprise）。

1. 核心思想：预测误差

R-W 模型认为，条件反射的形成并不是因为刺激（CS，如铃声）和结果（US，如食物）简单地同时出现，而是因为结果出乎意料。

如果动物已经能够完全预测到奖励的出现，那么即使奖励再次发生，学习也会停止。只有当预测与实际结果之间存在差异（即预测误差）时，大脑才会更新对刺激的认知。

2. 数学公式

模型用一个简单的线性差分方程来描述刺激关联强度的变化：

ΔVi​=αi​βj​(事实λj​​​−预期∑V​​)
  • ΔVi​：在当前试次中，刺激 i（如铃声）获得的关联强度（Associative Strength）的学习变化量。
  • αi​：刺激 i 的显著性（Salience），取决于刺激本身的特征（如铃声的大小）。
  • βj​：奖励（US）的学习速率，取决于奖励的性质（如食物的吸引力）。
  • λj​：该奖励所能提供的最大关联强度（即预测的目标值 -prediction）。
  • ∑V：当前环境中所有呈现刺激的关联强度总和（即大脑对奖励的总预期）。
  • (λj​−∑V)：预测误差（Prediction Error）。这是公式的核心，代表了实际结果与预期之间的差距。

更新规则

在标准的 Rescorla-Wagner 模型（以及大多数标准神经网络训练）中，同一个试次（Trial）内，所有刺激的 V 是 ” 同步更新 ” 的。

这意味着：你担心的 ” 先更新强刺激导致误差变小，从而导致弱刺激分不到更新量 ” 的情况，在同一个试次内部是不会发生的。但是，这种情况会在下一个试次发生。

3. 该模型解释的经典现象

阻塞效应 (Blocking Effect)

这是 R-W 模型最辉煌的成就。实验发现，如果先训练狗学会 ” 铃声 → 食物 “，等到狗已经完全掌握（∑V=λ）后，再同时呈现 ” 铃声 + 灯光 → 食物 “，狗将不会对 ” 灯光 ” 产生任何条件反射。

  • 解释：因为铃声已经预测了全部的奖励，此时预测误差 (λ−∑V)=0。因此 ΔV灯光​ 也为 0，学习被 ” 阻塞 ” 了。

阴影效应 (Overshadowing)

如果同时呈现两个不同显著性的刺激（如强光和弱音）指向同一个奖励，强刺激会占据大部分的关联强度，导致弱刺激的学习效果变差。

抑制性条件化（Conditioned Inhibition）

当一个刺激预示着原本预期的奖励不会出现时，它会获得负向的关联强度，在大脑中起到抵消奖励预期的 ” 安全信号 ” 或 ” 抑制信号 ” 作用。

过度预期效应（Overexpectation Effect）

当两个各自分别预示一份奖励的刺激同时出现时，大脑会产生双倍奖励的过高预期，若此时实际只给一份奖励，负向的预测误差会导致这两个刺激各自的关联强度都发生下降。

4. 逻辑流程图

我们可以通过这个反馈循环来理解学习是如何发生的：

5. 模型的局限性

虽然 R-W 模型非常强大，但它在以下几个方面存在显著不足：

  • 无法解释二级条件反射 (Second-order Conditioning)：模型将条件刺激 (CS) 与无条件刺激 (US) 视为性质完全不同的两类物体，无法处理刺激之间的预测传递。在现实中，如果刺激 A 预测刺激 B，而刺激 B 预测奖励，那么 A 也会获得奖励预测价值。这在人类社会中极其重要，例如金钱本身并没有生物学意义，但它是食物和住所等各种奖励的二阶预测因子。R-W 模型无法解释这种层级化的价值赋予。
  • 以 ” 试次 ” 为离散单位，缺乏时间敏感性：模型将学习过程简化为一个接一个离散的 ” 试次 “（Trial），这种划分具有实验者主导的主观性，忽视了现实中事件的连续性。它无法解释试次内部刺激出现的具体时空关系，例如刺激是同时出现还是先后出现、它们的出现顺序，以及刺激之间的时间滞后（Time Lag）对学习强度的具体影响。
  • 无法解释自发恢复 (Spontaneous Recovery)：它无法很好地解释为什么在消退训练（即只有刺激没有奖励）导致反应消失后，仅仅经过一段时间的休息，动物对该刺激的反应会再次出现。这暗示了消退并不是消除了原有的关联，而可能是一种新的抑制性学习。
  • 无法解释潜伏抑制 (Latent Inhibition)：模型预测对一个熟悉但无奖励刺激的学习速度应与新刺激一致。但在实际观察中，如果动物事先多次接触一个无关刺激，之后再将其与奖励关联时，学习速度会明显变慢。R-W 模型无法处理这种因 ” 熟悉感 ” 而导致的关联难度增加。

6. 从心理学到人工智能

R-W 模型是现代强化学习（Reinforcement Learning）的基石。在计算机科学中，它被进一步发展成了时序差分学习（TD Learning）。

  • R-W 模型中的误差 (λ−∑V) 就是深度学习中常见的 损失函数（Loss Function） 的雏形。
  • 学习率 α,β 对应着优化算法中的 Learning Rate。
  • 神经科学研究证明，中脑多巴胺神经元的放电频率完美地编码了这个数学公式中的 ” 预测误差 ” 信号。

如果你正在研究大型语言模型（LLM）的训练机制，你会发现 R-W 模型所代表的 ” 基于误差修正的学习 ” 与反向传播（Backpropagation）中权重的更新逻辑有着惊人的跨学科一致性。

Temporal Difference Learning

时序差分学习（Temporal Difference Learning, TD Learning），是对经典 Rescorla–Wagner（R-W）模型的重大演进。TD 模型不仅解决了 R-W 模型在时间和层级预测上的缺陷，还成为了现代人工智能与神经科学之间的核心桥梁。

1. 为什么要从 R-W 模型转向 TD 模型？

R-W 模型虽然经典，但存在两个致命的局限性，TD 学习正是为此而生：

  • 二级条件反射（Higher-order Conditioning）的缺失：在 R-W 模型中，预测的对象必须是无条件刺激（如食物）。但在现实中，如果刺激 A 预测刺激 B，而 B 预测食物，刺激 A 也会获得价值（比如金钱预测购买力，购买力预测食物）。TD 学习之所以能解释二阶条件反射（Second-order Conditioning），核心在于它改变了 ” 什么可以作为奖励 ” 的定义。在 TD 框架下，一个预示着未来奖励的信号（状态价值），其作用等同于奖励本身。
  • 时间维度的缺失：R-W 模型将一次实验（Trial）视为一个离散的点，不关心刺激出现的具体先后顺序。而 TD 模型在实验内部的每一个时间点都在进行计算，能够区分 ” 同时出现 ” 和 ” 先后出现 ” 的区别。

2. 核心数学框架：状态价值（State Value）

TD 学习的核心目标是估计一个状态的价值 V(St​)。

V(St​)=E[折现后的总回报 (Return)rt​+γrt+1​+γ2rt+2​+…​​∣St​] V(St​)=E[rt​+γrt+1​+γ2rt+2​+…∣St​]=E[i=t∑∞​γi−tri​∣St​]
  • 含义：当你处在状态 St​ 时，你期待的不仅仅是眼前的奖励 rt​，而是从现在开始到未来所有奖励的加权总和。
  • 折扣因子 γ：这是一个介于 0 到 1 之间的数。它代表了大脑对 ” 即时奖励 ” 和 ” 延迟奖励 ” 的偏好差。γ 越小，说明个体越短视，只看重眼前的利益；γ 接近 1，说明个体更有远见。

3. 递归的一致性：贝尔曼方程（Bellman Equation）

TD 学习最精妙的地方在于它发现了一个递归关系：

当前状态的价值 = 当前获得的奖励 + 打折后的下一个状态价值。

V(St​)=E[rt​∣St​]+γ∑P(St+1​∣St​)V(St+1​)

这种一致性是判断 ” 预测是否准确 ” 的唯一标准。如果这个等式两边不相等，就说明大脑的预测出错了。

4. 预测误差 δt​：学习的动力源

当预测不一致时，会产生一个时序差分误差（TD Error）：

δt​=rt​+γV(St+1​)−V(St​)
  • V(St​)：你之前的预期。
  • rt​+γV(St+1​)：你现在的 ” 实际体验 “。注意，这里的 ” 实际体验 ” 包含了即时奖励和对未来的新预期。
  • 学习逻辑：新的价值估算 = 旧的价值估算 + 学习率 × δt​。大脑会不断通过 δt​ 来修正自己的预期，直到误差归零。

5. 无模型学习（Model-Free Learning）

在复杂的现实环境中（比如股票市场或原始森林），个体很难提前知道环境的完整物理规律（即 P(r∣S) 和 P(St+1​∣S)）。

TD 学习提供了一种**随机采样（Stochastic Sampling）**的方法：你不需要知道概率分布，你只需要通过不断地尝试、犯错、采样，利用 δt​ 进行增量更新。这种 ” 边走边学 ” 的机制正是动物在陌生环境生存的逻辑，也是目前强化学习算法（如 AlphaGo）的基础。

6. TD 与 R-W 的深度对比

我们可以通过下表更直观地理解两者的演进关系：

特性Rescorla-Wagner 模型TD 学习模型
预测目标仅预测下一次即时奖励预测未来所有奖励的折现总和
时间敏感性粗略（以整个 Trial 为单位）精确（在 Trial 内部的每一秒更新）
二阶条件反射无法解释完美解释（下一个状态的价值 Vnext​ 充当了 ” 奖励 “）
多巴胺关联解释了奖励反馈解释了多巴胺在预测信号出现时的提前放电现象

State-Action Values / Q-Learning

状态 - 动作价值（State-Action Values）的学习代表了强化学习中另一种极具竞争力的 ” 无模型 ” 逻辑。如果说 Actor-Critic 架构是 ” 教练与运动员 ” 的关系，那么 Q 学习（Q-Learning）则更像是一位 ” 老练的实干家 “，他直接通过评估每一个具体动作的价值来做出决定。

1. Q-Value：数据结构

在之前的 TD 学习中，我们关注的是状态价值 V(S)，即 ” 身处这个状态我能拿多少奖金 “。而 Q 价值（Quality Value）则更进了一步，它将状态与动作绑定在了一起。

  • 定义： Q(S,a) 代表在状态 S 下执行动作 a 之后，能够预期的未来所有折扣奖励的总和。
  • 决策的极致简化： 这是 Q 价值最大的优势。在 Actor-Critic 中，你需要一个复杂的策略分布 π(S,a) 来决定该选什么。但在 Q 框架下，决策逻辑简化到了极点：
    • 如果你正站在十字路口（状态 S），你只需要查看左转的 Q(S,左转) 和右转的 Q(S,右转) 哪个分数更高。
    • 结论： 学习了 Q 价值，就等于直接拥有了最优策略。这种 ” 想 ” 和 ” 做 ” 的合一，使得算法在执行层面非常高效。

2. Q-Learning 与 SARSA：核心算法

这两者在数学公式上极其相似，但在 ” 如何看待未来 ” 的哲学观上完全不同。

Q-Learning：无畏的理想主义者（离策 / Off-policy）

其 TD 误差公式为：δt​=rt​+γmaxa​Q(St+1​,a)−Q(St​,at​)。

  • 核心算子： max。
  • 深度解析： 在计算误差时，Q-Learning 假设你下一时刻一定会采取那个最优的动作。哪怕你现在的策略很烂，一直在乱走，Q-Learning 评估的依然是那个 ” 理想状态下的最大潜力 ”。
  • 特性： 它非常激进。它致力于寻找全局最优解，而不关心你当前在实验中实际采取了什么动作。

SARSA：务实的经验主义者（同策 / On-policy）

其 TD 误差公式为：δt​=rt​+γQ(St+1​,at+1​)−Q(St​,at​)。

  • 核心变量： at+1​（你实际选出的下一个动作）。
  • 深度解析： SARSA 不看理想，只看现实。它在计算误差时，使用的是你实际上在下一刻选中的动作。如果你现在正处于探索阶段，可能会选错路，那么 SARSA 也会把这个潜在的风险计算在当下的评价中。
  • 特性： 它更稳健。由于它与当前采取的策略保持一致，因此在面对带有惩罚风险的环境时，SARSA 表现得比 Q-Learning 更加谨慎。

3. 生物学争议：多巴胺到底在编码什么？

这一部分是目前计算神经科学的前沿阵地。最初，人们深信多巴胺编码的是基于状态的误差（V 误差），但近年来的电生理证据打破了这一平衡：

  • 实验证据：
    • **Morris 等人（2006）**对非人灵长类动物的研究发现，多巴胺信号的强度与 Q-Learning 的预测误差高度吻合，即多巴胺似乎在根据 ” 未来最优选择 ” 来计算惊喜。
    • **Roesch 等人（2007）**在对大鼠的研究中则观察到了更符合 SARSA 特征的误差信号，即多巴胺在考虑大鼠下一步实际会做什么。
  • 核心矛盾： 之前的 Actor-Critic 模型（基于 V）无法完全解释这些 ” 与动作相关 ” 的多巴胺放电。这意味着，大脑可能不仅仅是在做全局的评价，而是在基底神经节内部直接对具体的动作进行估值。
  • 前沿设想： 大脑是否可能同时运行着这几种架构？或者大脑是将 Actor-Critic 进行了扩展，让 Critic 不仅仅观察状态，也观察动作？这仍是目前学术界最激烈的开放性讨论之一。

4. 核心对比总结：两条进化的道路

这两种架构代表了生物解决问题的不同方案：

  • Actor-Critic（分离式）：
    • 隐喻： 一个冷静的智囊团（Critic）指挥一个行动敏捷的将军（Actor）。
    • 优点： 灵活，可以处理连续的动作空间，更符合大脑解剖结构（腹侧与背侧纹状体的分工）。
    • 缺点： 信用分配需要通过两层传递，收敛过程有时较慢。
  • Q-Learning / SARSA（一体化）：
    • 隐喻： 一个拥有丰富实战经验、能直接根据直觉打分的特种兵。
    • 优点： 决策极其直接，在离散任务（如围棋、跳棋、迷宫）中效率惊人。
    • 缺点： 学习的是具体的 ” 状态 - 动作对 “，如果环境极其复杂，存储和计算这些组合的开销会变得很大。

无论大脑最终采用哪种方式，其核心目标始终未变：利用 TD 误差这一 ” 通用货币 “，将未来的希望转化为当下的行动动力。
