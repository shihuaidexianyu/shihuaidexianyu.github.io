五、效度与测验编制流程

这一讲核心速览

这一讲的灵魂问题是：你的测验到底测没测到你想测的东西？ 我们会搞懂三件事：① 效度是啥，跟信度啥关系；② 三种效度（内容效度、效标关联效度、构念效度）怎么区分、怎么算；③ 一个标准测验从立项到发布要经历哪些环节。考试常考概念辨析和几种效度的适用场景，务必分清！

01 效度概述

1. 效度所回答的核心问题

效度首先要回答两个灵魂拷问：

  • 第一，你这个测验到底测了个啥？有没有真的测到你想测的那个心理特质？
  • 第二，测得到底有多准？是随便蹭蹭边还是真的测到位了？

前者说的是”对象对不对”，后者说的是”程度够不够”。

用个比喻理解

想象你在射箭：靶子选错了（比如你要射A靶却对着B靶射），这就是效度问题——对象不对；靶子选对了但箭都射在边缘，这也是效度问题——程度不够。而信度是啥？信度是你每次射箭落点集不集中，跟你有没有射中靶心没关系。所以信度看稳定性，效度看对不对+准不准。

一个测验哪怕每次测出来结果都差不多（信度高），也可能因为测错了东西而完全没用（效度低）；反过来，测的对象对了，还得讨论测得够不够准、能不能支撑后续的解释和决策。

从更一般的方法论角度来说，效度不只是心理测验的事，而是所有测量和研究设计都要面对的。只要你拿数据去做解释、判断或预测，就必须先回答：“这些数据足够支撑我现在的解释吗？“

flowchart TD
    A[测验分数] --> B[它测的是什么]
    A --> C[它测到什么程度]
    B --> D[对象是否正确]
    C --> E[程度是否充分]
    D --> F[是否真的指向目标特质]
    E --> G[是否足以支持解释、预测与决策]

2. 效度的含义

效度就是测验对它想要测量的心理特质测到了什么程度的估计。现在更时髦的说法是：效度不是测验自带的固定属性，而是”测验分数在特定用途上的解释受到证据和理论支持的程度”。所以效度永远跟具体用途、目标人群、施测条件和解释方式绑在一起。同一个测验，换个用法，效度可能就变了。

重点：效度的本质

效度评估的不是”题目看起来像不像”，也不是”测验稳不稳定”，而是**“关于分数的解释是否合理、有没有证据支持、对当前用途有没有意义”**。高效度 = 分数可以被合理地解释为目标构念或目标行为的指标。

在经典测验理论框架下，效度可以写成：

V=SX2​SV2​​=rXY2​

参数速查

  • V：效度
  • SV2​：由目标心理特质引起的变异
  • SX2​：测验观察分数的总变异
  • rXY​：测验分数与效标之间的效度系数

这个式子告诉我们，效度其实在问：你观察到的总分数差异里，有多大比例是真正来自目标特质的？rXY2​ 表示测验和效标共享的变异比例，能说明两者相关到什么程度，但不能证明测验已经把构念的所有内容都测全了。

经典测验理论里还有一个变异分解的写法：

SX2​=ST2​+SE2​ ST2​=SV2​+SI2​

其中，ST2​ 是真分数变异，SE2​ 是随机误差变异，SI2​ 是系统性的无关变异（稳定地测到了不该测的东西）。于是：

V=SX2​SV2​​=SX2​ST2​−SI2​​=rXX​−SX2​SI2​​

这里的 rXX​ 就是测验信度。这个式子说明：效度不仅受随机误差影响，还受系统性无关变异影响。一个测验可以很稳定（信度高），但如果稳定地测错了东西，效度照样低。

经典误区：信度高 = 效度高？

错！ 信度高只说明测得稳，不代表测得对。就像你用一把刻度精准的尺子量身高，但每次量的是鞋跟高度——非常稳定，但完全没用。考试经常考这个辨析，记住：信度是效度的必要条件，不是充分条件。

flowchart LR
    A[观察分数总变异 S_X^2] --> B[真分数变异 S_T^2]
    A --> C[随机误差变异 S_E^2]
    B --> D[有效变异 S_V^2]
    B --> E[无关变异 S_I^2]

    D --> F[效度 V = S_V^2 / S_X^2]
    B --> G[信度 r_xx = S_T^2 / S_X^2]

3. 效度的基本性质

效度有四个关键性质，学长帮你总结好了：

  • 第一，效度依附于特定用途。离开用途谈效度没意义。一个测验用来诊断抑郁症可能效度不错，用来选拔销售员可能就拉胯。
  • 第二，效度评估的是对分数的解释，不是对测验材料本身的孤立评价。
  • 第三，效度只有程度高低，没有”有”或”无”的绝对划分。
  • 第四，同一个测验可以同时拥有多种效度证据，这些证据的高低可能并不一致。

用”偏差与变异量”的靶图可以直观理解信度与效度的差别：分数分布越集中，表示随机误差越小，越接近信度问题；分数整体越接近靶心，表示系统偏差越小，越接近效度问题。低变异但高偏差意味着”结果稳定但方向不对”，这正是”信度高而效度低”的典型情形。

02 效度种类及估计方法

效度验证是一个持续收集证据、检验解释是否成立的过程。通常从三个方面组织证据：内容效度、效标关联效度和构念效度。三者操作上能分开，逻辑上却互相联系：内容效度强调取样有没有代表目标领域，效标关联效度强调分数跟外部行为参照是否一致，构念效度强调理论解释是否成立。

flowchart LR
    A[效度验证] --> B[内容效度]
    A --> C[效标关联效度]
    A --> D[构念效度]

    B --> B1[代表性]
    C --> C1[与外部效标的一致性]
    D --> D1[构念解释与理论支持]

    B --> E[取样是否覆盖目标内容]
    C --> F[能否预测或区分外部表现]
    D --> G[能否支持理论假设]

1. 内容效度

内容效度说的是：测验项目对目标内容领域的代表性如何。它不问”得分跟外部行为是否相关”，而是问”题目有没有覆盖该覆盖的内容”。所以内容效度最适合内容范围清楚的测验，比如教育成就测验、职业选拔与分类测验。如果内容抽样都失真了，分数再稳定也没法合理解释。

（1）专家评定法

对于内容范围明确的测验，可以请一组独立专家来评估每个条目是否属于目标领域、有没有代表性。注意是”独立”专家，不是编制者本人。像焦虑、人格这种边界模糊的特质，内容效度尤其依赖专家判断，因为你很难像成就测验那样列出完整的知识清单。

专家评定可以用一致性指标来量化。如果两位专家把每个项目分为”弱相关”和”强相关”两类，就能构成一个 2×2 列联表，“双方都认为强相关”的比例就是：

CV=A+B+C+DD​

CV公式参数

  • A,B,C,D：列联表四个单元格的项目数
  • D：两位专家都评为”强相关”的项目数
  • CV 越接近 1，内容代表性越好

比如 100 个项目中四个单元格分别是 4,5,4,87，那么：

CV=4+5+4+8787​=0.87

这个值说明有 87% 的项目被两位专家一致认为与目标内容高度相关。它能帮助量化专家判断，但不能保证测验已经穷尽了所有应有内容——它能评价”现有项目合不合适”，却发现不了”该加但没加的项目”。

（2）双向细目表

双向细目表是内容效度评估的”神器”，把内容维度和认知维度同时列进去。内容维度是知识点、主题领域或行为范围；认知维度是记忆、理解、应用、分析、综合、评价等不同层次。

编题之前，用它来规定各类型项目占多大比重；编题之后，用它核查现有题目是否覆盖了预定结构。

flowchart TD
    A[确定内容范围] --> B[划分内容维度]
    A --> C[划分认知维度]
    B --> D[形成双向细目表]
    C --> D
    D --> E[按比例编题]
    E --> F[编题后回表核查]
    F --> G[判断内容代表性]

双向细目表还能提升测验的表面合理性，使使用者和被试直观感觉测验”确实在测这个领域”。但要注意——这种”看起来合理”只对应表面效度，不等于技术意义上的内容效度。

表面效度 ≠ 内容效度

这是考试最爱挖的坑之一：

  • 表面效度：题目看起来有没有效（主观感受，提高被试配合度）
  • 内容效度：题目实际上有没有代表目标内容（技术性评估）

表面效度高，内容取样不一定充分；内容效度高，也不一定自动带来高表面效度。表面效度甚至不算与技术效度并列的类型！

（3）统计法

内容效度也可以通过统计思路间接评估。复本法是为同一内容范围编两个独立复本，如果被试在两个复本上的得分高度相关，说明两个复本都较好地代表了同一内容范围。再测法则常用于成就测验：对起初了解较少的被试进行前测，经过系统学习后再后测；如果成绩显著提高，说明测验确实测到了培训内容，而非无关因素。

（4）表面效度与内容效度的边界

表面效度指测验在主观上看起来是否有效，它关心的是使用者、施测者和被试是否觉得题目”像是在测这个东西”。它的主要意义在于提高社会接受性和被试合作程度。严格来说，表面效度不是与内容效度、效标关联效度、构念效度并列的技术性效度类型。表面效度高，不代表内容取样一定充分；内容效度高，也不一定自动带来高表面效度。

2. 效标关联效度

效标关联效度说的是：测验分数与某一外部效标之间的一致程度。它所测量的是：测验分数能否反映或预测一个独立于测验本身、同时又真正代表测验目的的外部行为参照。

效标关联效度的典型场景

  • 招聘考试能否预测未来的工作绩效？
  • 飞行员选拔测验能否预测后续的飞行表现？
  • 高考分数能否预测大学学业成绩？

一个合适的效标应满足：相关性、有效性、无污染、客观性、实用性。所谓”无污染”，就是效标不能被测验分数本身反向影响，否则两者相关可能只是循环定义，不是真实预测。

常见效标包括学业成就、等级评定、临床诊断、训练成绩、实际工作表现以及其他现成有效测验。

（1）同时效度与预测效度

同时效度 vs 预测效度：时间差是关键

很多同学容易混淆这两个，核心区别不在计算公式，而在效标资料获取的时间：

  • 同时效度：测验分数和效标资料同时获得。比如心理障碍量表与医生面谈结果在同一时期的一致程度。适合诊断、筛查、判断当前状态。
  • 预测效度：先施测，未来再收集效标。比如入学测验预测毕业后工作表现。适合选拔、安置、预测未来。

记住口诀：“同时”看现在，“预测”看未来。

若测验分数与效标资料同时获得，所检验的是同时效度。例如，心理障碍量表与医生面谈结果在同一时期的一致程度。若先施测，再在未来一段时间内收集学习成绩或工作表现，则检验的是预测效度。二者的差别不在相关系数的计算方式，而在于效标资料获取的时间位置及用途。

flowchart LR
    A[效标关联效度] --> B[同时效度]
    A --> C[预测效度]

    B --> B1[测验与效标同时收集]
    B --> B2[适合诊断、筛查、当前状态判断]

    C --> C1[先施测 后获得效标]
    C --> C2[适合选拔、安置、预测未来表现]

（2）相关法

最常见的方法就是计算测验分数与效标分数的相关系数，即效度系数 rXY​。X 是测验分数，Y 是效标分数。rXY​ 越高，说明两者越一致。它适用于效标也是连续变量的情境，比如高考成绩与大学一年级成绩的相关。

需要注意：相关高说明线性关系强，但不能单独说明因果方向，也不能说明测验已经覆盖了构念的全部内涵。

（3）分组检验法

如果效标把被试明确分成两类或多类，比如”临床诊断阳性/阴性""高绩效/低绩效”，就可以比较不同组在测验上的得分有没有显著差异。如果理论上应该被区分的群体在分数上确实明显分离，就能为效标关联效度提供证据。适用于分类诊断和群体区分。

（4）取舍正确率

在选拔、安置和鉴别决策中，常根据测验临界值把被试分为”录取/不录取""阳性/阴性”，再跟实际效标分类比较。设四类结果为：正确接受 A、错误接受 B、错误拒绝 C、正确拒绝 D，则：

总命中率=A+B+C+DA+D​ 正命中率=A+BA​

取舍正确率公式速查

  • 总命中率：全部决策中，有多少比例与真实效标一致。
  • 正命中率：被测验判为”录取/阳性”的人中，有多少确实符合真实标准。
  • 优点：容易理解、实施方便。
  • 缺点：只利用了临界值两侧的信息，忽略了完整分值分布；临界值设置不当还会带来公平性问题。
flowchart TD
    A[测验分数] --> B{是否达到临界值}
    B -->|是| C[测验判定为合格]
    B -->|否| D[测验判定为不合格]

    C --> E{真实效标}
    D --> F{真实效标}

    E -->|合格| G[正确接受 A]
    E -->|不合格| H[错误接受 B]
    F -->|合格| I[错误拒绝 C]
    F -->|不合格| J[正确拒绝 D]

3. 构念效度

构念是心理学中的抽象、假设性概念或特质，比如智力、动机、抑郁。构念效度关心的是：测验分数能不能被恰当地解释为某个构念的表现。跟内容效度和效标关联效度相比，构念效度更强调理论框架的作用，它不仅问”测到了什么”，还问”为什么这样解释是合理的”。

构念效度的目标是揭示构念与具体行为的关系、构念之间的关系，并据此解释和预测行为。所以它特别适合智力测验、人格测验这类很难用单一内容清单来界定的测验。

（1）构念效度验证步骤

构念效度验证通常走三步：先界定构念、建立理论框架；再依据框架提出一堆可检验的假设；最后收集多方面资料，检验假设是否成立。资料可以来自测验内部结构，也可以来自构念随年龄、训练、群体差异而变化的外部动态。

flowchart TD
    A[界定构念] --> B[建立理论框架]
    B --> C[提出可检验假设]
    C --> D[收集多方证据]
    D --> E[检验假设是否成立]
    E --> F[支持或修正构念解释]

（2）内部一致性

内部一致性衡量测验条目之间是否”步调一致”，能否共同指向同一个测量方向。常用指标有分半相关、题总相关、Cronbach α、KR 系列等。内部一致性高，说明各条目比较协调，是构念效度成立的重要前提之一。

内部一致性高 ≠ 构念效度高

这是另一个考试重灾区！α 高只说明”条目彼此很合拍”，但不保证”这种合拍是围绕目标特质的”。如果所有条目都稳定地测错了同一个东西，内部一致性照样可以很高。所以：

  • α 高 ≠ 构念效度高
  • α 高 ≠ 单维性成立

记住：内部一致性是构念效度的必要前提，不是充分证明。

但内部一致性并不能保证测验已经测到了目标构念。它只能说明”条目彼此一致”，不能保证”这种一致正好围绕目标特质”。因此，α 系数高不等于构念效度高，也不自动推出单维性成立；多个高度相关但共同偏离目标构念的条目，同样可能得到较高的内部一致性。

（3）与同类测验的相关

如果你依据某个理论编了一份新测验，发现它和权威的同类旧测验相关很高，这能为聚合效度提供证据。聚合效度又叫求同效度或收敛效度，意思是：理论上应该相关的变量，确实表现出较高相关。

聚合效度 vs 区分效度

  • 聚合效度（收敛效度）：理论上相关的变量，实际也相关高。
  • 区分效度：理论上不相关的变量，实际相关低。 这两个概念在构念效度验证里经常成对出现，是多特质-多方法矩阵的核心。

然而，新测验如果只是跟旧测验高度相关，却没有表现出新的理论价值、应用优势或测量改进，也没法证明新测验存在的必要性。所以与同类测验的高相关只是支持性证据，不是新测验合理性的全部依据。

（4）发展变化与理论一致的群体差异

对于随年龄明显变化的构念，可以看看测验分数是否呈现合理的发展轨迹；对于可被训练显著提升的特质，也可以考察训练前后变化是否符合理论预期。如果分数变化方向跟理论一致，就能为构念效度加分。

另一种思路是比较理论上应该高分或低分的群体。

修女 vs 反社会群体

如果某社会兴趣量表真的测”对他人的关心与关注”，那么修女等理论上社会兴趣高的群体应该显著高于以自我中心或反社会倾向著称的群体。关键不是”群体均值是否不同”，而是”差异方向是否与理论一致”。

（5）因素分析

因素分析通过考察项目之间的相关结构，寻找背后的共同因素。

  • 探索性因素分析（EFA）：偏数据驱动，用于在数据中抽取潜在共同因素，适合结构尚不明确的早期开发阶段。
  • 验证性因素分析（CFA）：偏理论导向，用于检验预设结构与数据之间的拟合程度，适合在独立样本中验证既定模型。

设第 i 个项目分数为 Xi​，则共同因素模型可写为：

Xi​=ai1​F1​+ai2​F2​+⋯+aim​Fm​+ci​Ui​

其中，F1​,…,Fm​ 为共同因素，aij​ 为项目 i 在因素 j 上的负荷，Ui​ 为项目的独特因素，ci​ 为对应系数。项目在共同因素上被解释的方差可用共同度表示：

hi2​=j=1∑m​aij2​

这里，hi2​ 越大，表示该项目越多地反映了共同因素；若某测验的主要方差来自理论相关因素，则可视为其构念效度获得支持。需要注意的是，因素结构合理只能说明数据与理论相容，不能单独证明理论是唯一正确解释。

因素分析的局限

因素结构合理 ≠ 理论唯一正确。EFA/CFA 只能说明”数据不拒绝你的理论”，但不能证明”你的理论是唯一对的”。这是假设检验的基本逻辑，考试时可能出现在辨析题里。

（6）多特质-多方法矩阵

多特质-多方法设计同时使用多种方法测量多种特质，并计算相关矩阵，以此综合考察信度、聚合效度和区分效度。理想模式是：同一特质、不同方法之间相关较高，说明聚合效度高；不同特质之间相关较低，说明区分效度好；同一方法的信度系数也应保持较高。

flowchart TD
    A[多特质-多方法设计] --> B[同一特质 不同方法]
    A --> C[不同特质 同一方法]
    A --> D[同一方法 同一特质]

    B --> B1[相关应较高]
    B1 --> B2[支持聚合/收敛效度]

    C --> C1[相关应较低]
    C1 --> C2[支持区分效度]

    D --> D1[相关应较高]
    D1 --> D2[支持方法信度]

03 效度的影响因素和应用

1. 影响效度的因素

（1）测验本身

测验长度会影响效度。增加同质性条目通常能提高测验稳定性，从而间接提高效度；但如果增加的是与目标构念无关的重复内容，那就白加了。所以”题目更多”只有在”新增题目仍然测同一目标特质”的前提下才有意义。

测验信度也直接制约效度。观察分数里如果含有大量随机误差，分数就没法稳定反映目标特质；即使随机误差小，如果存在稳定的系统偏差，效度还是会下降。换言之，低信度必然妨碍高效度，但提高信度只是提高效度的必要条件之一。

（2）效标因素

在效标关联效度研究中，效标是否恰当至关重要。不合适的效标会直接削弱效度结论。若测验与效标的关系并非线性，要先通过散点图等方式识别关系形态，再选合适的分析方法。若效标本身测量不稳定，测验与效标之间的相关也会被压低，此时可通过多次测量取平均等方式提高效标稳定性。

（3）样本代表性

用于验证效度的样本应该能代表测验未来实际适用的总体。若样本范围过窄，特别是存在预选拔、范围限制等情况，相关往往会被压缩。

范围限制的坑

用已经被筛过一轮的新雇员来验证职业选拔测验，可能会低估测验对全部候选人的预测作用。因为剩下的人本身就比较同质，变异被压缩了，相关自然看起来就低了。

（4）干涉变量

当某个第三变量会改变测验分数与效标之间关系时，它就构成干涉变量或调节变量。比如动机水平可能调节智力测验分数与学习成绩之间的关系：高动机群体里这种关系可能更强，低动机群体里可能减弱。所以效度研究不能只看总体相关，还要警惕关系是否因情境或群体而异。

flowchart LR
    A[效度水平] --> B[测验本身]
    A --> C[效标质量]
    A --> D[样本代表性]
    A --> E[干涉变量]

    B --> B1[测验长度]
    B --> B2[测验信度]

    C --> C1[效标选择]
    C --> C2[关系类型]
    C --> C3[效标信度]

    D --> D1[范围限制]
    E --> E1[调节作用]

2. 信度与效度的关系

信度与效度的关系可以概括为：信度是效度的必要条件，但不是充分条件；效度高通常意味着信度也不能太低。如果一个测验连稳定、可靠地收集数据都做不到，就不可能真实反映研究对象；但一个测验即使非常稳定，也可能稳定地测错对象。

重点：信度与效度的逻辑边界

  1. 信度低，效度不可能高。

  2. 信度高，效度不一定高。

  3. 效度低，信度可能仍然较高。

  4. 效度高，信度必然不能太低。
    因此，信度为效度服务，是效度的基础；效度则是测量的最终目标。

四句话记忆口诀

低信必低效，高信非高效；低效可高信，高效需高信。记住这四句，选择题基本不会错。

在与外部效标相关的情境中，效度系数受测验信度和效标信度共同限制，其上限可写为：

rXY​≤rXX​rYY​​

效度上限公式

  • rXY​：效度系数
  • rXX​：测验信度
  • rYY​：效标信度
  • 意义：即使测验设计再好，只要测验或效标不可靠，相关就不可能无限提高。这是理论上限，不是实际效度值。

例如，若某测验信度为 0.80，效标信度为 0.20，则其效度系数的理论上限为：

rXY,max​=0.80×0.20​=0.16​=0.40

这表示即使测验与构念匹配得很好，由于效标本身过于不稳定，最终可观察到的效度系数也不可能超过 0.40。注意这里不能推出”真实关系只有 0.40”，只能说明”当前测量条件下最多只能观测到 0.40”。

flowchart LR
    A[高信度] --> B{是否测对对象}
    B -->|是| C[高效度]
    B -->|否| D[信度高但效度低]

    E[低信度] --> F[效度不可能高]

3. 效度的应用

（1）预测效标分数

当测验与效标之间存在线性关系时，可以用回归方程根据测验分数预测效标分数：

Y^=bYX​X+aYX​

其中，Y^ 为预测的效标分数，X 为被试的测验分数，bYX​ 为回归斜率，aYX​ 为回归截距。斜率和截距可写为：

bYX​=rXY​sX​sY​​ aYX​=Yˉ−bYX​Xˉ

回归参数含义

  • bYX​：测验分数每增加一个单位，效标预测值平均增加多少
  • aYX​：当 X=0 时的预测起点
  • 适用条件：线性关系。明显非线性时别硬套。

其中，rXY​ 为测验与效标的相关系数，sX​ 和 sY​ 分别为测验与效标的标准差，Xˉ 和 Yˉ 分别为它们的均值。斜率表示测验分数每增加一个单位，效标预测值平均增加多少；截距表示当 X=0 时的预测起点。该公式适用于线性预测，不适合直接处理明显非线性关系。

来看一个具体例子：

用数学推理测验预测代数成绩

已知：数学推理测验 Xˉ=4.30，sX​=1.79；代数成绩 Yˉ=2.15，sY​=1.22；两者相关 rXY​=0.60。某学生数学推理测验得分为 X=6，预测他的代数成绩。

第一步，算斜率： bYX​=0.60×1.791.22​≈0.409

第二步，算截距： aYX​=2.15−0.409×4.30≈0.3913

第三步，代入预测： Y^=0.409×6+0.3913≈2.8453

所以该学生的代数成绩预测值约为 2.85。这只是概率性估计，不是确定结论。

现有一例：数学推理测验的均值为 Xˉ=4.30、标准差为 sX​=1.79；学生代数成绩的均值为 Yˉ=2.15、标准差为 sY​=1.22；两者相关为 rXY​=0.60。若某学生数学推理测验得分为 X=6，则先计算斜率：

bYX​=0.60×1.791.22​≈0.60×0.6816≈0.409

再计算截距：

aYX​=2.15−0.409×4.30=2.15−1.7587≈0.3913

于是预测代数成绩为：

Y^=0.409×6+0.3913=2.454+0.3913≈2.8453

因此，该学生的代数成绩预测值约为 2.85。这个结果表示在样本总体的线性关系下，其预期成绩高于代数成绩均值，但它仍只是概率性估计，而不是确定结论。

（2）预测误差与区间估计

基于测验分数预测效标分数时，总会存在预测误差。预测误差的标准差常写为：

1−rXY2​=sY2​sest2​​ sest​=sY​1−rXY2​​

预测误差公式解读

  • rXY2​：效标总变异中可被预测解释的比例
  • 1−rXY2​：未被预测解释的”残留”比例
  • sest​：估计标准误，越小预测越精确
  • 意义：告诉你”预测大致会偏离多少”，但不保证对每个个体都同样准确

其中，sest​ 为估计标准误，sY​ 为效标分数的标准差。该式说明，rXY2​ 表示效标总变异中可被预测解释的比例，剩余的 1−rXY2​ 则是未被预测解释的比例。估计标准误越小，说明围绕回归线的离散程度越低，预测越精确。它能告诉研究者”预测大致会偏离多少”，但不能保证对每个个体都同样准确。

在上例中，rXY​=0.60，故：

rXY2​=0.36

这表示代数成绩总变异中约 36% 可由数学推理测验分数解释，仍有约 64% 来自未解释因素。进一步有：

sest​=1.221−0.36​=1.220.64​=1.22×0.8=0.976

这说明预测成绩周围仍存在约 0.98 个分数单位的典型误差。若在进一步假设残差分布近似正态的前提下，可以围绕 Y^ 构造预测区间；但区间估计的合理性依赖于线性关系、误差分布和样本适用性等条件。

（3）选拔决策中的应用

在人才选拔中，效度不仅体现在相关系数上，也体现在决策命中率上。此时需要在录取率、误录成本、漏录成本之间取得平衡。单纯追求更高命中率，并不一定意味着更合理的选拔；因为录取比例改变后，命中率本身也会变化。因此，效度应用必须结合具体决策成本来理解。

（4）补充：提高验证研究敏感性的做法

在效度研究或测验开发的实证检验中，提高统计检验能力有助于更清楚地发现真实关系。常见做法包括：降低误差变异，强化实验操作，增加样本量，适当调整显著性标准，改进分析方法，优化实验设计，减少不必要条件，增加每个条件下的试次数，改变任务或反应方式，减少极端数据，并通过重复测量或更清晰的任务说明降低个体差异带来的噪声。需要注意，这些做法提高的是研究检测真实效应的能力，而不是直接”制造”效度本身。

04 测验的编制

1. 测验编制的总体流程

标准测验编制通常走六步：确定测验目的 → 拟订编制计划 → 产生测题 → 检验条目 → 修改测验 → 发表测验。整体上分为准备阶段、编制阶段和发布阶段。准备阶段定方向，编制阶段反复试测修订，发布阶段形成标准化手册和使用规范。

flowchart TD
    A[步骤1 确定测验目的] --> B[步骤2 拟订编制计划]
    B --> C[步骤3 产生测题]
    C --> D[步骤4 检验条目]
    D --> E[步骤5 修改与合成测验]
    E --> F[步骤6 发表测验]

    A --> A1[准备阶段]
    B --> A1
    C --> C1[编制阶段]
    D --> C1
    E --> C1
    F --> F1[发布阶段]

2. 准备阶段

（1）确定测验目的

编制测验首先要明确：你到底想测什么？给出理论基础跟操作化定义。是测能力、人格、心理健康还是学业成就？不同对象对应不同的条目来源、评分方式和效度证据类型。

同时还得确定适用对象：年龄、智力水平、社会经济地位、文化背景等。面对青少年、儿童、老年人还是特殊群体，直接影响题目表述、难度和常模构建。

另外要决定是开发新测验还是修订旧测验。开发新测验适用于新概念或原有工具明显不足的情形；修订已有测验则更适合在熟悉领域中做文化适配、结构优化或题项更新。

最后必须明确用途是描述、诊断、选拔还是预测。用途不同，分数解释框架也不同。若关注是否达到既定标准，属于标准参照解释；若关注个体在群体中的相对位置，属于常模参照解释。前者强调与目标标准比较，后者强调与同龄或同类群体比较，二者不能混同。

（2）拟订编制计划

编制计划的核心工具仍是双向细目表。它不仅用于内容效度评估，也用于实际编题规划：确定测验包括哪些内容、哪些能力层次、各部分占多大比重，以及最终应编多少题、编什么类型的题。题目形成后，还应回到该表进行核查，判断结构是否兑现了最初的蓝图。

3. 编制阶段

（1）产生测题：来源

测题来源主要有三种。第一，咨询领域专家。专家可提供目标群体在认知、情绪和行为上的典型表现，并将这些表现转化为条目。第二，借鉴已有量表，但必须谨慎处理文化和情境差异，不能简单照搬。第三，结合理论与质性研究编制条目，使题项既有理论基础，又保留目标人群的真实语言和经验结构。

在临床或人格测验中，专家访谈尤其重要，因为很多条目需要来自对症状表现和典型行为模式的归纳。借鉴已有量表时，则需警惕文化背景改变后项目内容失真，例如国外常识题在本土环境中可能不再等价。

（2）质性研究在编题中的作用

当构念边界尚不清楚或需要贴近本土生活经验时，可借助质性研究生成条目框架。一般流程是：先明确研究问题，再设计结构化或半结构化提纲，通过访谈或焦点小组收集材料，对目标群体进行适当抽样，随后进行开放式编码、主轴编码和选择性编码，最后归类并形成结构框架或理论框架。

flowchart TD
    A[研究问题] --> B[访谈/焦点小组提纲]
    B --> C[目标人群抽样]
    C --> D[资料收集]
    D --> E[开放式编码]
    E --> F[主轴编码]
    F --> G[选择性编码]
    G --> H[归类与结构整合]
    H --> I[形成构念框架]
    I --> J[生成题项池]

质性研究编题实例

  • 小学儿童勇气心理结构：通过访谈获得儿童对勇气的日常理解，经编码后形成”无所畏惧、不逃避、坚持”等二级维度，并进一步整合为”突破、坚毅、担当”等更高层结构。
  • 疫苗犹豫问题：对开放回答编码，归纳出”没必要、安全性、有效性”等主题。

质性分析的作用不只是”整理材料”，而是把零散叙述转化为可测量的结构维度。

以”小学儿童勇气心理结构”为例，可通过访谈获得儿童对勇气的日常理解，经编码后形成”无所畏惧、不逃避、坚持”等二级维度，并进一步整合为”突破、坚毅、担当”等更高层结构。类似地，对疫苗犹豫问题的开放回答可通过编码归纳出”没必要、安全性、有效性”等主题。可见，质性分析的作用不只是”整理材料”，而是帮助研究者把零散叙述转化为可测量的结构维度。

在文本量较大时，还可结合文本分析方法，如中文分词、去重、停用词删除、词频排序和关联规则算法，用于发现高频概念与共现主题。但这些技术只能帮助研究者识别候选内容，最终仍需人工筛查和理论判断，不能机械地把高频词直接当作优质条目。

（3）产生测题：形式选择

测题形式会影响作答方式、统计性质和误差来源。

常见题型特点速查

  • 二分式：编制和计分简单，但容易受猜测作答影响，稳定性较差。
  • 多选式：可以在较短时间内覆盖大量内容，但干扰项设计难度更高。
  • Likert 式：使用广泛，便于统计分析，但本质上首先是顺序量尺，其相邻等级不必然具有严格等距意义。
  • 类别式：选项容易受情境影响，因此必须把各类别含义界定清楚。
  • 其他：简答、论述、联想、改错、图解以及视觉模拟量表等。视觉模拟量表可用连续线段表达强度，如”对感染新冠有多害怕”，能提供比离散等级更细的反应。

二分式项目编制和计分简单，但容易受猜测作答影响，稳定性较差。多选式可以在较短时间内覆盖大量内容，但干扰项设计难度更高。Likert 式使用广泛，便于统计分析，但本质上首先是顺序量尺，其相邻等级不必然具有严格等距意义。类别式选项容易受情境影响，因此必须把各类别含义界定清楚。除这些形式外，还可使用简答、论述、联想、改错、图解以及视觉模拟量表等。视觉模拟量表可用连续线段表达强度，如”对感染新冠有多害怕”，能提供比离散等级更细的反应。

（4）检验条目

条目形成后，需要通过预测或试测检验其质量。试测对象应来自未来正式施测的人群，施测环境尽量接近正式情况，但可适当放宽时间，以便充分收集问题与反馈。条目复核时应考虑对象特征、人数安排以及环境、时间、施测者等标准化因素。课件提示样本量配置可按约 1:10 的比例进行考虑，其实质是在强调条目分析需要足够样本支持。

试测之后，至少要关注四类指标：项目难度，即回答正确的比例；项目鉴别度，即区分高低分被试的能力；整体信度，即分数是否稳定可靠；整体效度，即是否真正测到了目标构念。对于结构性量表，还应进一步进行 EFA 与 CFA，以检验维度结构是否合理。

（5）修改与合成测验

修改阶段要决定哪些条目删除、哪些条目保留、哪些条目需要重写。没有辨别度、明显拉低信效度或偏离测量目标的条目应予以删除；表述略显模糊但能够有效触及构念的条目则可考虑修订后保留。修改的原则不是追求语言形式上的完美，而是围绕”是否有助于测量目标构念”来决策。

测验合成阶段还要安排题目顺序，可综合考虑题目类型、所测内容和题目难度。良好的编排既有助于作答流畅，也有助于减少被试疲劳、情境转换和顺序效应带来的无关变异。

4. 问卷设计的具体要求

问卷设计应同时关注结构清晰、语义清晰、价值观与文化问题、正反问题与计分、注意力控制和时间掌控。一个完整问卷通常包括标题与前言、指导语、正式题项以及结束语和其他说明。

在题项表述上，应尽量准确、简洁，每一题只表达一个问题，避免多重含义。例如，“你喜欢锻炼和自然吗”实际上把两个偏好放进了同一条目。还应尽量少用双重否定，因为双重否定会显著提高理解负担，增加无关误差。

问卷设计的常见坑

  1. 一题多问：“你喜欢锻炼和自然吗？” → 锻炼和自然是两件事！
  2. 双重否定：显著提高理解负担，增加误差。
  3. 诱导性措辞：如”每个人都应当……”这类评价性前导语，会把规范压力混入作答。
  4. 反向题过多：虽然能抑制单向应答，但太多太复杂会引发理解错误和方法效应。

问卷用词还会显著影响结果。过于严格、带强烈价值色彩的措辞，可能改变被试对问题的接受方式，导致不同表述得到不同分布。因此应尽量保持中性，避免诱导作答和社会赞许性偏差。例如，带有”每个人都应当……”这类评价色彩的前导语，会把规范压力混入作答，从而降低测量纯度。

正向与反向题可以在一定程度上抑制单向应答倾向，但反向题过多、过复杂，也可能引发理解错误和额外方法效应。因此，反向计分并不是越多越好，而应服务于控制反应定势和提高测量质量这一目的。

注意力控制是问卷设计中的独立环节。既然问卷结果依赖被试认真阅读和区分题意，实施中通常需要通过显式注意力检测题、反向题一致性、重复题一致性、作答时长以及整行同答等反应模式综合判断被试是否认真作答。任何单一指标都不宜机械地作为剔除标准，而应结合样本特点、题目长度和研究目的综合决策。这一要求的根本理由在于：低注意力作答会引入大量无关误差，既损害信度，也损害效度。

5. 发布阶段

（1）等值复本编造

在需要复本的测验中，等值复本必须满足若干条件：测量同一种心理特质，内容与形式大致相同，没有重复条目，题目数量相等，难度与区分度大致相当，分数分布也应接近。常见做法是先准备足够数量的条目，按难度排序后，根据需要编成的复本个数进行均衡分配。这样做的目的是使不同复本之间尽量保持可替换性，从而避免”换了一套卷子就不是同一个测验”的问题。

（2）测验标准化

测验发表并不只是把题目印出来，还必须完成标准化。标准化通常包括施测内容、统一指导语、时限、评分规则、评分工具、常模以及测验指导手册。没有这些内容，测验即使题目本身不错，也很难保证不同施测者、不同时间、不同地点下结果具有可比性。

标准化七要素

  1. 统一施测内容
  2. 统一指导语
  3. 统一时限
  4. 统一评分规则
  5. 评分工具
  6. 常模
  7. 测验指导手册

没有这些，不同施测者、不同时间、不同地点的结果就没法比较。

flowchart LR
    A[正式测验发布] --> B[统一施测程序]
    A --> C[统一指导语]
    A --> D[统一时限]
    A --> E[统一评分规则]
    A --> F[评分工具]
    A --> G[常模建立]
    A --> H[测验指导手册]

    B --> I[结果可比]
    C --> I
    D --> I

    F --> I
    G --> J[分数解释有参照]
    H --> K[测验可复制使用]

常模的作用在于为分数解释提供参照，使个体分数能够被置于适当群体中理解；指导手册则负责把测验的编制目的、适用范围、施测要求、评分方法和解释原则系统化。发布阶段的核心不是”把题目公之于众”，而是形成一套可以稳定实施、稳定评分、稳定解释的测量体系。

6. 问卷开发中的信效度报告

问卷开发完成后，应系统报告信度和效度证据。信度方面，常包括重测信度、复本信度、折半信度、Cronbach α、KR 系列指标和评分者信度，它们分别对应时间稳定性、形式等值性、内部一致性与评分一致性等不同来源的一致性。效度方面，通常报告内容效度、效标效度、EFA、CFA 以及聚合效度和区分效度等证据。这里必须再次强调：信度指标说明结果是否稳定一致，效度指标说明分数解释是否合理，二者不能互相替代。
