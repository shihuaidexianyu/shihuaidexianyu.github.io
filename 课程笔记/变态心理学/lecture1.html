第一章 概论

Summary

变态心理学第一章其实就是给你搭个“认知框架”。本章核心就四块：① 研究对象和任务（描述、找原因、干预）；② 一堆容易混的概念（心理病理学、神经病、精神障碍、精神病、心理障碍——考试爱考辨析）；③ 正常 vs 异常的判定标准（没有绝对标准！）；④ 历史流派和研究方法。后面所有章节都是在这个地基上盖楼，所以概念边界和判定逻辑务必理清楚。

本章构成变态心理学的入门框架，核心内容包括学科对象与相关概念、正常与异常的判定标准、异常心理学历史上的主要解释路径，以及研究异常心理现象的基本方法。

一、变态心理学的研究对象与基本任务

变态心理学，又译异常心理学，是心理学中专门研究异常心理现象及其发生、发展和变化规律的分支。其研究对象并不限于“症状”本身，而是同时关注异常现象的表现形式、成因机制、发展过程以及干预方式。作为一门科学心理学分支，它的基本任务可以概括为三个方面。

Note

记任务的时候可以套个口诀——“描原治”：描述现象、发现原因、治疗干预。答题时按这三层展开基本不会漏。

  • 描述现象
    • 描述的重点在于明确异常心理与正常心理的差异，识别症状的具体表现、持续方式、病程变化及预后特点。没有准确的现象描述，就无法建立稳定的诊断标准，也无法形成可比较的研究对象。
  • 发现原因
    • 异常心理现象的形成通常不是单一因素导致，而是在生物、心理和社会多层因素共同作用下出现。成因研究既包括神经系统和生理机制，也包括认知、情绪、人格、学习经历、家庭关系和社会环境等层面。
  • 治疗干预
    • 变态心理学不仅说明“异常是什么”，还要说明“如何干预”。干预研究涉及不同理论取向下的治疗原则、治疗方法、适应对象及疗效评价。

与变态心理学相关但不完全等同的概念包括心理病理学、神经病、精神障碍、精神病和心理障碍。它们的研究重心、适用范围和学科归属并不相同。

二、相关概念及其边界

Trap

这几个概念是选择题和名词解释的重灾区！核心区别在于视角和严重程度：神经病 = 神经系统病变（医学/神经科）；精神障碍/心理障碍 = 异常心理与行为（前者偏精神病学，后者偏心理学）；精神病 = 严重到现实检验受损、自知力缺失的那一类。不要把“神经病”当成骂人的话写进卷子里，它是个严格的神经病学概念。

1. 心理病理学

心理病理学强调对异常心理现象的原因和机制进行分析。与变态心理学相比，心理病理学更偏向解释层面，关注异常现象为何发生、通过什么机制维持以及不同理论如何说明这些机制。可以说，变态心理学兼顾描述、解释与干预，而心理病理学更突出机制性说明。

2. 神经病

神经病属于神经病学的研究对象，指神经系统发生器质性或功能性损害后所表现出的疾病状态。它首先是神经系统本身的病变，例如周围神经受损、脑部损伤或其他神经系统异常。神经病的核心是神经系统病理改变，而不是以心理或行为失常为主要定义依据。

3. 精神障碍

精神障碍是精神病学中的核心概念，常与精神疾病近义使用，指各种显著影响心理、情绪、认知和行为功能的临床状态。其范围很广，既包括精神分裂症、抑郁障碍、双相障碍、焦虑障碍、人格障碍等，也包括由脑器质性损伤引发的精神症状。也就是说，某些异常心理和行为虽然源自神经系统损伤，但只要其主要临床表现属于精神与行为异常，仍可归入精神障碍的范畴。

4. 精神病

精神病既有广义用法，也有狭义用法。广义上，精神病可以近似指各种严重精神疾病；狭义上，它专指一类以现实检验能力明显受损为核心特征的临床状态。日常和临床语境中更常用的是狭义概念。

Intuition

怎么快速判断是不是“狭义精神病”？问自己三个问题：① 他还能分得清主观体验和外部现实吗？（现实检验）② 还能正常生活工作吗？（社会功能）③ 他承认自己有问题吗？（自知力）。三个里头至少前两个崩了，才靠近狭义精神病。

狭义精神病具有三个关键特征：

  1. 现实检验能力严重受损，个体不能恰当区分主观体验与外部现实，可能出现妄想、幻觉或严重思维紊乱；
  2. 社会功能严重受损，个体难以维持日常生活、工作或人际关系；
  3. 缺乏症状自知力，不能认识到自己的体验和行为已经异常。

精神病不是某一种单独疾病的名称，而是某些障碍中可能出现的一组严重症状状态。精神分裂症是典型例子，但某些双相障碍发作期也可能伴有精神病性症状。传统上，精神病还可区分为器质性与功能性两类：前者与明确脑部病变、中毒或躯体因素有关，后者主要指精神分裂症、偏执性或情感性精神病等。

5. 心理障碍

心理障碍是心理学中对多种心理、情绪和行为失常的统称。广义上，它与精神病学中的精神障碍范围接近；狭义上，它更强调从心理学角度理解异常现象，常用来指那些不以重性精神病和器质性障碍为核心的异常状态，如焦虑障碍等。

因此，心理障碍与精神障碍并非绝对对立。二者在广义上有很大重合，但“精神障碍”更偏向医学—精神病学视角，“心理障碍”更偏向心理学视角。精神病则是其中更严重、更具现实检验缺损特征的一类状态，而神经病则属于神经系统疾病范畴。

几者之间的边界关系可以概括为：神经病强调神经系统病变，精神障碍和心理障碍强调异常心理与行为，精神病强调其中严重到损害现实接触、自知力和社会功能的状态。

三、正常与异常的界定

1. 文化、常模与情境

异常心理的界定不存在脱离文化与情境的绝对标准。文化相对论强调，行为是否被视为异常，必须放回其所处的文化、社会规范和具体情境中判断。同样的行为在一种文化中可能是正常礼俗，在另一种文化中却可能被视为怪异。例如，见面时亲吻面部在某些文化中属于日常礼仪；拒绝进食可能是宗教禁食、节食行为，也可能提示进食障碍；喃喃自语既可能只是自我言语，也可能在某些情境下提示认知或知觉异常。

因此，正常与异常的判定不能脱离社会常模，也不能脱离行为发生时的背景、频率、持续时间和功能后果。

Note

正不正常是一个相对的概念，没有绝对的标准，需要依托于常模。

2. 多维判定标准

Example

Barlow & Durand 的三标准很好记：痛苦 + 功能缺损 + 异常/违反规范。Nevid 等人的六条可以当成“ checklist ”：奇不奇怪？接不接受？对不对现实？痛不痛苦？适不适应？危不危险？做题时如果案例描述里占了多条，基本就往“异常”靠。

心理障碍的界定通常采用多维度标准，而不是依赖单一指标。

Barlow 和 Durand 将心理障碍的界定概括为三个方面：内心痛苦、社会功能缺损、行为异常或违反社会规范。这个框架强调异常不仅体现在主观体验，也体现在社会适应与行为外显上。

Nevid、Rathus 和 Greene 提出的标准更细，主要包括六项：第一，行为不同寻常；第二，行为不被社会接受或打破社会常模；第三，对现实的感知或解释错误；第四，个体处于明显痛苦之中；第五，行为具有非适应性或自我挫败性；第六，行为具有危险性，可能危及自己或他人。这个框架突出异常行为的现实功能与风险后果。

张伯源、陈仲庚提出的标准则从四个角度界定异常。其一是个体经验标准，指当事人的主观痛苦及研究者对异常现象的判断；其二是社会常模与社会适应标准，强调是否偏离社会期待以及是否损害适应；其三是病因与症状存在标准，强调致病因素和临床症状的证据；其四是统计学标准，认为许多心理特征在人群中呈近似常态分布，显著偏离平均水平者可能被视为异常。

Trap

统计学标准 ≠ 诊断标准！ 统计上少见不代表就是病。比如智商 130 的人在统计上也是极端值，但显然不是障碍。反过来说，有些问题（比如考试焦虑）很常见，但只要造成严重痛苦和功能损害，照样可以构成心理障碍。答题时如果看到“偏离常模就是异常”这种说法，基本是错的。

统计学标准只能说明“少见”，不能自动推出“病理”。某种特质即使在人群中罕见，只要不引起痛苦、不损害功能，也不能直接判为障碍。相反，某些明显有害的状态即使并不十分少见，也可能构成心理障碍。单一标准不足以完成判断，临床上通常需要综合主观痛苦、现实检验、危险性、功能损害、社会适应和症状证据等多个方面。

3. 为什么不要求所有标准同时满足

正常与异常之间并不存在一条绝对清晰的单一边界。不同障碍的表现并不相同，有些障碍以主观痛苦为主，有些则以社会功能损害为主，还有些障碍的危险性突出而主观痛苦并不明显。因此，各种判定标准不是必须全部同时满足的必要条件，而是共同构成一个综合判断框架。

例如，精神病性状态的核心在于现实检验能力损害和自知力缺失；某些人格障碍患者未必强烈主诉痛苦，却可能长期在人际和社会功能上严重失调；某些焦虑障碍患者主观痛苦显著，但现实检验仍然完整。由此可见，异常心理的界定是多维度、分层次和依情境而定的。

4. 现行专业实践中的诊断体系

现代专业实践主要依赖医学诊断分类描述体系对心理障碍进行标准化界定，最具代表性的体系是《精神障碍诊断与统计手册》第五版和 ICD-11。它们的作用是提供较为一致的症状描述、诊断分类和临床沟通标准，提高不同专业人员之间诊断的一致性。诊断体系解决的是“如何稳定命名和分类”的问题，并不自动等于“已经完全解释其病因”。异常心理学中的分类、机制研究和治疗研究，仍需在诊断体系之外继续展开。

四、变态心理学历史上的主要解释路径

Intuition

历史这块不要死记硬背年份，要理解“人类怎么一步一步把异常从鬼神手里抢回来”：先是神/魔（神秘主义）→ 然后是身体（体液、脑）→ 再后来是心理（冲突、认知、关系）→ 现在是生物-心理-社会综合模型。记住这个脉络，考论述题就能串起来。

异常心理的历史解释大体经历了由神秘主义、躯体解释到心因性解释的发展过程。这个变化并不是简单的线性替代，而是研究重点逐渐从超自然原因转向躯体病理，再转向心理机制与社会关系。现代异常心理学通常采用更综合的生物—心理—社会视角。

1. 神秘主义

神秘主义阶段将异常心理和行为理解为神灵、鬼魔或超自然力量作用的结果。头盖骨钻洞术是最早期的处理方式之一，其基本假设是通过在颅骨上开孔，可以让附体的邪灵逃出体外。巫师传统则强调通过仪式性手段与神灵沟通，解释并处置异常状态。

在魔鬼附体观中，如果症状被赋予宗教性或神秘性意义，有时会被视为“神灵附体”；更多情况下则被视为“恶魔附体”，对应的干预方式是驱魔。中世纪的巫术观进一步将异常行为与超自然威胁联系起来。这个阶段的共同特点是把异常归因于外在神秘力量，而不是归因于人体、心理或社会过程。

2. 躯体解释

躯体解释的重要代表是希波克拉底的体液病理学说。该理论认为，人体由四种基本体液构成：血液对应空气，性质为热和湿；黄胆汁对应火，性质为热和干；黑胆汁对应土，性质为冷和干；黏液对应水，性质为冷和湿。健康状态取决于四种体液的平衡，异常心理和行为则被看作这种平衡失调的结果。这个理论的重要意义不在于其具体内容是否正确，而在于它首次尝试用自然的、身体性的原因解释异常，而不是诉诸鬼神。

Example

希波克拉底的体液说现在看当然是错的，但考试爱考它的“历史意义”——第一次用自然/身体原因解释异常，摆脱了鬼神说。这是从神秘主义转向躯体解释的关键标志。

在躯体解释占主导的时期，还出现过所谓“冒险式”或“英雄式”治疗，例如放血、冷水浸泡、快速旋转等。这些方法建立在“通过强力身体操作来纠正异常”的观念上，反映出早期医学对病因和干预机制认识的局限。

3. 脑研究与医学模式

脑研究的发展推动了精神病学逐渐走向科学化。加尔提出颅相学，试图把心理功能定位到脑的不同区域。虽然颅相学本身并不科学，但“心理功能具有脑基础”这一方向性判断促进了后来的神经系统研究。19 世纪相继发现言语中枢、感觉中枢和运动中枢，说明心理与行为过程可以与特定脑区活动联系起来。

Trap

颅相学（phrenology）本身不科学，但“心理功能定位于脑”这个方向是对的。考试如果问“颅相学对异常心理学的贡献”，答“推动了心理功能脑定位的研究方向”即可，不要夸它科学。

医学模式的代表人物是克雷珀林。他采用自然科学方法研究精神疾病，明确区分躁狂忧郁性精神病和早发性痴呆，并提出系统的精神疾病分类方法。医学模式的核心是把精神障碍视为可以描述、分类和研究的疾病对象，从而为现代诊断体系奠定基础。

4. 心因性解释的形成

心因性解释强调，异常心理不仅可能由身体病变引起，也可能由心理冲突、认知偏差、学习过程和社会关系失调导致。其思想源头可以追溯到古希腊。苏格拉底的辩证法强调通过追问澄清观念，这种方式后来在认知治疗中发展为苏格拉底式提问或盘诘式质疑。亚里士多德关于“宣泄”的论述，体现了情感表达与心理调节之间的关系。帕拉塞尔苏斯则强调，心理异常应被视为需要治疗的状态，而不是道德罪责或神秘惩罚。

弗洛伊德的精神分析是心因性解释的重要里程碑。精神分析强调潜意识、早期经验和内在冲突在症状形成中的作用，使异常心理的解释从单纯外显行为转向内在心理结构和发展史。

5. “边缘”概念的历史演变

“边缘”概念最初并不指某一种具体人格障碍，而是用来描述一种介于神经症与精神病之间的模糊状态。20 世纪 20 年代，费伦奇首次使用 borderline 一词，指称那些既不符合典型神经症、又未达到典型精神病程度的患者。

Note

钟杰老师的神秘borderline。

这一时期，“边缘性”主要指临床上的过渡地带：患者有时接近神经症水平，有时又出现接近精神病的特征。

1966 年，科恩伯格提出“边缘型人格组织”概念，强调其根源在于早期客体关系发展的缺陷，例如依恋创伤，并将情感爆发、冲动行为和身份感紊乱视为重要表现。在这一理论框架中，边缘性首先是人格组织水平的问题，而不仅仅是一组表面症状。

Intuition

BPO vs BPD 是很多人第一次看会懵的点。简单理解：BPO（边缘型人格组织）是“人格运作水平的连续谱上的一个段位”，神经症性组织 > 边缘型组织 > 精神病性组织；BPD（边缘型人格障碍）是 DSM/ICD 里的一个具体诊断标签。一个人可以有 BPO 的水平但不一定满足 BPD 的诊断，反之确诊 BPD 的一般也处于边缘型人格组织水平。科恩伯格的贡献就是把“症状”和“功能水平”分开了。

人格组织理论进一步把人格功能理解为连续水平：从较高功能的神经症性人格组织，到高边缘性和低边缘性人格组织，再到更低功能的精神病性人格组织。BPO 指的是边缘型人格组织这一更广的功能层级概念，BPD 则是边缘型人格障碍这一具体诊断实体。二者不能混同。前者回答的是“人格在何种组织水平上运作”，后者回答的是“是否符合某种特定障碍的诊断描述”。这一划分避免了把功能水平概念和疾病命名混为一谈。

6. 行为治疗与认知治疗

行为治疗建立在学习理论基础上。

Example

行为治疗的三大学习理论其实是层层递进的：经典条件作用（华生的小阿尔伯特——恐惧可以习得）→ 操作性条件作用（斯金纳的强化/惩罚——行为频率怎么变）→ 观察学习/社会学习（班杜拉——不用亲自被电，看别人就行）。记住这三个实验和核心概念，考试够用了。

  • 经典条件反射说明，原本中性的刺激可以因与无条件刺激反复配对而获得唤起情绪反应的能力；
  • 华生与“小阿尔伯特”实验则展示了恐惧可能通过条件作用被习得。操作性条件反射强调行为结果对行为发生概率的影响。桑代克提出练习律与效果律：刺激—反应联结会因反复使用而增强，因不用而减弱；能带来满意结果的联结会被强化，带来烦恼结果的联结会被削弱。
  • 斯金纳进一步区分正强化、负强化与惩罚。正强化是给予愉快结果以增加行为，负强化是移除厌恶刺激以增加行为，惩罚则是通过不良后果降低行为发生概率。
  • 班杜拉指出，学习并不完全依赖直接强化。观察学习说明，个体可以通过观察他人而获得新的行为方式；自我效能则指个体对自己处理具体情境能力的知觉，它直接影响行为的启动、坚持和应对方式。

认知治疗把异常情绪和行为的核心原因定位于扭曲的思维、信念和图式。其哲学根源包括“人不是被事情困扰，而是被对事情的看法困扰”这一思想，以及苏格拉底式辩证追问的传统。认知治疗认为，人既有理性思考的能力，也有形成扭曲认知的倾向；情绪困扰并非简单由外部事件直接决定，而是受到个体解释方式和基本信念的中介。治疗的目标是识别、检验并修正不合理认知，从而改变情绪与行为。

Note

认知治疗的核心逻辑可以一句话概括：不是事件本身，而是你对事件的看法决定了你的情绪。这就是 ABC 理论的精髓（虽然这里是贝克和埃利斯的版本）。

认知治疗具有若干典型特征：治疗通常限时、短程；会谈结构化，包含作业、讨论与小结；强调契约性的治疗关系，但治疗师具有明确的指导和教育功能；治疗围绕当前问题展开，重视 here and now 的困难；广泛使用苏格拉底式提问；并强调科学研究与疗效评估的重要性。贝克的认知疗法与埃利斯的理性情绪疗法都属于这一大方向，但前者更突出自动思维、核心信念与认知检验，后者更突出非理性信念的论辩与重建。

Trap

贝克 vs 埃利斯不要混！贝克的关键词：自动思维、认知三角（对自己/世界/未来的负性看法）、核心信念、认知检验、作业。埃利斯的关键词：ABC 理论、非理性信念（绝对化、糟糕至极、过度概括）、理性情绪想象、驳斥。如果题干里出现“必须”“应该”“糟糕透了”，偏向埃利斯；出现“自动思维记录表”“证据检验”，偏向贝克。

7. 人本主义

人本主义改变了传统病理取向中把来访者视为“病人”的方式，更强调人的尊严、价值、自主性和自我实现潜能。在这一取向中，来访者不是被动接受矫治的对象，而是具有成长能力的主体。只要环境具备真诚、共情和无条件积极关注等条件，个体就可能朝向更健康、更整合的方向发展。

人本主义重视当下体验，而不把注意力完全放在过去创伤或潜意识冲突上。它认为，治疗性关系本身就是重要的改变力量。罗杰斯提出以来访者为中心的治疗，马斯洛则从需要层次和自我实现的角度理解人格发展。与强调解释、矫正或训练的取向相比，人本主义更强调陪伴、理解与促进。

Intuition

人本主义在考试里经常是“反例”角色——别的流派都在“修 bug”，人本主义说“人本来就有成长的潜能，你要做的是提供好土壤（真诚、共情、无条件积极关注）”。记住罗杰斯的三大核心条件，考多选题经常直接考。

8. 家庭治疗

家庭治疗把个体置于家庭系统中理解。其发展大致经历了从早期家庭介入，到 20 世纪 40 年代的萌芽，再到 50 年代系统家庭治疗的成形，以及 70 年代以后多学派并行发展的过程。它之所以重要，是因为许多症状并不能仅靠个体内部机制充分说明，家庭互动方式、代际传承模式和角色结构都可能在问题形成和维持中起作用。

Example

家庭治疗一个很经典的视角：孩子的“问题行为”有时候是家庭系统失衡的“指示灯”。比如父母婚姻冲突激烈但都不愿面对，孩子突然出问题，全家的注意力就转移到孩子身上了——症状替家庭“扛”了冲突。这不是说孩子装病，而是症状在系统里有了“功能位置”。

家庭系统观认为，个体的问题可能是家庭运作方式的表征。症状并不只是“这个人有病”，也可能是系统失衡的一个信号。治疗因此不仅关注个体症状，也关注家庭规则、沟通模式、联盟关系、代际边界以及家族文化。家庭治疗的目标不只是让某一成员“恢复正常”，而是促成系统层面的改变，因为系统改变往往会反过来改变个体。

家谱图是家庭治疗中的重要工具。它以图示方式记录家庭成员、婚姻与生育关系、代际结构以及重要临床信息。通常用方形表示男性、圆形表示女性，横线表示婚姻或伴侣关系，纵线表示亲子关系，符号上的特殊标记可表示死亡、疾病或某种问题行为。家谱图的价值在于把零散的家庭史转化为结构化信息，使代际传承、症状聚集和关系模式能够被直观识别。

个体治疗与家庭治疗的差别并不只是“多人参与”与“单人参与”的区别。以前者处理抑郁为例，个体治疗通常把重点放在症状、病因、认知与情绪调节以及个人生活史上；家庭治疗则会进一步追问家庭规则、父母与同胞互动、家庭对性别和角色的期待，以及症状在家庭系统中的功能位置。

五、变态心理学的研究方法

Intuition

研究方法这块其实和普心/研究方法有重叠，重点看异常心理学里的“特殊困难”：没法直接测主观体验、状态变化快、多因素交织、伦理限制大。这四条困难是异常心理研究方法的“总纲”，后面的具体方法都是想办法克服这些困难的。

1. 研究异常心理的四大挑战

异常心理现象的研究具有天然困难。第一，许多异常体验无法直接客观测量，只能依赖个体自我报告或他人评估，而这两类信息都可能失真。第二，人的心理状态不断变化，而且变化速度可能很快，研究结果容易受到时间点和情境的影响。第三，多数异常心理现象由多种因素共同导致，任何单一研究都很难同时覆盖所有相关生物、心理和社会变量。第四，研究者往往不能像自然科学实验那样自由操纵关键变量，因为这会涉及伦理问题，不能以制造痛苦或风险的方式进行严格控制实验。

2. 实验法

实验法的核心是通过操纵自变量来观察因变量的变化，并尽可能控制其他无关变量。典型设计包含实验组与对照组，通过比较两组差异推断因果关系。随机分配用于减少组间系统性差异，双盲处理则用于降低实验者期待和被试反应偏差。

Note

双盲在异常心理学研究中尤其重要。你想，如果被试知道自己吃了抗抑郁药，他的期待本身就可能改善症状（安慰剂效应）；如果研究者知道谁吃了真药，他在评估时也可能会不自觉地“看出”疗效。双盲就是要把这两种偏差都摁住。

在量化研究的一般流程中，研究首先要明确问题，然后提出假设，接着设计方法验证假设，之后分析结果，并在此基础上得出结论。实验法的优势在于因果推断能力较强，但在异常心理学中常受伦理和现实条件限制。

3. 单被试实验设计

单被试实验设计适合对个体进行连续追踪和干预评估，其中最常见的是 ABAB 设计。A 阶段是基线阶段，记录目标行为在未干预条件下的自然水平；B 阶段引入治疗或干预，观察行为是否变化；随后撤销干预，进入第二个 A 阶段，检验行为是否回到基线附近；最后再次引入干预，进入第二个 B 阶段，检验行为是否再次改变。若行为变化与干预的引入和撤销同步出现，因果推断就更有力。

Trap

ABAB 设计的关键不是“AB 各来一次”，而是反转（reversal）+ 重复验证。伦理上有个问题：第二个 A 阶段要撤销有效干预，如果目标行为是自我伤害类的，撤掉干预可能有风险。所以 ABAB 更适用于那些撤掉干预后不会带来严重后果的行为问题。

ABAB 设计的关键不在于单次改变，而在于重复验证：如果行为只在干预存在时发生系统变化，就更能说明变化与干预有关，而非偶然波动。

4. 流行病学方法

流行病学方法主要用于描述不同人群中心理障碍发生的频率及其分布。常用手段是问卷调查或结构化会谈，关注某种障碍在总体中有多常见、在不同性别、民族或社会阶层之间如何分布，以及这些分布是否随时间变化。

Example

发病率和患病率是很多同学第一次看会搞混的。你可以这么记：发病率 = 新增病例速度（一段时间内新得了多少人）；患病率 = 池子里现有的总量（某一时点/时期内，得了病的有多少，不管他是昨天得的还是三年前得的）。如果某病病程很短（比如普通感冒），发病率可能高但患病率不高；如果某病慢性迁延（比如糖尿病、精神分裂），患病率就会累积得很高。

其中，发病率指在一定时间内某种障碍的新病例出现频率；患病率指在某一特定时点或时期内，现有患病人数占总人数的比例，其中既包括新病例，也包括旧病例。二者回答的问题不同：发病率强调新增速度，患病率强调现存负担。

流行病学研究还常与双生子研究、领养研究和家族研究结合，用于比较遗传与环境因素的作用。将具有精神病阳性家族史的儿童置于不同养育环境中观察，或比较同卵、异卵双生子的共病情况，都有助于区分遗传风险和环境影响。

5. 案例研究

案例研究是对个体进行深入、细致和长期分析的方法，曾在异常心理学中发挥过极大作用。克雷珀林对精神疾病的区分、弗洛伊德精神分析理论的建构，都与大量案例研究密切相关。案例研究通常收集广泛资料，包括家庭背景、成长史、早期生活环境、重要生活事件、患病史、个人日记和作品，以及家属、朋友、邻居等他人的描述。

Note

案例研究是异常心理学的“老祖母”方法。精神分析几乎就是建立在案例上的。它的优点是资料丰富、能抓独特现象、能启发新理论；缺点是难以推广、重复性差、容易受研究者偏向影响。考试如果问“弗洛伊德/克雷珀林用什么方法”，答“案例研究”准没错。

案例研究的优势在于资料丰富、情境完整，能够捕捉非常独特或少见的个体现象，也容易启发新的理论假设。其局限在于结果难以推广到更大样本，重复性较差，而且受访谈者与个案双方偏差影响较大。个案可能夸大、遗漏或选择性讲述经历，研究者也可能带着既有理论偏好去提问和解释，因此案例研究在解释上需要特别谨慎。

6. 其他研究方法

脑成像技术为异常心理学提供了重要的客观研究手段。其应用大致可分为三类。

Example

脑成像三类问题可以对应三层问题：结构像 MRI 回答“脑子长什么样/哪块大了小了”；功能成像（fMRI/PET）回答“做任务时哪块亮了”；分子影像（PET + 示踪剂）回答“受体/神经递质有多少”。一层比一层深入，也越来越贵。

第一类是形态测量分析，主要利用高分辨率 MRI 对脑结构进行精确测量，用于比较不同群体在脑区体积、皮层厚度或结构完整性上的差异。

第二类是神经功能成像，利用与神经活动相关的信号构建脑活动图像，常见手段包括 MRI 和 PET。这类方法主要回答“在进行某种任务或处于某种状态时，大脑哪些区域更活跃”。

第三类是分子层面的神经影像，通常借助 PET 和放射性示踪剂确定大脑中特定分子的分布和数量，用于研究受体、神经递质系统或代谢过程。它回答的是更接近生物机制层面的问题，而不仅是结构和功能激活的差异。

六、本章的理论指向

变态心理学并不只是关于“异常行为的罗列”或“诊断名称的记忆”。它试图同时回答四类问题：异常现象是什么，正常与异常如何区分，这些现象在历史上被怎样理解，又应当用什么方法研究它们。进一步说，异常心理学最终还会进入更深层的问题：人为什么会出现心理疾患，哪些心理机制是损伤、学习、认知扭曲、关系失调或社会环境作用的结果，又有哪些问题需要放到更长的进化与适应背景中理解。第一章的意义，正在于为这些后续问题建立概念、方法和历史视野的基础。

Summary

最后再画个重点：第一章的论述题高频考点——① 正常与异常的多维判定标准（结合具体案例说明为什么不能用单一标准）；② 几个核心概念的辨析（神经病/精神病/精神障碍/心理障碍）；③ 历史脉络的梳理（神秘主义→躯体→心因→生物-心理-社会）；④ BPO 与 BPD 的区别。名词解释则留意：精神病、精神障碍、发病率/患病率、ABAB 设计、家谱图。
