14.particleFilter

Apr 23, 202616 min read

Summary

这篇文章介绍粒子滤波——当系统不再是线性高斯时，我们如何用一群“粒子”来近似任意形状的后验分布。你会学到重要性采样、序列重要性采样（SIS）、权重退化问题，以及重采样如何把 SIS 升级为实用的 SIR 算法。读完之后，你应该能理解：为什么粒子滤波是处理非线性、非高斯状态估计的强大武器，以及它的高维局限在哪里。

卡尔曼滤波很优雅，但它有个前提：系统必须是线性的，噪声必须是高斯的。可现实世界常常不听话——飞机做机动转弯、金融市场突然跳变、机器人打滑……这些时候，后验分布可能歪七扭八，完全不是漂亮的高斯钟形。粒子滤波的思想就是：既然分布形状千奇百怪，那我们就用一群带权重的“粒子”（随机样本）去近似它！

Intuition

想象你要估计一片云的形状。解析方法试图用数学公式精确描述云的轮廓；而粒子滤波则向天空中发射很多彩色小球，每个小球落在云里的某个位置，并带有一个权重表示“这里云有多浓”。只要小球足够多，你就能大致还原出云的形状——即使这朵云根本不对称、甚至有多个峰。

一、问题背景与基本思想

在状态空间模型中，隐状态 zt​ 满足一阶马尔可夫性，观测 xt​ 只依赖当前状态：

p(zt​∣z1:t−1​)=p(zt​∣zt−1​) p(xt​∣z1:t​,x1:t−1​)=p(xt​∣zt​)

滤波任务是持续估计后验分布 p(zt​∣x1:t​)。粒子滤波不直接求解析表达，而是用 N 个粒子 {zti​}i=1N​ 及其权重 {wti​}i=1N​ 来表示这个分布。

Note

粒子的位置代表“状态可能在哪里”，权重代表“这个状态与观测序列的一致性有多高”。加权粒子的集合，就是我们对后验分布的离散近似。

二、Monte Carlo 近似与重要性采样

对于任意函数 f(z)，如果能直接从目标分布 p(z) 中抽样，期望可以近似为：

E[f(z)]=∫f(z)p(z)dz≈N1​i=1∑N​f(zi)

但很多时候，p(z) 虽然能写出形式，直接采样却很困难。于是引入重要性采样：选一个容易采样的提议分布 q(z)，将期望改写为：

E[f(z)]=∫f(z)q(z)p(z)​q(z)dz≈N1​i=1∑N​f(zi)q(zi)p(zi)​

定义重要性权重：

wi∝q(zi)p(zi)​

归一化后：

w~i=∑j=1N​wjwi​

于是：

E[f(z)]≈i=1∑N​w~if(zi)

Intuition

重要性采样就像“借鸡生蛋”：我们真正关心的是 p(z)，但它太难采样了；于是我们用一个更方便的 q(z) 来采样，再用权重 q(z)p(z)​ 弥补两者之间的差距。如果 q(z) 在某些 p(z) 概率很高的区域采样太少，那些地方的粒子权重就会很大，提醒我们“这里采样不足，但非常重要”。

三、滤波问题中的带权表示

在时序问题中，我们关心的是 p(zt​∣x1:t​)。如果每个时刻独立采样，不仅样本需求量大，而且后验通常难以直接写出。粒子滤波采用对整个状态轨迹递推抽样的方式，用样本路径 {z1:ti​}i=1N​ 近似联合后验 p(z1:t​∣x1:t​)。

未归一化权重写为：

wti​∝q(z1:ti​∣x1:t​)p(z1:ti​∣x1:t​)​

Intuition

这里的关键是“递推”。我们不需要在每个时刻都从头计算整条路径的权重，而是希望在上一时刻权重的基础上，只需乘上一个“增量比值”就能更新。这正是序列重要性采样（SIS）要做的事。

四、序列重要性采样（SIS）

1. 递推形式的来源

SIS 关注的是联合后验 p(z1:t​∣x1:t​)。根据贝叶斯分解与马尔可夫性：

p(z1:t​∣x1:t​)​∝p(x1:t​,z1:t​)=p(xt​∣zt​)p(zt​∣zt−1​)p(x1:t−1​,z1:t−1​)∝p(xt​∣zt​)p(zt​∣zt−1​)p(z1:t−1​∣x1:t−1​)​

Intuition

这个分解非常优美：当前时刻的后验 = 前一时刻后验 × 状态转移概率 × 当前观测似然。状态转移把旧的不确定性“传播”到新时刻，观测似然再用新证据“修正”它。这和卡尔曼滤波的预测-更新框架异曲同工，只是把解析积分换成了采样加权。

2. 提议分布的递推分解

为了匹配分子的结构，提议分布也写成递推形式：

q(z1:t​∣x1:t​)=q(zt​∣z1:t−1​,x1:t​)q(z1:t−1​∣x1:t−1​)

这意味着：我们在时刻 t−1 的已有路径末端，继续“生长”出新的状态分量 zt​。

3. 权重递推公式

将分子分母的递推分解代入权重定义：

wti​​∝q(z1:ti​∣x1:t​)p(z1:ti​∣x1:t​)​∝q(zti​∣z1:t−1i​,x1:t​)q(z1:t−1i​∣x1:t−1​)p(xt​∣zti​)p(zti​∣zt−1i​)p(z1:t−1i​∣x1:t−1​)​=q(zti​∣z1:t−1i​,x1:t​)p(xt​∣zti​)p(zti​∣zt−1i​)​wt−1i​​

这就是 SIS 的核心递推式。

Intuition

当前权重 = 历史权重 × 增量比值。增量比值衡量的是：在当前时刻，真实后验的“新增因素”（状态转移 × 观测似然）与提议分布的“新增因素”之间的匹配程度。如果提议分布恰好就是状态转移先验，那么增量比值就简化为单纯的观测似然 p(xt​∣zti​)。

五、SIS 的基本流程

SIS 可以概括为以下递推过程：

  1. 在时刻 t−1，已拥有粒子路径 {z1:t−1i​} 及其权重 {wt−1i​}。
  2. 在时刻 t，对每个粒子，从提议分布 q(zt​∣z1:t−1​,x1:t​) 中抽样，生成 zti​，接到原路径后形成 z1:ti​。
  3. 利用权重递推公式计算新的未归一化权重 wti​。
  4. 对所有权重归一化，使其满足 ∑i=1N​w~ti​=1。
  5. 用归一化权重与粒子集合近似当前后验分布。

Note

如果只关心当前状态 zt​（而不是整条轨迹），可以只保留每个粒子的最新状态。但如果要做平滑或参数学习，通常需要保留完整路径。

六、权值退化问题

SIS 虽然实现了递推估计，但会出现典型的权值退化现象。随着时间推进，权重通过连乘方式累积，许多粒子的权重会渐渐逼近于 0，只有极少数粒子保持较大权重。

Trap

权值退化是 SIS 的结构性问题，不是偶然的数值 bug。想象你在一个越来越高的维度空间里扔飞镖：维度越高，固定数量的飞镖就越难覆盖目标分布的高概率区域。于是，大部分飞镖都落在了“无关紧要”的区域，权重趋于零；只有少数幸运儿命中靶心，垄断了几乎所有的概率质量。此时，虽然名义上有 N 个粒子，但真正起作用的粒子数量可能只有几个，样本效率急剧下降。

七、重采样思想

为解决权值退化，最常用的方法是重采样。基本思想是：把当前权重视为一个离散概率分布，从现有粒子中按概率重新抽取 N 个样本。权重大者更可能被重复保留，权重很小者更可能被淘汰。重采样后，粒子数不变，但分布会更集中于后验的高概率区域，各粒子权重通常重新置为 N1​。

若记归一化权重为 w~ti​，则重采样相当于依据分布

P(选中第 i 个粒子)=w~ti​

对粒子集合进行有放回抽样。

Intuition

重采样就像是一场“优胜劣汰”的进化：资源（粒子数）有限，我们要把它们分配到最有希望的地方。低权重的粒子被自然淘汰，高权重的粒子被复制繁衍。注意，重采样不会引入新的状态信息，它只是重新分配计算资源。

但凡事有利有弊：重采样会牺牲样本多样性。某些高权重粒子可能被复制很多次，而其他粒子被整体淘汰。所以重采样解决的是“权重过度集中”，而不是“粒子多样性始终充足”。

八、SIR 与基本粒子滤波

在 SIS 的基础上引入重采样，就得到了最基本的粒子滤波框架。若进一步将提议分布选为状态转移先验：

q(zt​∣z1:t−1​,x1:t​)=p(zt​∣zt−1​)

则权重递推式大幅简化为：

wti​∝p(xt​∣zti​)wt−1i​

Intuition

当提议分布等于状态转移先验时，提议分布与状态转移项恰好抵消，只剩下观测似然来区分粒子。这就像一个“生成-测试”循环：先按系统动力学生成候选状态，再用观测似然对这些候选状态打分测试。

若在每一步都结合重采样，该算法通常称为 SIR（Sampling Importance Resampling），也叫 bootstrap 粒子滤波。其流程为：

  1. 依据状态转移模型传播粒子；
  2. 依据观测似然计算权重；
  3. 权重归一化；
  4. 按归一化权重重采样；
  5. 进入下一时刻递推。

Note

SIR 简单、稳定、容易实现，是粒子滤波最具代表性的基础形式。它的弱点在于：如果状态转移先验传播出的粒子与真实观测不太匹配，大量粒子会落在低似然区域，导致权重严重分化。此时，更好的提议分布设计（比如融入当前观测信息）可以显著提升效率。

九、粒子滤波的理解框架

粒子滤波可以从三个层次理解：

第一层：分布表示层。不再用闭式函数表达后验，而是用有限个带权样本进行离散近似。粒子位置 = 可能的状态取值，粒子权重 = 与当前观测序列的一致程度。

第二层：递推推断层。继承了贝叶斯滤波“预测—更新”的思想，但把解析推导换成了采样实现。状态转移负责传播不确定性，观测似然负责重新评估各粒子的可信度。

第三层：数值近似层。重要性采样解决“目标分布难采样”，序列重要性采样解决“时序问题如何递推采样”，重采样解决“权重逐步退化”。三者首尾相接，构成了完整的数值推断机制。

十、适用条件、优势与局限

粒子滤波适用于后验分布形状复杂、无法用均值和协方差充分概括的情形，尤其擅长处理非线性、非高斯、甚至多峰分布的在线状态估计问题。

优势局限
表示能力强，可逼近任意形状的后验计算代价随粒子数增长
无需预先假设后验属于某个简单参数族高维问题中退化严重
概念通用，适用于多种状态空间模型重采样导致样本贫化

Intuition

粒子滤波是一把双刃剑：它用计算资源（粒子数）换取了表示灵活性。只要粒子足够多，它能刻画非常复杂的后验；但维度一高，所需粒子数会指数级增长，这就是著名的“维度诅咒”。因此，粒子滤波在低维状态空间（如 2D/3D 跟踪）中表现优异，但在高维问题（如高维姿态估计）中往往需要更精巧的提议分布和重采样策略。

SIS 与 SIR 已经给出了粒子滤波最核心的思想框架：以采样近似分布，以权重注入观测信息，以重采样维持递推有效性。无论后续如何改进，这三根支柱都不会改变。
