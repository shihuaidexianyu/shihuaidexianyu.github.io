线性代数

Apr 14, 202650 min read

Summary

这份笔记是一场关于”矩阵如何描述形状、变换与不确定性”的漫游。我们会从最基本的转置和逆矩阵出发，一路走到特征分解、SVD、PCA，最后串起多维高斯分布的全部几何直觉。读完后，希望你能明白：协方差矩阵 Σ 不仅仅是一个表格，它是一张描绘数据椭球形状的地图——而读懂这张地图的工具，就藏在正交对角化、行列式、Schur 补这些概念里。

一、矩阵的基本运算规则

设 A,B 为维度相容的矩阵。

1. 转置

转置的定义是把矩阵的行和列交换。常见规则有：

(AT)T=A (A+B)T=AT+BT (cA)T=cAT (AB)T=BTAT

Intuition

你可以把转置想象成”镜子里的倒影”。前三个规则都很自然，但最后一个尤其值得玩味：(AB)T=BTAT 告诉我们，转置会把乘法顺序反过来。这就像你先把衣服穿上身（A），再扣上扣子（B）；如果倒过来看（转置），你得先解开扣子（BT），再脱下衣服（AT）。

2. 逆矩阵

若方阵 A 存在矩阵 A−1，使得

AA−1=A−1A=I

则称 A 可逆。

常用规则：

(A−1)−1=A (AB)−1=B−1A−1 (AT)−1=(A−1)T

Intuition

逆矩阵就像是”撤销操作”。A 把空间拉伸了，A−1 就按原路把它缩回来。注意 (AB)−1=B−1A−1 再一次把顺序反了过来——你先做 A 再做 B，撤销时得先撤销 B 再撤销 A。

矩阵可逆的等价条件很多，最常见的有：

  1. det(A)=0。
  2. A 满秩。
  3. Ax=0 只有零解。
  4. A 的特征值都不为 0。

Note

这四个说法其实是同一件事的不同角度：矩阵没有”压扁”空间（行列式非零），它的列向量之间没有冗余（满秩），它不会把非零向量变成零（只有零解），以及它在每个特征方向上都没有”压垮”（特征值非零）。

3. 正交矩阵

若方阵 Q 满足

QTQ=QQT=I

则称 Q 为正交矩阵。于是：

Q−1=QT

Intuition

正交矩阵是我们最喜欢的矩阵之一，因为它只”旋转”或”反射”空间，绝不拉伸。你可以把它想象成一个刚体的运动——球还是球，只是换了朝向。

正交矩阵表示一种保持长度和夹角不变的线性变换。几何上，它对应旋转、反射，或者它们的组合。

如果向量 x 经过正交变换 y=Qx，则：

∣y∣2=yTy=xTQTQx=xTx=∣x∣2

Example

在二维平面上，Q 可以是一个 30∘ 的旋转矩阵。无论你转多少度，向量的长度始终不变。这就是正交变换的”君子协定”：我改变你的方向，但绝不改变你的大小。

所以正交变换不改变欧氏长度。

二、行列式

1. 行列式的本质

对于一个 n×n 方阵 A，行列式 det(A) 或 ∣A∣ 描述的是该线性变换对体积的放缩倍数。

Intuition

想象你手里有一个单位立方体。矩阵 A 把它拉伸、压缩、倾斜成一个平行多面体。行列式的绝对值，就是这个变形后多面体的体积与原体积的比值。如果行列式为 0，那就意味着立方体被”拍扁”到了更低的维度上——比如三维立方体被压成了一张纸。

  • 若 ∣det(A)∣>1，体积被放大。
  • 若 ∣det(A)∣<1，体积被压缩。
  • 若 det(A)=0，体积被压扁到低维空间，说明矩阵不可逆。
  • 符号的正负还反映定向是否翻转。

在多维高斯里，∣Σ∣1/2 控制的是分布 ” 总体体积尺度 “，因此它出现在归一化常数里。

2. 基本计算规则

det(AB)=det(A)det(B)

Example

这就像我们说的：先做变换 B 体积放大 2 倍，再做变换 A 体积放大 3 倍，总体积当然放大 2×3=6 倍。

det(AT)=det(A)

Intuition

转置只是”换了个角度看”，不会改变体积，所以行列式不变。

det(A−1)=det(A)1​

Intuition

逆变换把体积缩回来，缩放的倍数当然是原倍数的倒数。

若 A 是 n×n 矩阵，c 是标量，则

det(cA)=cndet(A)

Trap

注意这里有个 cn 而不是 c！因为标量 c 作用在矩阵的每一行（或每一列）上，n 维空间里每一维都被拉长了 c 倍，总体积自然是 cn 倍。忘记这个 n 是初学者最常犯的错误之一。

若 A 是三角矩阵或对角矩阵，则行列式等于对角线元素之积：

det(A)=i=1∏n​aii​

Example

对角矩阵对每个坐标轴独立缩放，体积就是各轴缩放倍数的乘积。上三角矩阵虽然会倾斜，但体积只取决于”底乘高”，结果依然是对角线元素的乘积。

这条性质非常实用。

3. 初等变换对行列式的影响

设对矩阵做行变换，则：

  1. 交换两行，行列式变号。
  2. 某一行乘以常数 c，行列式也乘以 c。
  3. 某一行加上另一行的若干倍，行列式不变。

Intuition

这三条规则对应着三种几何操作：交换两行相当于翻转了空间的定向；拉伸某一行就是拉伸某一维；而把一行的倍数加到另一行上，只是”倾斜”了平行多面体，并没有改变它的底和高，所以体积不变。

因此，很多时候会先把矩阵化成上三角矩阵，再求对角线乘积。

4. 低维例子

对于 2×2 矩阵：

A=(ac​bd​)

有

det(A)=ad−bc

Example

在二维平面上，ad−bc 就是由向量 (a,c) 和 (b,d) 张成的平行四边形的面积（带符号）。你可以画个图验证：底乘高最后化简出来就是 ad−bc。

对于 3×3 矩阵，可以按第一行展开：

det(A)=a11​M11​−a12​M12​+a13​M13​

其中 Mij​ 是对应余子式。

5. 用特征值计算行列式

若矩阵 A 可对角化为

A=QΛQ−1

其中

Λ=diag(λ1​,λ2​,⋯,λn​)

则

det(A)=det(Q)det(Λ)det(Q−1)=det(Λ)=i=1∏n​λi​

Intuition

这是对行列式最深刻的理解之一：特征值告诉我们矩阵在每个”特征方向”上拉伸了多少倍。既然这些方向是正交的，总体积的放缩就是各方向放缩倍数的乘积。行列式等于特征值的乘积，这直接把”体积”和”特征方向上的拉伸”联系在了一起。

也就是说，行列式等于全部特征值的乘积。 若 A 是实对称正定矩阵，则所有特征值都为正，因此 det(A)>0。

三、秩、线性无关与可逆性

矩阵的秩 rank(A) 表示矩阵中线性无关行或列的最大个数。

Intuition

秩是矩阵的”真实维度”。一个 3×3 的矩阵，如果它的秩是 2，那说明它虽然住在三维空间里，实际上只是把空间压成了一个二维平面。秩告诉我们：这个矩阵到底有多少”自由度”。

对 m×n 矩阵 A 来说：

  • 列满秩表示列向量线性无关。
  • 行满秩表示行向量线性无关。
  • 若 A 是 n×n 方阵，且 rank(A)=n，则 A 可逆。

这与多维高斯密切相关。若协方差矩阵 Σ 不是满秩的，说明某些维度之间存在严格线性依赖，那么分布实际上被限制在某个低维子空间上，此时 Σ−1 不存在，普通形式的高斯密度也不能直接写出。

Example

想象一个三维高斯分布，如果它的协方差矩阵秩为 2，那所有数据点都像被压扁的煎饼一样贴在一个平面上。这个”煎饼”没有体积，所以行列式为 0，逆矩阵也不存在。

因此在很多推导中会强调：这里考虑 Σ 为正定矩阵，而不只是半正定矩阵。

四、对称矩阵、正定矩阵与协方差矩阵

1. 对称矩阵

若

AT=A

则 A 是对称矩阵。

Intuition

对称矩阵关于主对角线”照镜子”时完全重合。在机器学习中，我们最熟悉的对称矩阵就是协方差矩阵和相关矩阵。

协方差矩阵天然是对称的，因为

Σ=Cov(x)=E[(x−μ)(x−μ)T]

并且

ΣT=Σ

2. 正定与半正定

若对任意非零向量 z 都有

zTAz>0

则称 A 为正定矩阵。

若对任意向量 z 都有

zTAz≥0

则称 A 为半正定矩阵。

Intuition

正定矩阵就像一个”碗”——无论朝哪个方向看，它都向上开口。半正定矩阵则可能是一个”槽”或”平面”，某些方向上不升不降，但绝不会向下开口。

协方差矩阵总是半正定，因为

zTΣz​=zTE[(x−μ)(x−μ)T]z=E[zT(x−μ)(x−μ)Tz]=E[((x−μ)Tz)2]≥0​

Intuition

这个推导很优雅：zTΣz 其实是随机向量在 z 方向上投影的方差（差一个常数）。方差怎么可能为负呢？所以协方差矩阵天然半正定。如果数据在任何非零方向上都有 spread，那它就是正定的。

如果这个量对所有非零 z 都严格大于 0，那么协方差矩阵就是正定的。

3. 正定矩阵的重要性质

若 A 为实对称正定矩阵，则：

  1. 所有特征值都大于 0。
  2. A 一定可逆。
  3. 行列式大于 0。
  4. 可以做 Cholesky 分解。
  5. 可以正交对角化。

Note

这些性质一脉相承：正定意味着所有方向都”向上”，所以没有方向被压扁（特征值>0），体积不为零（行列式>0），逆矩阵存在，而且可以被干净利落地分解成三角矩阵的乘积（Cholesky）。

这些性质使得正定矩阵在高斯分布里极其重要。

五、特征值、特征向量与特征分解

1. 定义

若存在非零向量 v 和标量 λ，使得

Av=λv

则称 λ 为矩阵 A 的特征值，v 为对应特征向量。

Intuition

特征向量是矩阵的”特权方向”——当矩阵作用在这些方向上时，向量不会被扭转到别的方向，只会被拉长或缩短。特征值 λ 就是拉伸的倍数。如果 λ=2，那说明矩阵在这个方向上把一切都放大了一倍；如果 λ=−1，那就是反方向翻转并保持长度不变。

特征值描述的是：在线性变换 A 作用下，有些特殊方向不会改变方向，只会被伸缩。

2. 特征多项式

特征值由方程

det(A−λI)=0

求出。

Intuition

这个方程在问：是否存在某个方向，使得 A 在这个方向上的作用等价于纯粹的缩放 λI？如果 A−λI 把空间压扁了（行列式为零），那就意味着存在非零向量被它”吞掉”了，也就是 Av=λv 成立。行列式与特征值的关系，又一次通过”体积是否为零”联系了起来。

这说明行列式与特征值之间关系非常紧密。

3. 特征分解

若矩阵 A 有一组线性无关的特征向量 v1​,⋯,vn​，令

P=(v1​,v2​,⋯,vn​) Λ=diag(λ1​,λ2​,⋯,λn​)

则

A=PΛP−1

Intuition

特征分解告诉我们：如果我们站在特征向量构成的坐标系里看 A，它就是一个简单的对角矩阵 Λ——只在各个坐标轴上做伸缩，没有任何”扭结”。P 是从特征坐标系回到原坐标系的变换，P−1 则是反过来。

这称为特征分解。 但一般矩阵不一定能这样分解。实对称矩阵是最好的情况，它不仅可分解，而且可以用正交矩阵来分解。

Note

一般矩阵可能没有足够的线性无关特征向量（比如 Jordan 块），或者特征向量不正交。但对于实对称矩阵，这些烦恼统统不存在——它总有 n 个互相正交的特征向量，而且特征值都是实数。

六、实对称矩阵可正交对角化

这是多维高斯里最关键的矩阵结论之一。

1. 定理

若 A 是实对称矩阵，则存在正交矩阵 Q 和实对角矩阵 Λ，使得

A=QΛQT

其中 Λ 的对角线元素是 A 的特征值，Q 的列向量是单位正交特征向量。

Intuition

这是特征分解的”豪华版”：不仅坐标系是特征向量的坐标系，而且这个坐标系的基向量还是互相垂直、长度为一的标准正交基。QT=Q−1 让求逆变得异常简单。

这称为实对称矩阵的正交对角化。

2. 为什么重要

因为一旦正交对角化成立，很多运算就瞬间变简单了。 若

A=QΛQT

则

A−1=QΛ−1QT

Example

求逆本来很麻烦，但现在只需要把对角线上的特征值取倒数就行！这就像你只需要把各个方向上的伸缩倍数取倒数，旋转部分 Q 和 QT 完全不动。

前提是所有特征值非零。 并且

Ak=QΛkQT

Example

矩阵的 k 次幂，现在只是每个特征值的 k 次幂。这在解递推关系、图论的随机游走、以及马尔可夫链的稳态分析中都非常实用。

det(A)=i=1∏n​λi​

Intuition

我们又见到了这个美丽的公式：行列式 = 特征值的乘积。对于正交对角化，这个结论几乎是显然的，因为 Q 是旋转（行列式为 ±1），体积放缩完全由 Λ 负责。

若 A 正定，则

λi​>0

所以很多关于 A 的问题都可以转化成对标量 λi​ 的问题。

3. 与协方差矩阵的关系

由于协方差矩阵 Σ 是实对称矩阵，所以一定可以写成

Σ=QΛQT

Intuition

这意味着：无论你的数据有多复杂，总存在一组互相垂直的方向（Q 的列），在这些方向上数据的波动是完全独立的。协方差矩阵在这些方向上只表现为简单的伸缩 Λ，不再有不同坐标之间的”纠缠”。

这意味着：协方差矩阵总能找到一组互相正交的主轴方向，在这些方向上，协方差矩阵只表现为简单的伸缩，而不再混合不同坐标。 这正是多维高斯椭球结构的来源。

Example

想象一团三维高斯云，形状像一个橄榄球。橄榄球的长轴、中轴、短轴就是它的三个主轴方向。沿着长轴，数据拉开得最远；沿着短轴，数据最集中。Λ 里的特征值就分别对应这三个轴上的方差。

七、二次型

1. 定义

形如

xTAx

的标量称为二次型。

Intuition

二次型可以看作一个”曲面的高度函数”。给定一个点 x，二次型告诉你这个点在一个以 A 为参数的”碗”或”马鞍”上的高度是多少。

若 A 是 n×n 矩阵，x∈Rn，那么 xTAx 的结果是一个标量。

2. 二次型只由对称部分决定

任意矩阵 A 都可以写成

A=2A+AT​+2A−AT​

即 ” 对称部分 + 反对称部分 ”。 而对任意实向量 x，有

xTAx=xT(2A+AT​)x

Intuition

反对称部分就像一个”旋转”，它只会让 x 在一个与 x 垂直的方向上偏移，因此 x 与偏移后的向量的内积为零。所以二次型只”感受”到对称部分，完全无视反对称部分。

因为反对称部分对二次型没有贡献。 所以研究二次型时，只需研究实对称矩阵。

3. 二次型与正交对角化

若 A 是实对称矩阵，且

A=QΛQT

令

y=QTx

则

xTAx=xTQΛQTx=yTΛy=i=1∑n​λi​yi2​

Intuition

这是整个笔记里最美妙的等式之一。它说：任何一个实对称二次型，只要换到特征向量的坐标系里，就变成了若干个独立平方项的和。复杂的交叉项 xi​xj​ 全部消失了，只剩下”每个坐标方向上的抛物线高度之和”。

这一步非常重要，它说明任意实对称二次型都可以通过正交变换化成若干平方项之和。

4. 几何意义

式子

xTAx=c

在不同情况下对应不同几何对象。

  • 若 A 正定，则这是椭球面。
  • 若 A 不定，则可能对应双曲面等。
  • 若 A 半正定，则可能退化。

Example

当 A=I 时，xTx=c 就是一个标准的球面。当 A 是对角矩阵 diag(1,4,9) 时，它就变成了一个椭球，三个半轴长度分别是 1,1/2,1/3。特征值越大，那个方向上半轴越短——因为要在更短的距离内达到相同的二次型值 c。

多维高斯里指数项正是一个二次型：

(x−μ)TΣ−1(x−μ)

由于 Σ−1 是正定对称矩阵，所以其等值面一定是同心椭球。

Intuition

这也是为什么高斯分布的等高线永远是椭球：二次型的等值面就是椭球，而高斯密度沿着这些椭球衰减。μ 是椭球的中心，Σ−1 决定了椭球的”胖瘦”和”朝向”。

九、马氏距离与多维高斯中的椭球

若

Σ=QΛQT

则

Σ−1=QΛ−1QT

于是

Δ=(x−μ)TΣ−1(x−μ)

令

y=QT(x−μ)

则

Δ=yTΛ−1y=i=1∑p​λi​yi2​​

Intuition

这个推导把马氏距离剥得干干净净：它不过是先把向量 x−μ 旋转到主轴坐标系（y=QT(x−μ)），然后在每个方向上按方差”标准化”（除以 λi​），最后求平方和。方差大的方向，同样的偏移被”打折”；方差小的方向，同样的偏移被”加价”。

这说明：

  1. yi​ 是点 x−μ 在主轴方向上的坐标。
  2. 每个方向的距离都要除以对应方差 λi​。
  3. 方差越大，该方向允许的波动越大；方差越小，该方向偏离一点就会被惩罚得更重。

Example

假设你测量身高和体重。如果身高的方差是 100cm2，体重的方差是 25kg2，那么一个 10cm 的身高偏差在马氏距离里只算 1 分（因为 102/100=1），而一个 10kg 的体重偏差却算 4 分（因为 102/25=4）。马氏距离会自动根据各方向的”自然波动幅度”来调整惩罚力度。

因此，马氏距离不是普通欧氏距离，而是 ” 按协方差缩放后的距离 ”。 对应的等值面

i=1∑p​λi​yi2​​=c

就是椭球面，半轴长度与 λi​​ 成正比。

Note

这个椭球方程和标准的 a2x2​+b2y2​+c2z2​=1 完全是一个模样。只不过这里的半轴长度是 cλi​​，而方向由 Q 的列给出。

十、行列式在多维高斯中的作用

高斯密度中的归一化常数有一项

∣Σ∣1/2

若

Σ=QΛQT

则

∣Σ∣=i=1∏p​λi​

于是

∣Σ∣1/2=i=1∏p​λi​​

Intuition

各主轴上的尺度分别是 λi​​，把它们乘起来就是椭球的”体积尺度”。行列式的平方根 ∣Σ∣1/2 就是这个体积尺度。分布铺得越开，体积越大，∣Σ∣1/2 就越大，为了归一化（总面积保持为1），密度峰值就必须越低。

这表示各主轴尺度的乘积，也就是椭球 ” 体积尺度 ” 的量。

直观上：

  • Σ 越大，分布越分散，峰值越低。
  • Σ 越小，分布越集中，峰值越高。

所以行列式控制的不是某一条轴，而是整体扩张程度。

Example

想象一维高斯，它的”体积”就是宽度。二维高斯的”体积”是椭球面积，三维是椭球体积。无论多少维，∣Σ∣1/2 都在告诉我们这个高斯云团”占据了多少空间”。

十一、矩阵求逆的规则与常见技巧

1. 基本规则

(AB)−1=B−1A−1 (AT)−1=(A−1)T (QΛQT)−1=QΛ−1QT

Intuition

最后一个公式是正交对角化的”福利”：因为 Q−1=QT，求逆时旋转部分相互抵消，只剩下把特征值取倒数。但切记，这要求所有特征值都非零，也就是矩阵可逆。

其中最后一个公式只在 Λ 对角元都非零时成立。

2. 对角矩阵的逆

若

D=diag(d1​,d2​,⋯,dn​)

且 di​=0，则

D−1=diag(d1​1​,d2​1​,⋯,dn​1​)

Example

对角矩阵是最简单的矩阵之一：每个坐标轴独立缩放，求逆就是每个缩放倍数取倒数。如果一个房间的长宽高分别被放大了 2,3,4 倍，要把它变回原样，只需要分别缩小 1/2,1/3,1/4。

因此，一旦把矩阵化成对角形式，求逆就非常容易。

3. 分块矩阵的逆

对于

M=(AC​BD​)

若 A 可逆，定义 Schur 补

S=D−CA−1B

则当 S 也可逆时，

M−1=(A−1+A−1BS−1CA−1−S−1CA−1​−A−1BS−1S−1​)

Intuition

这个公式看起来吓人，但其实有章可循。Schur 补 S 就像是”消去 A 的影响后，D 的有效部分”。右下角的块直接是 S−1，左上角则是在 A−1 的基础上修正了 B 和 C 带来的耦合。如果把 B=C=0（两块完全独立），公式立刻退化为简单的分块对角求逆：diag(A−1,D−1)。

类似地，若 D 可逆，可定义另一个 Schur 补

T=A−BD−1C

并得到对应公式。

这个结果在条件高斯分布里非常重要，因为条件协方差恰好就是某种 Schur 补。

4. 不要误记的错误公式

下面这个式子一般不成立：

(A+B)−1=A−1+B−1

Trap

这是线性代数里最经典的”想当然”错误。矩阵求逆不像数的倒数那样有”和的倒数等于倒数的和”的性质。如果你记不住为什么，至少记住：看到 (A+B)−1 时，千万不要手贱把它拆开！

这点非常容易混淆。

5. 数值计算上的实际建议

在实际算法里，通常不建议显式计算 Σ−1，而是更倾向于解线性方程组。原因是：

  1. 显式求逆通常数值稳定性更差。
  2. 计算成本也往往更高。
  3. 对正定矩阵，用 Cholesky 分解通常更稳。

Note

在实际编程时，如果你需要求 Σ−1b，更好的做法是调用 solve(\Sigma, b) 而不是先算 inv(\Sigma) 再乘 b。这就像我们解方程 3x=6 时直接说 x=2，而不是先算 1/3 再乘以 6。

十二、Cholesky 分解

若 A 是实对称正定矩阵，则存在唯一的下三角矩阵 L，使得

A=LLT

这称为 Cholesky 分解。

Intuition

Cholesky 分解就像是给正定矩阵开平方根。普通数 a>0 可以写成 (a​)2，正定矩阵 A 可以写成 LLT。而且 L 还是下三角的，这让计算特别高效。

它有几个非常重要的用途：

1. 快速求行列式

因为 L 是下三角矩阵，所以

det(A)=det(L)det(LT)=det(L)2

而

det(L)=i=1∏n​lii​

因此

det(A)=(i=1∏n​lii​)2

Example

下三角矩阵的行列式就是对角线元素的乘积，所以 det(L) 一算就有，平方一下就得到 det(A)。这比直接对 A 做 LU 分解还要快。

2. 快速解线性方程组

若要求解

Ax=b

且 A=LLT，则只需先解

Ly=b

再解

LTx=y

Intuition

解三角方程组非常简单：从第一个方程直接读出 y1​，代入第二个方程读 y2​，依此类推（前向代入）。解 LTx=y 则是从最后一个方程倒着来（后向代入）。两次简单的代入，避免了直接求逆的数值不稳定。

这样比直接求逆更稳定。

3. 用于高斯采样和密度计算

若 z∼N(0,I)，令

x=μ+Lz

则

x∼N(μ,LLT)=N(μ,Σ)

Example

想从一个复杂的高斯分布 N(μ,Σ) 采样？不需要直接处理 Σ，只要从标准正态 z 采样，再用 x=μ+Lz 变换一下就行。这是生成模型和高斯过程里最常用的采样技巧之一。

所以 Cholesky 分解是从多维高斯采样的经典方法。

十三、奇异值分解 SVD

1. 定义

对任意 m×n 实矩阵 A，都存在正交矩阵 U、V 和非负对角矩阵 Σ，使得

A=UΣVT

这称为奇异值分解。

Intuition

SVD 是线性代数里的”瑞士军刀”。它说：任何矩阵，无论是不是方阵，无论是否可逆，都可以分解为”旋转 → 拉伸 → 再旋转”三步。这是比特征分解更普适的存在。

其中：

  • U 的列向量叫左奇异向量。
  • V 的列向量叫右奇异向量。
  • Σ 的对角元 σ1​,σ2​,⋯ 叫奇异值，且通常按从大到小排列。

2. 几何意义

SVD 可以理解为三步变换：

  1. 先用 VT 做一次正交变换。
  2. 再用 Σ 在各坐标轴方向上做缩放。
  3. 最后用 U 再做一次正交变换。

Example

想象你把一张照片上传到电脑，照片被压缩、旋转、再拉伸。SVD 就是说，任何线性变换都可以拆解成这种”先转一转，再各方向压一压，最后再转一转”的过程。没有扭结，没有弯曲，就是简单的旋转加伸缩。

所以它本质上是 ” 旋转（或反射）+ 拉伸 + 旋转（或反射）“。

3. 与特征值分解的关系

由

A=UΣVT

可得

ATA=VΣ2VT AAT=UΣ2UT

Intuition

ATA 和 AAT 都是对称矩阵，所以它们有漂亮的特征分解。SVD 的右奇异向量 V 就是 ATA 的特征向量，左奇异向量 U 就是 AAT 的特征向量。而奇异值 σi​ 就是特征值 λi​ 的平方根（这里 λi​ 是 ATA 的特征值）。

也就是说：

  • ATA 的特征向量是 V 的列。
  • AAT 的特征向量是 U 的列。
  • 特征值是奇异值的平方。

所以 SVD 是对一般矩阵最普适的分解，而特征值分解更像是对方阵，尤其是对称方阵的特殊情形。

4. 与秩的关系

非零奇异值的个数等于矩阵的秩：

rank(A)=∣{i:σi​>0}∣

Intuition

奇异值为零意味着在那个方向上没有”拉伸”，也就是矩阵把那个方向压扁了。非零奇异值的个数，就是矩阵真正保留的维度数，也就是秩。

5. 与逆矩阵的关系

若 A 为可逆方阵，且

A=UΣVT

则

A−1=VΣ−1UT

Example

这和正交对角化 A=QΛQT 时的 A−1=QΛ−1QT 如出一辙。旋转部分 U 和 V 求逆时相互抵消，只剩下把奇异值取倒数。

这与正交对角化形式非常相似，但适用范围更广。

6. 伪逆

若矩阵不可逆，可以用 Moore-Penrose 伪逆：

A+=VΣ+UT

其中 Σ+ 是把非零奇异值取倒数后再转置位置得到的矩阵。

Intuition

伪逆是逆矩阵的”最佳替代”。对于可逆矩阵，伪逆就是真逆；对于不可逆或长方矩阵，伪逆给出最小二乘意义下的最优解。线性回归里最常见的 β^​=(XTX)−1XTy 在 XTX 不可逆时，就可以用伪逆 (XTX)+XTy 来代替。

伪逆在线性回归、最小二乘、降维里都很常见。

十四、PCA：主成分分析

1. PCA 想解决什么问题

高维数据通常不是在所有方向上都同样重要。有些方向变化很大，包含大量信息；有些方向变化很小，更多是噪声。PCA 的目标就是找到数据方差最大的几个方向，用更低维的表示保留主要信息。

Intuition

想象你站在山顶俯瞰一群羊在山坡上吃草。羊群主要沿着山坡的长轴方向分布，而在垂直于山坡的方向上几乎没有分布。PCA 就是帮你找到这个”山坡长轴”——数据变化最大的方向。

2. 协方差矩阵视角

设中心化后的数据矩阵为

X∈Rn×p

其中每一行是一个样本，每一列是一个特征，并且每列都已减去样本均值。

Note

“中心化”是 PCA 不可或缺的一步。如果不先减去均值，PCA 找到的方向可能只是指向数据的整体偏移，而不是数据内部的波动模式。

则样本协方差矩阵可写成

S=n1​XTX

或有时写作

S=n−11​XTX

差别只在无偏估计与否，不影响主方向。

对 S 做特征分解：

S=QΛQT

则：

  • Q 的列向量就是主成分方向。
  • Λ 的对角元就是各方向上的方差大小。

Intuition

协方差矩阵 S 描述了数据在各个方向上”展开”的程度。对它的特征分解，恰好把数据最”展开”的方向（最大特征值对应的特征向量）找了出来。这些方向彼此正交，依次解释了数据中从多到少的方差。

因此 PCA 本质上就是对协方差矩阵做正交对角化。

3. SVD 视角

若对中心化后的数据矩阵直接做 SVD：

X=UΣVT

则

XTX=VΣ2VT

因此：

S=n1​VΣ2VT

Intuition

这揭示了 PCA 和 SVD 之间深刻而简洁的联系：PCA 的主成分方向就是 SVD 的右奇异向量 V，而各主成分的方差与奇异值的平方成正比。所以，如果你会算 SVD，你就已经会算 PCA 了。

这说明：

  1. PCA 的主方向就是 V 的列向量。
  2. 各主方向上的方差由 Σ2/n 给出。

所以 PCA 与 SVD 是紧密联系的，很多实际实现里直接用 SVD 做 PCA，因为数值更稳。

4. PCA 与多维高斯的联系

若数据近似服从高斯分布：

x∼N(μ,Σ)

对 Σ 做特征分解：

Σ=QΛQT

则主轴方向就是 PCA 的方向，特征值就是各方向上的方差大小。因此：

  • PCA 告诉我们高斯椭球沿哪些方向拉得最长。
  • 特征值告诉我们每个方向拉伸了多少。

Example

如果你有一团二维高斯云，形状像个细长的雪茄，PCA 的第一主成分就会沿着雪茄的长轴方向，第二主成分垂直于它。特征值则分别告诉我们：沿着长轴数据有多分散，沿着短轴有多集中。

这就是为什么多维高斯、协方差矩阵、PCA 经常一起出现。

十五、白化变换

白化是理解马氏距离和高斯标准化的一个特别好的工具。

若

Σ=QΛQT

定义

z=Λ−1/2QT(x−μ)

则可以验证：

E[z]=0 Cov(z)=I

Intuition

白化变换分两步：首先 QT 把数据旋转到主轴坐标系（去掉了坐标之间的相关性），然后 Λ−1/2 把每个方向按标准差缩放（让各方向的方差都变成 1）。经过这样”洗牌”之后，原本歪歪扭扭、胖瘦不均的高斯椭球，变成了一个标准的球形高斯。

也就是说，原本具有协方差 Σ 的随机变量，被变换成了协方差为单位阵的变量。

此时马氏距离就变成普通欧氏距离：

(x−μ)TΣ−1(x−μ)=zTz=∣z∣2

Example

马氏距离其实就是在”白化后的世界”里计算欧氏距离。你先花点功夫把数据标准化成球形，之后就可以放心地用熟悉的欧氏距离来衡量远近了。

所以马氏距离本质上就是：先把数据旋转到主轴坐标系，再把每个方向按标准差归一化，然后再算欧氏距离。

十六、分块矩阵与 Schur 补

在多维高斯条件分布中，经常把随机向量分块：

x=(xa​xb​​),μ=(μa​μb​​),Σ=(Σaa​Σba​​Σab​Σbb​​)

这时 Schur 补就会自然出现。

1. 定义

若 Σaa​ 可逆，则关于左上角块的 Schur 补为

Σbb⋅a​=Σbb​−Σba​Σaa−1​Σab​

若 Σbb​ 可逆，则关于右下角块的 Schur 补为

Σaa⋅b​=Σaa​−Σab​Σbb−1​Σba​

2. 几何与统计意义

Schur 补表示：在消去另一块变量影响之后，剩余部分的 ” 有效协方差 ”。

Intuition

你可以把 Schur 补理解为”已知一部分信息后，对另一部分不确定性的修正”。Σba​Σaa−1​Σab​ 表示 xb​ 中能被 xa​ 解释掉的部分。从总协方差 Σbb​ 里扣掉这部分，剩下的就是 xb​ 真正的”残余不确定性”。

在高斯条件分布中，它正好就是条件协方差：

Cov(xb​∣xa​)=Σbb​−Σba​Σaa−1​Σab​ Cov(xa​∣xb​)=Σaa​−Σab​Σbb−1​Σba​

3. 与行列式的关系

对于分块矩阵

M=(AC​BD​)

若 A 可逆，则

det(M)=det(A)det(D−CA−1B)

若 D 可逆，则

det(M)=det(D)det(A−BD−1C)

Intuition

这和分块矩阵求逆是配套的：行列式也可以”分块计算”。Schur 补 D−CA−1B 或 A−BD−1C 恰好补全了缺失的那一块信息。在概率图模型里，这个公式经常用于计算边缘似然。

这说明 Schur 补不仅出现在求逆里，也出现在行列式计算里。

十七、线性变换下的高斯分布

这里顺带把你原文中的一个写法校正一下。严格地说，不是写 "y∼Ax+b"，而是写

y=Ax+b

或者更一般地写

y=Ax+b+ϵ

若

x∼N(μ,Σ)

且

y=Ax+b

则

y∼N(Aμ+b,AΣAT)

Intuition

高斯分布有一个令人安心的性质：经过任何线性变换后，它仍然是高斯分布。只不过均值被线性搬动了（Aμ+b），协方差也被线性”拉扯”了（AΣAT）。这就像一团橡皮泥被均匀拉伸，形状还是那类形状，只是变大了或变扁了。

这是因为：

E[y]=AE[x]+b=Aμ+b Cov(y)=Cov(Ax+b)=ACov(x)AT=AΣAT

这个定理非常重要，因为它告诉我们：高斯分布在线性变换下仍然是高斯分布。

十八、为什么条件高斯公式长成那样

若

(xa​xb​​)∼N((μa​μb​​),(Σaa​Σba​​Σab​Σbb​​))

则

xb​∣xa​∼N(μb​+Σba​Σaa−1​(xa​−μa​),Σbb​−Σba​Σaa−1​Σab​)

这里有两部分非常值得理解：

1. 条件均值为什么是线性的

因为联合高斯有一个非常特殊的性质：条件期望一定是观测变量的线性函数。也就是说，已知 xa​ 后，对 xb​ 的最优预测是一个线性修正项：

μb​+Σba​Σaa−1​(xa​−μa​)

Intuition

这个公式在说：我对 xb​ 的初始猜测是 μb​。当我观察到 xa​ 偏离了它的均值 μa​ 时，我需要根据 xa​ 和 xb​ 的相关性（Σba​）来调整预测。Σaa−1​ 则是把 xa​ 的观测值”标准化”，避免被 xa​ 自身的大方差误导。

其中：

  • Σba​ 描述两部分变量如何耦合。
  • Σaa−1​ 负责把 xa​ 的偏移 ” 标准化 “。

2. 条件协方差为什么变小

条件协方差是

Σbb​−Σba​Σaa−1​Σab​

Intuition

知道了 xa​ 之后，xb​ 中能被 xa​ “解释”的部分就被减掉了。剩下的 Σbb​−Σba​Σaa−1​Σab​ 就是 xb​ 的”残余不确定性”。因为减掉了一些，所以条件协方差一定小于等于原来的边缘协方差 Σbb​。信息不会增加不确定性。

这意味着：知道了 xa​ 之后，关于 xb​ 的不确定性会减少。减少的部分就是由相关性解释掉的那一部分。

这也是 Schur 补在概率里的直观意义。

十九、线性高斯模型里的矩阵结构

你最后写的模型是：

p(x)=N(μ,Λ−1) p(y∣x)=N(Ax+b,L−1)

这里有两个特别值得注意的矩阵点。

1. 精度矩阵

你这里用的是 Λ−1 和 L−1 作为协方差，所以 Λ 和 L 本身称为精度矩阵。

Intuition

精度矩阵是协方差矩阵的逆。如果协方差矩阵告诉我们”变量有多不确定”，精度矩阵就告诉我们”变量有多确定”。精度矩阵越大（特征值越大），不确定性越小。

精度矩阵的好处在于：

  1. 它直接出现在指数项中。
  2. 多个高斯相乘时，精度矩阵往往可以直接相加。
  3. 在图模型和贝叶斯推断里特别方便。

Example

在卡尔曼滤波或高斯信念传播中，经常需要把多个高斯分布相乘。如果用协方差形式，需要多次求逆；如果用精度形式，只需要把精度矩阵和精度加权后的均值向量相加，计算简洁得多。

2. 边缘分布

由

y=Ax+b+ϵ,ϵ∼N(0,L−1)

可得

p(y)=N(Aμ+b,AΛ−1AT+L−1)

Intuition

边缘分布的协方差有两部分贡献：AΛ−1AT 来自隐变量 x 的不确定性经过线性变换传递到观测空间；L−1 来自观测噪声本身。两者相加，因为这两种不确定性是独立的。

这说明协方差传播遵循一个很清晰的结构：

  • 先验不确定性 Λ−1 经由线性变换 A 传到观测空间，变成 AΛ−1AT。
  • 再叠加观测噪声协方差 L−1。

3. 后验分布

后验均值和协方差写成

E[x∣y]=μ+Λ−1AT(L−1+AΛ−1AT)−1(y−Aμ−b) Var(x∣y)=Λ−1−Λ−1AT(L−1+AΛ−1AT)−1AΛ−1

Intuition

这两个公式看起来很复杂，但结构非常优美。后验均值 = 先验均值 + 修正项；修正项的大小取决于观测残差 (y−Aμ−b) 和卡尔曼增益 Λ−1AT(…)−1。后验协方差 = 先验协方差 - 被观测信息”削掉”的部分。这完全符合我们的直觉：观测让我们更确定，所以协方差变小了。

它们的结构本质上仍然来自分块协方差矩阵和 Schur 补。

也就是说，这些看起来复杂的公式，底层其实就是两件事：

  1. 线性变换下高斯仍然是高斯。
  2. 条件高斯由分块矩阵和 Schur 补给出。

二十、把这些知识串起来看

现在把所有内容连起来，多维高斯的矩阵结构就会很清楚。

1. 协方差矩阵为什么一定对称

因为它来自

E[(x−μ)(x−μ)T]

所以天然对称。

2. 为什么通常要求协方差矩阵正定

因为高斯密度里要用到 Σ−1 和 ∣Σ∣。只有正定时：

  1. 逆矩阵存在。
  2. 行列式为正。
  3. 二次型严格大于零。
  4. 等值面是正常的椭球而不是退化对象。

3. 为什么正交对角化特别关键

因为它把复杂的矩阵问题化成按主轴分解的标量问题：

Σ=QΛQT

于是

Σ−1=QΛ−1QT

并且

(x−μ)TΣ−1(x−μ)=i=1∑p​λi​yi2​​

这让马氏距离、椭球形状、主方向都一目了然。

4. 为什么行列式会出现在归一化常数中

因为它描述总体体积放缩：

∣Σ∣1/2=i=1∏p​λi​​

所以它控制分布”铺得有多开”。

5. 为什么 PCA、SVD 会和高斯一起出现

因为 PCA 就是在找协方差矩阵的主轴，而高斯的椭球主轴也正由协方差矩阵决定；SVD 则是 PCA 的数值实现和更一般的矩阵分解工具。

二十二、最后的理解框架

如果只用一句话概括这份笔记，那么可以这样说：

多维高斯分布的全部几何结构，都藏在协方差矩阵 Σ 里；而理解 Σ 的关键，就是理解对称矩阵、正定矩阵、正交对角化、二次型、行列式、分块矩阵、SVD 与 PCA。

你可以把它们之间的关系记成下面这个链条：

协方差矩阵 Σ → 对称正定 → 可正交对角化 → 得到主轴方向与方差大小 → 二次型变成主轴坐标下的平方和 → 马氏距离与椭球结构清晰 → 行列式给出总体体积尺度 → 分块矩阵与 Schur 补给出条件高斯 → SVD 与 PCA 进一步解释数据的主方向和降维结构
