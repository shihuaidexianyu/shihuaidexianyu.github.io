降维

维度灾难与降维的基本思想

降维是处理高维数据的重要方法，其目标并不只是压缩变量个数，而是在尽量保留关键信息的前提下，用更低维的表示描述数据的主要结构。高维情形下，样本分布会迅速变得稀疏，许多基于距离、邻域和统计稳定性的分析方法都会受到影响，这一现象通常称为维度灾难。过拟合问题虽然可以通过正则化、增加数据量等方式缓解，但当特征空间本身过高且冗余明显时，降维往往是更直接的处理路径。

设半径为 R 的 n 维球体体积为

Vn​(R)=Γ(2n​+1)πn/2​Rn=Cn​Rn

其中 Cn​=Γ(2n​+1)πn/2​ 只与维数 n 有关。边长为 2R 的 n 维超立方体体积为 (2R)n，因此球体积与同尺度超立方体体积之比为

(2R)nVn​(R)​=Γ(2n​+1)2nπn/2​

当 n→∞ 时，上式趋于 0。这表明随着维数升高，即使球体与立方体具有相同的尺度参数，球体在整个空间中所占比例也会迅速减小。直观上看，高维空间中的大量区域集中在“边缘”，样本之间的相对距离差异被压缩，数据表现出显著稀疏性。

降维方法大体可以分为三类。第一类是直接降维，即特征选择，其思想是在原始特征中筛选出更有代表性的部分变量，而不改变变量本身的含义。第二类是线性降维，以主成分分析为代表，其思想是在原始空间中寻找一个低维线性子空间，并将样本投影到该子空间上。第三类是非线性降维或流形学习，其思想是认为数据虽然处于高维观测空间，但本质上分布在某个低维流形附近，因此需要用非线性方式恢复其内在结构。

数据中心化与协方差矩阵的矩阵形式

设数据集共有 N 个样本，每个样本是 p 维向量，记数据矩阵为

X=[x1T​ x2T​ ⋮ xNT​​]∈RN×p

其中 xi​∈Rp 表示第 i 个样本。样本均值向量为

xˉ=N1​i=1∑N​xi​

样本协方差矩阵写为

S=N1​i=1∑N​(xi​−xˉ)(xi​−xˉ)T

协方差矩阵描述了各维度围绕均值的联合波动情况，是线性降维方法中最核心的二阶统计量。

为了把中心化写成统一的矩阵运算，引入全 1 向量 1∈RN 和中心矩阵

H=IN​−N1​11T

其中 IN​ 是 N 阶单位矩阵。中心化后的数据矩阵为

Xc​=HX

因为 H 满足

HT=H

以及

H2=H

所以 H 既是对称矩阵，又是幂等矩阵，因此它是一个投影矩阵，表示把数据投影到“去均值”的子空间中。利用这一性质，协方差矩阵可以写成

S=N1​XcT​Xc​=N1​XTHX

这一表达式有两层意义。其一，中心化和协方差计算可以统一到矩阵乘法框架中，便于推导与实现。其二，之后无论采用特征值分解还是奇异值分解，实质上都围绕中心化后的数据矩阵 Xc​ 展开。

线性降维与主成分分析

主成分分析的目标

主成分分析的核心任务，是在原始 p 维空间中寻找一组新的正交基，使得数据在这组新坐标轴下尽可能集中地表达主要变化方向。设 u1​,u2​,…,up​ 是一组标准正交基，其中每个 uj​∈Rp，并满足

ujT​uk​=δjk​

其中 δjk​ 是 Kronecker 符号。当数据完成中心化后，任一样本都可以在这组基下分解为

xi​−xˉ=j=1∑p​(ujT​(xi​−xˉ))uj​

若只保留前 q 个方向，则样本的低维近似重构为

x^i​=xˉ+j=1∑q​(ujT​(xi​−xˉ))uj​

其中 q<p。这意味着 PCA 实际上完成了两件事：一是把原始变量重新表示为一组新的坐标轴，二是只保留最重要的前 q 个坐标，从而达到降维目的。

PCA 的目标可以从两个等价角度表述。第一种表述是保留尽可能大的方差，即投影后的数据在低维子空间中尽可能分散。第二种表述是使重构误差尽可能小，即被丢弃的正交补空间成分尽可能少。这两个目标在正交约束下是等价的。

方差最大化形式

设保留前 q 个方向，则投影后样本在这些方向上的坐标为 ujT​(xi​−xˉ)。用这些坐标平方的平均值衡量投影方差，总目标函数为

J=N1​i=1∑N​j=1∑q​(ujT​(xi​−xˉ))2

将协方差矩阵代入可得

J=j=1∑q​ujT​Suj​

因此，PCA 的问题变为：在正交归一约束下，寻找一组方向，使得 ∑j=1q​ujT​Suj​ 最大。对单个方向而言，可写拉格朗日函数

L(u,λ)=uTSu−λ(uTu−1)

对 u 求导并令其为零，有

Su=λu

这表明最优方向必须是协方差矩阵 S 的特征向量，而对应的目标值由特征值决定。若将特征值按从大到小排列为

λ1​≥λ2​≥⋯≥λp​≥0

则前 q 个主成分方向应取为对应于最大 q 个特征值的特征向量。第一个主成分方向表示数据方差最大的方向；第二个主成分方向与第一个正交，并在此约束下使剩余方差最大；其余主成分依次类推。

重构误差最小化形式

若只保留前 q 个主成分，则被丢弃的部分为

ri​=(xi​−xˉ)−j=1∑q​(ujT​(xi​−xˉ))uj​

其平方误差平均值为

N1​i=1∑N​∣ri​∣22​

利用正交分解，可得

N1​i=1∑N​∣ri​∣22​=j=q+1∑p​ujT​Suj​

当 uj​ 取为 S 的特征向量时，上式进一步化为

N1​i=1∑N​∣ri​∣22​=j=q+1∑p​λj​

因此，最小化重构误差等价于让被舍弃方向上的特征值之和最小，也即让被保留方向上的特征值之和最大。这就是“方差最大化”和“信息损失最小化”能够导向同一组主成分的原因。

协方差矩阵做特征值分解后可写为

S=UΛUT

其中

U=[u1​​u2​​⋯​up​​]

而

Λ=diag(λ1​,λ2​,…,λp​)

于是 PCA 的全部结构都由协方差矩阵的谱分解给出。特征向量确定坐标轴方向，特征值刻画各方向所包含的方差大小。

奇异值分解、主成分得分与主坐标分析

由奇异值分解计算 PCA

在实际计算中，通常不直接对协方差矩阵做特征值分解，而是先对中心化后的数据矩阵做奇异值分解。设

Xc​=HX=UΣVT

其中 U∈RN×N 与 V∈Rp×p 为正交矩阵，Σ 为非负奇异值组成的对角型矩阵。则

S=N1​XcT​Xc​=N1​VΣTΣVT

因此，协方差矩阵的特征向量正是 V 的列向量，而协方差矩阵的特征值为

λj​=Nσj2​​

其中 σj​ 是第 j 个奇异值。

这一定理说明，只要对中心化后的数据矩阵做一次 SVD，就能够同时得到 PCA 所需的主方向与方差信息。若取前 q 个主方向组成矩阵

Vq​=[v1​​v2​​⋯​vq​​]

则样本在主成分坐标系中的表示为

Z=Xc​Vq​

又因为 Xc​=UΣVT，所以

Z=Uq​Σq​

其中 Uq​ 和 Σq​ 分别表示保留前 q 个奇异向量和奇异值后的矩阵。这些坐标通常称为主成分得分，它们给出了每个样本在低维空间中的位置。

与主坐标分析的关系

若从样本间内积结构出发，可以构造矩阵

T=Xc​XcT​=HXXTH=UΣΣTUT

这是一个 N×N 的样本 Gram 矩阵。对 T 进行特征值分解，同样可以恢复主成分坐标。因为当 T=UΣΣTUT 时，前 q 个样本坐标可由

Uq​Σq​

给出。

这与从协方差矩阵 S=N1​XcT​Xc​ 出发得到的结果在本质上是一致的，只是计算对象不同。前者处理的是特征维度上的协方差结构，矩阵大小为 p×p；后者处理的是样本之间的内积结构，矩阵大小为 N×N。当特征维数很高而样本量相对较小时，直接处理 T 往往更方便。这种从样本坐标恢复低维表示的思路，与主坐标分析的形式密切相关。

概率主成分分析

概率模型的设定

概率主成分分析从生成模型角度重新解释 PCA。其基本假设是：观测数据 x∈Rp 并不是直接在高维空间中自由变化，而是由一个低维潜变量 z∈Rq 经过线性映射并叠加各向同性高斯噪声生成，其中 q<p。模型写为

z∼N(0,Iq​) x=Wz+μ+ε ε∼N(0,σ2Ip​)

其中，μ∈Rp 表示数据均值，W∈Rp×q 表示从潜变量空间到观测空间的线性映射，ε 表示噪声项，且其协方差被设定为 σ2Ip​，这意味着噪声在各个观测维度上具有相同方差且彼此独立。

由该模型可得观测变量的边际分布为

x∼N(μ,WWT+σ2Ip​)

因此，概率 PCA 把高维数据的协方差结构分解为两部分：一部分来自低维潜变量通过 W 所张成的主子空间，另一部分来自各向同性噪声。

后验推断与参数学习

在推断阶段，需要根据观测到的 x 反推出潜变量 z 的分布。定义

M=WTW+σ2Iq​

则后验分布为

p(z∣x)=N(M−1WT(x−μ),σ2M−1)

利用矩阵恒等变换，也可以把后验均值写成

WT(WWT+σ2Ip​)−1(x−μ)

这一形式与线性高斯模型中的标准后验推断完全一致。后验均值给出潜变量的最优线性估计，后验协方差反映在给定观测之后仍然保留的不确定性。

参数 μ、W 与 σ2 可以通过极大似然方法估计，实际求解中常采用 EM 算法。与传统 PCA 相比，概率 PCA 的特点不在于改变主子空间本身，而在于给出了一个明确的概率生成解释，使“主成分”不再只是代数上的正交投影结果，而成为潜在变量模型中的隐变量表示。

方法之间的关系

特征选择、线性降维与流形学习分别对应三种不同的建模思想。特征选择不构造新变量，而是从原始变量中筛选子集，适合强调可解释性和原始语义保持的场景。PCA 属于线性投影方法，通过协方差结构提取全局最主要的变化方向，适合主要结构能够由线性子空间刻画的数据。流形学习则进一步放宽线性假设，认为数据可能分布在嵌入高维空间的低维曲面附近，因此降维目标不再是寻找一个全局线性平面，而是恢复更一般的内在几何结构。Isomap、LLE 等方法都属于这一类，其共同特征是把低维表示建立在非线性邻域关系或流形结构之上。

小结

降维的理论起点是高维空间中的稀疏性与维度灾难。中心化矩阵把去均值过程写成标准的投影运算，协方差矩阵则把样本的主要变化结构压缩为可分解的二阶统计量。在线性降维中，PCA 通过求解协方差矩阵的特征向量，得到方差最大的正交方向；从等价角度看，它也同时给出了重构误差最小的低维子空间。SVD 提供了更直接的数值计算路径，而主坐标分析则展示了从样本内积结构恢复低维坐标的另一种等价表达。概率 PCA 进一步把这一过程写成带高斯潜变量的生成模型，从而把主成分表示与统计推断联系起来。非线性方法则把线性子空间推广为更一般的低维流形。
