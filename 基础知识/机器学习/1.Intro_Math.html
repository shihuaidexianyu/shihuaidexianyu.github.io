1.Intro_Math

Apr 22, 202620 min read

Summary

这篇笔记带你走进机器学习中最底层的两种概率世界观——频率派与贝叶斯派，并系统梳理高斯分布的“全家桶”：从一维参数估计到多维几何直观，再到分块条件分布与线性高斯模型。读完之后，你应该能熟练写出高斯密度的公式、解释 MLE 与 MAP 的哲学差异，并理解为什么“高斯在线性变换下仍然是高斯”这件事如此重要。

概率的两种世界观

在机器学习的起点，我们总会遇到一个根本问题：概率到底该怎么理解？ 面对同一组观测数据，统计学家们分成了两大阵营——频率派与贝叶斯派。它们并非水火不容，而是像两位性格不同的向导：一位喜欢用优化寻找“最像真相”的参数；另一位则喜欢保留完整的不确定性，用分布说话。

假设我们有 N 个样本，每个样本都是 p 维向量，可以写成

XN×p​=(x1​,x2​,⋯,xN​)T,xi​=(xi1​,xi2​,⋯,xip​)T

我们通常假设这些样本来自某个参数化的分布 p(x∣θ)。那么，参数 θ 究竟是一个固定但未知的常量，还是一个随机变量？这就是两大学派分歧的起点。

Note

这里的 θ 往往不是单个数字，而是一组参数构成的向量，比如高斯分布的均值和方差。

频率派：数据随机，参数固定

频率派把 θ 看作一个固定但未知的常量。样本是随机的，参数本身没有分布。在独立同分布（i.i.d.）假设下，整个数据集的联合概率是

p(X∣θ)iid=​i=1∏N​p(xi​∣θ)

这个式子叫做似然函数。它的角色很特别：把已经观测到的数据 X 固定，把参数 θ 当作自变量。频率派的核心任务是找到让这组数据出现概率最大的参数，也就是最大似然估计（MLE）。

Intuition

想象你在黑暗中投掷飞镖，靶心就是真实的 θ。每次投掷得到的数据点 xi​ 会落在靶心附近，但带有随机噪声。MLE 的想法是：看看所有可能的靶心位置，哪个位置最“像”会生成眼前这一堆飞镖落点？

因为对数函数单调递增，最大化似然和最大化对数似然是等价的：

θMLE​=argθmax​logp(X∣θ)=argθmax​i=1∑N​logp(xi​∣θ)

对数把连乘变成连加，既方便求导，也更适合数值优化。频率派的方法论可以概括为三步：建模似然 → 构造目标函数 → 优化求解。

贝叶斯派：参数也是随机变量

贝叶斯派则不认为 θ 是固定的。在他们看来，参数在观测之前就已经有一个先验分布 p(θ)，它凝聚了我们的经验、结构偏好或领域知识。观测到数据 X 后，我们用贝叶斯定理更新对参数的信念，得到后验分布：

p(θ∣X)=p(X)p(X∣θ)p(θ)​=∫θ​p(X∣θ)p(θ)dθp(X∣θ)p(θ)​

Intuition

你可以把 p(X) 理解为“证据 X 发生的总概率”。它相当于对所有可能的参数世界做加权平均：在每个参数值下计算数据出现的概率 p(X∣θ)，再按该参数的先验概率 p(θ) 加权积分。

如果我们只想得到一个单点估计，可以取后验概率最大的那个点，这就是最大后验估计（MAP）：

θMAP​=argθmax​p(θ∣X)=argθmax​p(X∣θ)⋅p(θ)

由于分母 p(X) 与 θ 无关，求极值时可以忽略。MAP 与 MLE 的区别在于：MAP 同时考虑了数据证据和先验约束，而 MLE 只依赖数据本身。

Example

假设你要估计一枚硬币正面朝上的概率。如果你只抛了 3 次，恰好 3 次都是正面，MLE 会告诉你正面概率是 1.0。但如果你有一个“硬币应该是公平的”先验，MAP 可能会给出一个更保守的估计，比如 0.7。这就是先验的“刹车”作用。

不过，贝叶斯方法的真正精髓并不只是求一个点估计，而是保留整个后验分布。后验分布完整刻画了观测之后参数的不确定性。对新样本 xnew​，贝叶斯预测不是依赖某个单一参数，而是对所有可能参数按后验概率加权平均：

p(xnew​∣X)=∫p(xnew​∣θ)p(θ∣X)dθ

Intuition

这就像是召集一群专家（每个专家对应一个参数值），让每个专家按他的后验可信度投票，最后取加权平均。贝叶斯推断天然地把参数的不确定性传递到了预测结果中。

两种观点的差异与联系

维度频率派贝叶斯派
参数定义固定未知常量随机变量
计算任务优化目标函数（如 MLE）积分与推断（如后验、边缘似然）
不确定性只存在于数据中同时存在于数据和参数中

在实际机器学习中，这两种立场经常交融。频率派方法常与最优化理论、经验风险最小化结合；贝叶斯方法则更强调先验建模与不确定性表达。当复杂后验无法解析求解时，我们往往需要借助近似推断或采样积分方法，比如后面会讲到的马尔可夫链蒙特卡洛（MCMC）。

高斯分布：机器学习的“老朋友”

高斯分布（正态分布）是概率论和机器学习中最基本、最重要的连续分布。它不仅形式简洁、数学性质优美，还出现在噪声建模、参数估计、后验推断等 countless 场景中。可以说，不懂高斯分布，就很难真正理解线性模型和贝叶斯推断。

一维高斯分布与最大似然估计

设样本来自一维高斯分布，参数为均值 μ 和方差 σ2。记参数向量

θ=(μ,σ2)

p 维高斯分布的通用密度函数是

p(x∣μ,Σ)=(2π)p/2∣Σ∣1/21​exp(−21​(x−μ)TΣ−1(x−μ))

当 p=1 时，协方差矩阵退化为标量 σ2，密度函数简化为

p(x∣μ,σ2)=2π​σ1​exp(−2σ2(x−μ)2​)

Intuition

这个“钟形曲线”的形状完全由两个数字控制：μ 决定钟摆在横轴上的位置，σ 决定钟的胖瘦。σ 越大，数据越分散，钟越矮胖；σ 越小，数据越集中，钟越高瘦。

对独立同分布样本 X={x1​,x2​,⋯,xN​}，对数似然为

logp(X∣μ,σ2)=i=1∑N​logp(xi​∣μ,σ2)

把一维高斯密度代入，得到

logp(X∣μ,σ2)=−2N​log(2π)−Nlogσ−2σ21​i=1∑N​(xi​−μ)2

均值的最大似然估计

与 μ 有关的部分只有平方误差项，所以最大化对数似然等价于最小化平方误差和：

μMLE​=argμmin​i=1∑N​(xi​−μ)2

对 μ 求导并令导数为零：

∂μ∂​i=1∑N​(xi​−μ)2=0

于是得到

μMLE​=N1​i=1∑N​xi​

Example

如果你测了 5 次身高，分别是 170、172、168、171、169 cm，那么均值的 MLE 就是它们的平均值 170 cm。这个结论非常直观：在钟形曲线假设下，数据中心就在样本均值处。

方差的最大似然估计

把 μMLE​ 代回对数似然，对 σ 求极值。与 σ 有关的部分为

−Nlogσ−2σ21​i=1∑N​(xi​−μMLE​)2

等价地，可以写成最小化问题：

σMLE​=argσmin​[Nlogσ+2σ21​i=1∑N​(xi​−μMLE​)2]

对 σ 求导并令导数为零，可得

σMLE2​=N1​i=1∑N​(xi​−μMLE​)2

Trap

注意这里的分母是 N，而不是我们中学课本里熟悉的 N−1。这是 MLE 的结果。虽然在大样本时差别微乎其微，但在小样本时它会系统性地低估真实方差。

无偏性与偏差修正

样本均值的 MLE 是无偏的。对所有可能数据集取期望：

ED​[μMLE​]=ED​[N1​i=1∑N​xi​]=N1​i=1∑N​ED​[xi​]=μ

但方差的 MLE 不是无偏的。因为它用的是同一数据集上估计出的 μMLE​，在重复抽样时会多消耗一个自由度。其期望满足

ED​[σMLE2​]=NN−1​σ2

Intuition

为什么方差会被低估？因为我们用样本均值 μMLE​ 去“贴合”了数据，使得每个 xi​ 到均值的距离比到真实均值的距离平均要稍微小一点。就像你为了让所有人离自己近一点，下意识地把营地扎在了人群的中央。

修正偏差后，得到无偏方差估计：

σ^2=N−11​i=1∑N​(xi​−μMLE​)2

这里以 N−1 为分母，正是因为均值估计已经占用了一个自由度。这也是统计教材中样本方差的标准形式。

多维高斯分布

基本形式与几何解释

在 p 维空间中，多维高斯分布写作

p(x∣μ,Σ)=(2π)p/2∣Σ∣1/21​exp(−21​(x−μ)TΣ−1(x−μ))

其中 x,μ∈Rp，Σ∈Rp×p 是协方差矩阵，必须是对称半正定矩阵；在密度严格可逆的情形下，通常要求 Σ 正定。

指数项中的二次型

Δ=(x−μ)TΣ−1(x−μ)

叫做马氏距离的平方。它不是普通的直线距离，而是考虑了不同方向方差大小和变量相关结构之后的“加权距离”。

Example

想象你在山坡上找路：有些方向坡度平缓（方差大），有些方向陡峭狭窄（方差小）。马氏距离就像一张“地形图”，它告诉你沿着哪个方向走才是真正“远”的。

若对对称正定矩阵 Σ 做特征值分解：

Σ=UΛUT

其中

U=(u1​,u2​,⋯,up​),Λ=diag(λ1​,λ2​,⋯,λp​)

于是

Σ−1=UΛ−1UT=i=1∑p​ui​λi​1​uiT​

将其代入马氏距离可得

Δ=(x−μ)TΣ−1(x−μ)=i=1∑p​λi​yi2​​

其中 yi​=uiT​(x−μ) 是向量 x−μ 在特征向量 ui​ 方向上的投影。

Intuition

这说明多维高斯分布的等密度面是以 μ 为中心、沿着特征向量方向展开、由特征值控制伸缩尺度的同心椭球。特征值大的方向（λi​ 大）椭圆拉得长，分布平缓；特征值小的方向椭圆压得扁，分布集中。

参数自由度与结构约束

多维高斯分布的均值有 p 个自由度，协方差矩阵由于对称性共有

2p(p+1)​

个自由参数。总体参数规模约为 O(p2)。在高维数据中，这种规模会带来估计不稳定、样本需求过大和计算负担增加等问题。

降低复杂度的一个常见思路是对协方差结构施加约束。如果假设协方差矩阵为对角矩阵，那么不同维度条件上不相关，只保留各自的方差参数；如果进一步假设所有对角元素相同，就得到各向同性结构，分布在所有方向上具有相同的扩散程度。前者对应更一般的降维与潜变量建模思路，后者则常与概率主成分分析（PPCA）等模型联系在一起。

单峰局限与混合建模

单个高斯分布只有一个峰值，因此适合刻画单峰、近似椭球形的数据分布。对于存在多个聚类中心或多个模式的数据，单个高斯往往力不从心。此时常用高斯混合模型（GMM），通过若干个高斯分量的加权和来刻画复杂的多峰结构。

高斯分布的线性变换性质

高斯分布最核心的性质之一是对仿射变换封闭。若

x∼N(μ,Σ),y=Ax+b

则有

y∼N(Aμ+b,AΣAT)

Intuition

这就像一个“高斯变形器”：你拿一堆服从高斯分布的点，对它们做旋转、缩放、平移，得到的仍然是一堆服从高斯分布的点，只是中心位置和形状变了。这让高斯分布在推导边缘分布、条件分布和线性模型时特别好用。

其原因在于

E[y]=E[Ax+b]=AE[x]+b=Aμ+b

以及

Var[y]=Var[Ax+b]=Var[Ax]=AVar[x]AT=AΣAT

这一性质在推导边缘分布、条件分布、线性高斯模型与卡尔曼滤波等方法时都具有基础性作用。

分块高斯向量的边缘分布与条件分布

设随机向量按两个子块划分为

x=(xa​xb​​),μ=(μa​μb​​),Σ=(Σaa​Σba​​Σab​Σbb​​)

并且满足

x∼N(μ,Σ)

其中 xa​∈Rm，xb​∈Rn。

边缘分布

由高斯分布在线性映射下的封闭性可知，任一子块的边缘分布仍然是高斯分布：

xa​∼N(μa​,Σaa​) xb​∼N(μb​,Σbb​)

Example

如果你有一个二维高斯分布，描述一个人的“身高和体重”，那么只看“身高”这一维，它的分布就是一维高斯，均值和方差直接从联合分布的对应块里取就行。

条件分布

联合高斯最重要的性质之一，是条件分布仍为高斯。其结果为

xb​∣xa​∼N(μb​+Σba​Σaa−1​(xa​−μa​),Σbb​−Σba​Σaa−1​Σab​)

以及

xa​∣xb​∼N(μa​+Σab​Σbb−1​(xb​−μb​),Σaa​−Σab​Σbb−1​Σba​)

Intuition

条件均值可以看作“先猜一个均值 μb​，再根据观测到的 xa​ 与期望的偏差 (xa​−μa​) 做线性校正”。校正强度由交叉协方差 Σba​ 和已知部分的协方差 Σaa​ 决定。如果 xa​ 和 xb​ 相关性很强，观测到 xa​ 就能让我们对 xb​ 的估计大幅调整。

条件协方差不再依赖观测值本身，只依赖于协方差结构：

Σbb⋅a​=Σbb​−Σba​Σaa−1​Σab​ Σaa⋅b​=Σaa​−Σab​Σbb−1​Σba​

这两个矩阵分别是相应块的 Schur 补。它们表示在已知另一部分信息后，剩余不确定性的大小。由于部分信息已被观测吸收，条件协方差通常小于原始边缘协方差。

Note

在贝叶斯线性回归和卡尔曼滤波中，你都会反复见到这种“先验均值 + 线性校正”的形式。这不是巧合，而是联合高斯条件分布结构的直接体现。

线性高斯模型

设先验分布与条件分布分别为

p(x)=N(μ,Λ−1) p(y∣x)=N(Ax+b,L−1)

其中 Λ 与 L 是精度矩阵（协方差矩阵的逆）。条件模型可写为

y=Ax+b+ϵ,ϵ∼N(0,L−1)

并假定噪声 ϵ 与 x 相互独立。

边缘分布 p(y)

根据线性变换性质，y 的均值与方差分别为

E[y]=Aμ+b Var[y]=AΛ−1AT+L−1

因此

p(y)=N(Aμ+b,AΛ−1AT+L−1)

Intuition

y 的不确定性来自两个“源头”：一部分是潜变量 x 传播过来的不确定性（AΛ−1AT），另一部分是观测噪声本身（L−1）。这就像我们测量温度时，误差既来自环境本身的波动，也来自温度计的读数误差。

联合分布与后验分布 p(x∣y)

令

z=(xy​)

则联合分布仍是高斯分布。根据协方差定义，

Cov(x,y)=E[(x−μ)(y−E[y])T]

代入 y=Ax+b+ϵ 并利用独立性，可得

Cov(x,y)=Λ−1AT

于是联合分布写为

(xy​)∼N((μAμ+b​),(Λ−1AΛ−1​Λ−1ATAΛ−1AT+L−1​))

再由联合高斯的条件分布公式可得后验分布 p(x∣y) 仍为高斯，其条件均值为

E[x∣y]=μ+Λ−1AT(AΛ−1AT+L−1)−1(y−Aμ−b)

条件协方差为

Var[x∣y]=Λ−1−Λ−1AT(AΛ−1AT+L−1)−1AΛ−1

Summary

这两个公式是贝叶斯线性推断的“灵魂”。它们告诉我们：观测 y 出现后，我们对隐变量 x 的最优高斯更新就是“先验均值 + 观测残差的线性校正”；而条件协方差则告诉我们，利用观测信息后，原先关于 x 的不确定性被压缩了多少。这一结构构成了贝叶斯线性回归、状态空间模型与多种概率图模型推断公式的基础。
