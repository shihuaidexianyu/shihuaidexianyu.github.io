Lecture 5.1 基础人工神经网络：目标函数的选择

Summary

这一讲解决的核心问题是：神经网络的损失函数到底怎么来的？答案可以一句话概括——先假设数据的概率分布，再取负对数似然，就得到了损失函数。

  • 回归问题 + 高斯噪声假设 → 均方误差（MSE）
  • 二分类问题 + 伯努利分布假设 → 二元交叉熵（BCE）
  • 多分类问题 + 多项分布假设 → 多类交叉熵（Cross-Entropy）
  • 参数上加高斯先验 → L2​ 正则化（权重衰减） 掌握这套“分布假设 → 似然函数 → 负对数似然”的推导链，考试里遇到相关推导就不会慌。

1. 目标函数的统计学基础

这一讲的核心思路其实特别统一——神经网络里的目标函数不是拍脑袋定的，而是由输出变量的概率模型决定的。训练过程可以理解为：先给目标变量设定一个条件分布 p(t∣x,θ)，再根据观测数据去选择怎么估计参数。线性输出对应均方误差，Sigmoid 输出对应二元交叉熵，Softmax 输出对应多类交叉熵——这三种常见组合背后，其实是不同的噪声假设和似然函数。所以选目标函数，本质上是个统计推断问题。

2. 概率论预备

2.1 贝叶斯公式

贝叶斯公式是这整个推演的根基，它告诉我们：拿到观测数据 D 之后，参数的后验分布该怎么算。设参数是 w，那么

p(w∣D)=p(D)p(D∣w)p(w)​

其中，p(w∣D) 是后验分布，表示“看到数据后，参数长什么样”；p(D∣w) 是似然函数，表示“参数给定后，数据出现的可能性”；p(w) 是先验分布，表示“还没看数据时，我对参数的信念”；p(D) 是证据项，用来归一化。于是我们也可以写成

p(w∣D)∝p(D∣w)p(w)

这说明后验分布由数据证据和先验约束共同决定。最大似然估计（MLE）只用似然；最大后验估计（MAP）则把两者都用上了。

Intuition

可以把贝叶斯推断想象成“学术圈更新认知”的过程：

  • 先验 p(w)：你导师凭经验觉得参数应该在某个范围；
  • 似然 p(D∣w)：你做实验跑出来的数据说参数可能在哪儿；
  • 后验 p(w∣D)：综合导师经验和实验数据后，你最终相信的参数分布。 数据越多，似然越 dominant，先验的影响就越小。

2.2 一维高斯分布

高斯分布是连续变量里最常用、也最重要的概率模型。设随机变量 x 的均值为 μ、方差为 σ2，则

N(x∣μ,σ2)=(2πσ2)1/21​exp{−2σ21​(x−μ)2}

μ 决定分布中心在哪里，σ 决定分布“胖瘦”。标准差越大，曲线越平缓；标准差越小，曲线越尖。课件里的示意图就是以 μ 为峰值，宽度大约 2σ 的区间来表示尺度。

高斯分布的一阶矩和二阶矩分别为

E[x]=∫−∞∞​xN(x∣μ,σ2)dx=μ E[x2]=∫−∞∞​x2N(x∣μ,σ2)dx=μ2+σ2

所以方差满足

Var(x)=E[x2]−E[x]2=σ2

均值刻画位置，方差刻画波动强度。均值相同但方差不同的两个高斯分布，中心重合，但一个更“集中”，一个更“分散”。

Note

记住这个结论：高斯分布的方差就是 σ2。推导过程考试不一定会考完整积分，但 E[x2]=μ2+σ2 这个式子偶尔会用来构造计算题，留个印象。

2.3 高维高斯分布与独立同分布样本的似然

对 D 维随机向量 x，高斯分布写成

N(x∣μ,Σ)=(2π)D/2∣Σ∣1/21​exp{−21​(x−μ)TΣ−1(x−μ)}

其中，μ 是均值向量，Σ 是协方差矩阵。对角元描述各维度的方差，非对角元描述维度之间的协方差。指数项里的二次型叫马氏距离，它不仅看每一维偏离了多少，还考虑了不同维度之间的相关性。

若样本 xn​n=1N​ 独立同分布（i.i.d.），则数据集在参数 μ,σ2 给定下的联合概率密度为

p(x∣μ,σ2)=n=1∏N​N(xn​∣μ,σ2)

i.i.d. 假设的意思是：每个样本的生成机制一样，而且样本之间互不干扰。所以整个数据集的似然，就是每个样本密度的连乘。课件里的示意图把所有样本点投影到同一条高斯曲线上，想强调的正是：每个样本都为总似然贡献一个密度值，学习过程就是找一组参数，让这组密度的乘积最大。

Trap

这里很容易犯一个直觉错误：似然是概率密度的乘积，不是求和！因为样本独立，联合分布等于边缘分布的乘积。后面取对数后才会变成求和形式，推导时千万别提前把乘积改成求和。

2.4 伯努利分布

对二值变量 x∈0,1，若 p(x=1∣μ)=μ，则其分布为

Bern(x∣μ)=μx(1−μ)1−x

这里 μ∈[0,1] 是取 1 的概率。x=1 时式子变成 μ；x=0 时变成 1−μ。这个写法很聪明，把两个离散取值统一到了一个表达式里。

伯努利分布的均值和方差分别为

E[x]=μ Var(x)=μ(1−μ)

方差在 μ=0.5 时达到最大，说明正负两类最均衡的时候不确定性最高；μ 越接近 0 或 1，随机性就越弱。若样本独立同分布，数据集的联合概率为

p(D∣μ)=n=1∏N​p(xn​∣μ)=n=1∏N​μxn​(1−μ)1−xn​

这个形式正是二分类最大似然推导的出发点。

Example

抛一枚不均匀硬币，正面朝上概率为 μ。抛 5 次，结果依次是 1, 0, 1, 1, 0（1 表示正面）。那么这组数据的似然就是： p(D∣μ)=μ⋅(1−μ)⋅μ⋅μ⋅(1−μ)=μ3(1−μ)2 最大似然估计就是让这个式子最大的 μ，一眼就能看出是 μ^​=3/5=0.6。二分类交叉熵的推导和这个思路完全一致。

Trap

常见误区：伯努利分布的方差 μ(1−μ) 在 μ=0.5 时最大，在 μ=0 或 μ=1 时为 0。有些同学会直觉地认为“概率越极端，方差越大”，其实正好相反——概率越接近 0 或 1，结果越确定，方差越小。

2.5 多项分布变量

对 K 类互斥类别，可用 one-hot 向量表示单个样本：

x=(x1​,…,xK​)T

其中 xk​∈0,1，且满足

k=1∑K​xk​=1

若第 k 类出现的概率为 μk​，则有 μk​≥0 且

k=1∑K​μk​=1

此时单个样本的分布为

p(x∣μ)=k=1∏K​μkxk​​

因为 one-hot 编码里只有一个分量是 1，上式实际上只保留对应类别的那个概率项，其余都是 μk0​=1。

若数据集为 x1​,…,xN​，且样本独立同分布，则似然函数为

p(D∣μ)=n=1∏N​k=1∏K​μkxnk​​=k=1∏K​μkmk​​

其中

mk​=n=1∑N​xnk​

表示第 k 类在数据中出现的次数。多类分类中的交叉熵，正是从这一似然结构推导出来的。

Note

记忆技巧：多项分布的似然 ∏k=1K​μkmk​​ 和伯努利分布的 ∏μxn​(1−μ)1−xn​ 本质上是一个套路——都是“真实类别的概率连乘”。把二分类推广到 K 类，伯努利就变成了多项分布。

3. 信息论预备

3.1 KL 散度

Kullback–Leibler 散度用来度量两个分布之间的“差距”。对分布 p(x) 与 q(x)，定义为

KL(p∣q)=∫p(x)lnq(x)p(x)​dx=−∫p(x)lnp(x)q(x)​dx

KL 散度总是非负的，而且只有当 p=q 时才等于 0。它不是对称的，所以一般来说 KL(p∣q)=KL(q∣p)。在学习问题里，可以把 p 当作真实数据分布，q 当作模型分布。最小化 KL 散度，就是让模型分布尽量贴近真实分布。

Intuition

KL 散度可以理解为“用分布 q 去近似分布 p 时，平均多花了多少信息量”。它不是真正的距离（因为不对称），但衡量的是两个分布的“偏离程度”。在机器学习中，我们常常最小化 KL(pdata​∣pmodel​)，也就是让模型去逼近数据的真实分布。

3.2 Cramér–Rao 下界与 Fisher 信息

Cramér–Rao 下界给出了无偏估计量方差的理论下限。对参数 θ，Fisher 信息定义为

F(θ)=−∫dx,p(x∣θ),∇θ​∇θ​lnp(x∣θ)

在标量参数情形下，无偏估计量 θ^ 的方差满足

Var(θ^)≥F(θ)1​

Fisher 信息越大，说明样本对参数越敏感，可估计性越强，估计方差的下界就越小。这个下界为讨论最大似然估计的渐近有效性提供了理论基准。

Note

有个直观的对应关系：Fisher 信息 F(θ) 衡量的是“似然函数在真实参数附近的弯曲程度”。弯曲得越厉害（曲率越大），说明数据对参数值越挑剔，估计起来就越精确，方差下界 1/F 就越小。

4. 最大似然推断

设真实分布为 p(X∣θ)，模型分布为 p(X∣θ^)。学习的目标可以表述为：最小化真实分布与模型分布之间的 KL 散度：

KL![p(X∣θ),∣,p(X∣θ^)]=EX∼p(X∣θ)​[logp(X∣θ)−logp(X∣θ^)]

进一步展开可得

KL![p(X∣θ),∣,p(X∣θ^)]=EX∼p(X∣θ)​[logp(X∣θ)]=EX∼p(X∣θ)​[logp(X∣θ^)]

第一项只由真实分布决定，跟待估计的参数 θ^ 没关系，所以优化时可以不管它。于是，最小化 KL 散度等价于最大化模型对数据的对数似然。最大似然估计写成

θ^=argθmax​lnp(x∣θ)

这个结论特别重要，它给了我们一个统一的思路：只要先规定输出变量的条件分布，负对数似然就自然成为可以优化的损失函数。

Intuition

这段话的精髓可以一句话概括：让模型分布逼近真实分布（最小化 KL）≈ 让模型对真实数据赋予最高概率（最大化对数似然）。之所以这样做，是因为真实分布的那一项我们算不了（不知道真实分布），但能算的是“模型在真实数据上的对数概率”，这就是损失函数的来源。

5. 最大似然估计的渐近有效性

设 θ∗ 为真实参数，θ^ 为由样本 x 得到的最大似然估计。因为极大点处对数似然的一阶导数为 0，有

∇lnp(x∣θ^)=0

在 θ∗ 附近做一阶 Taylor 展开，可得

∇lnp(x∣θ∗)+∇∇lnp(x∣θ∗)(θ^−θ∗)≈0

因此

θ^−θ∗≈−∇∇lnp(x∣θ∗)∇lnp(x∣θ∗)​R2​R1​​

其中 R1​ 是 score function，R2​ 是负 Hessian。对于独立同分布样本，在大样本极限下，R2​ 近似于 Fisher 信息 F，并且有

E[R1​]=0 E[R12​]=F

于是得到

E[θ^−θ∗]≈0 E[(θ^−θ∗)2]≈F1​

这说明，在满足正则条件且样本量足够大时，最大似然估计近似无偏，并且达到了 Cramér–Rao 下界，因此具有渐近有效性。课件里的推导用的是标量参数形式；如果参数是向量，方差下界由 Fisher 信息矩阵的逆给出。

Note

考试重点：MLE 的渐近有效性是大样本性质，意思是“样本量 N→∞ 时才成立”。小样本时 MLE 不一定无偏，也不一定达到 Cramér–Rao 下界。如果题目问“MLE 是否总是无偏”，答案是否——只有在渐近意义下才是无偏的。

6. 从概率模型到神经网络损失函数

6.1 高斯噪声假设与回归中的均方误差

设训练数据为 (xn​,tn​)n=1N​，网络输出为 y(xn​,w)。若目标值在给定输入与参数后服从高斯分布，且噪声精度为 β，则

p(t∣x,w,β)=N(t∣y(x,w),β−1)

这里 β−1 是噪声方差，表示目标值围绕网络输出波动的强度。方差越小，模型认为观测值应该越紧密地聚集在预测值附近。

在独立同分布采样下，整个数据集的似然函数为

p(t∣x,w,β)=n=1∏N​N(tn​∣y(xn​,w),β−1)

对应的对数似然为

lnp(t∣x,w,β)=−2β​n=1∑N​(y(xn​,w)−tn​)2+2N​lnβ2N​ln(2π)

当 β 固定时，后两项与 w 无关，因此最大化对数似然等价于最小化平方误差和。常用的目标函数写成

E(w)=N1​n=1∑N​(y(xn​,w)−tn​)2

这说明，回归里的均方误差不是经验性的任意选择，而是高斯噪声假设下负对数似然的等价形式。这个结论能解释：在给定模型假设下怎么得到最可能的参数；但它不能单独证明高斯噪声假设本身一定正确——如果真实噪声明显不是高斯的，均方误差就不一定是最合适的目标函数。

Intuition

为什么回归用 MSE？想象你在靶场上射箭，箭的落点围绕靶心呈圆形散布——这就是高斯噪声。MSE 就是在找“让箭的平均落点最接近靶心的参数”。如果箭的落点分布有严重的偏斜或厚尾（不是高斯），那 MSE 就会出问题，因为它对异常值太敏感。

Trap

经典误区：MSE 是最常用的回归损失，但这不意味着它总是对的。它只是“高斯噪声假设”下的产物。如果数据噪声是拉普拉斯分布（厚尾），那么绝对值损失（MAE）才是对应的最大似然估计。考试如果让你推导，务必先写清楚分布假设，不能直接“显然，目标函数为 MSE”。

6.2 伯努利分布与二分类交叉熵

对于二分类问题，可以使用单个输出单元，并令激活函数为 Logistic Sigmoid：

y=σ(a)=1+exp(−a)1​

由于 Sigmoid 输出值落在 (0,1)，可以把它解释为后验类别概率

y(x,w)=p(C1​∣x) 1−y(x,w)=p(C2​∣x)

若目标标记 t∈0,1，则其条件分布服从伯努利分布：

p(t∣x,w)=y(x,w)t[1−y(x,w)]1−t

其中，t=1 时保留正类概率项，t=0 时保留负类概率项。

在独立同分布假设下，对数似然为

lnp(t∣x,w)n=1∑N​{tn​lnyn​+(1−tn​)ln(1−yn​)}

因此，最大似然对应的最小化目标函数为

E(w)=−n=1∑N​{tn​lnyn​+(1−tn​)ln(1−yn​)}

这就是二分类交叉熵，也叫伯努利负对数似然。它和均方误差的根本区别，不在于数值形式更复杂，而在于它直接对应二值随机变量的概率模型，并且把输出严格解释为类别后验概率。

Example

假设模型对某张猫狗分类图片的输出是 y=0.9（认为是猫的概率），而真实标签 t=1（确实是猫）。这个样本对交叉熵的贡献是 −ln(0.9)≈0.105。如果模型输出 y=0.1（很自信地认为是狗），贡献就是 −ln(0.1)≈2.303。可以看到，错得越离谱、越自信，惩罚呈指数级增长——这就是交叉熵的“脾气”。

Note

一个有意思的细节：Sigmoid + 交叉熵的组合，在梯度下降时比“Sigmoid + MSE”好训练得多。因为 MSE 在输出接近 0 或 1 时梯度会消失（Sigmoid 导数趋于 0），而交叉熵的梯度形式是 y−t，直接和误差成正比，不会出现梯度消失。很多教材把这个当作独立知识点，其实它可以从推导里自然看出来。

6.3 多项分布、Softmax 与多类交叉熵

对于 K 类互斥分类问题，输出层通常设置 K 个单元，并将第 k 个输出解释为

yk​(x,w)=p(tk​=1∣x)

为了保证所有输出非负且总和为 1，引入 Softmax 激活函数：

yk​(x,w)=∑j​exp(aj​(x,w))exp(ak​(x,w))​

其中 ak​(x,w) 是第 k 类对应的输出层线性输入。Softmax 的作用不是简单地“压缩数值”，而是把一组任意实数映射为合法的类别概率分布。

设目标向量 tn​=(tn1​,…,tnK​)T 为 one-hot 编码，则在独立同分布采样下，数据似然可写为

p(D∣w)=n=1∏N​k=1∏K​yk​(xn​,w)tnk​

取负对数后得到误差函数

E(w)=−n=1∑N​k=1∑K​tnk​lnyk​(xn​,w)

这就是多类交叉熵。因为 one-hot 向量里只有一个分量为 1，上式对每个样本实际上只保留真实类别对应概率的对数项。所以最小化交叉熵，等价于提高真实类别的预测概率。二分类里的 Sigmoid–Bernoulli 模型和多分类里的 Softmax–多项分布模型，在结构上是完全平行对应的，区别只是输出空间的维度和归一化方式不同。

Trap

Softmax 的数值稳定性陷阱：直接按公式计算 Softmax 时，如果 ak​ 很大，exp(ak​) 会爆炸；如果 ak​ 是很负的数，exp(ak​) 会下溢。实际代码里都会做数值稳定性处理：令 m=maxj​aj​，然后分子分母同时除以 exp(m)，也就是计算 ∑j​exp(aj​−m)exp(ak​−m)​。考试写推导时虽然不用写这个，但如果涉及编程实现或者面试，这个 trick 是必会的。

Note

速查：多类交叉熵的求和里，∑k=1K​tnk​lnyk​ 看起来是对所有 K 类求和，但因为 one-hot 只有一个 1，所以实际上每次只选中一个 lnytrue​。二分类交叉熵可以看成多类交叉熵在 K=2 时的特例，此时 Softmax 退化为 Sigmoid（差一个参数冗余）。

7. 正则化的贝叶斯解释

7.1 参数先验与最大后验估计

若对参数 w 引入零均值高斯先验，并设先验精度为 α，则

p(w∣α)=N(w∣0,α−1I)(2πα​)(M+1)/2exp{−2α​wTw}

其中 M+1 表示参数总维数，wTw 是权重向量的平方范数。这个先验“偏爱”较小的参数值，所以对过大的权重会施加惩罚。

由贝叶斯公式，后验分布满足

p(w∣x,t,α,β)∝p(t∣x,w,β)p(w∣α)

若似然仍采用高斯输出模型，则最大后验估计等价于最小化

E(w)=2β​n=1∑N​(y(xn​,w)−tn​)2+2α​wTw

将整体按 β 缩放后，可以写成“数据拟合项 + 权重衰减项”的形式，其中正则化系数为

λ=βα​

这说明，常见的 L2​ 正则化并不是纯粹的数值稳定技巧，而可以解释为在参数上施加零均值高斯先验后得到的最大后验估计。α 越大，先验越强，参数越倾向于被“压”到零附近；β 越大，说明观测噪声越小，数据拟合项的相对权重就越高。

Intuition

把 L2​ 正则化理解为“给参数套上一个高斯紧箍咒”：先验说“参数最好靠近 0”，似然说“参数最好拟合数据”，MAP 估计就是在这两者之间找平衡。λ=α/β 这个比值决定了“紧箍咒”有多紧——λ 大，模型更保守（参数更小，可能欠拟合）；λ 小，模型更激进（更贴近数据，可能过拟合）。

Note

参数速查：

  • α：先验精度，控制正则化强度；
  • β：噪声精度（噪声方差的倒数），控制数据拟合的重要性；
  • λ=α/β：正则化系数，深度学习中通常直接调这个超参。 很多深度学习框架里写的 weight_decay 对应的就是 λ（或 λ/2，取决于具体实现）。

7.2 全贝叶斯预测

最大似然与最大后验都输出单一参数估计值，而全贝叶斯方法保留参数的后验分布，并在预测时对参数不确定性进行积分。预测分布写成

p(t∣x,D)=∫p(t∣x,w),p(w∣D),dw

其中 D 为训练数据。这个式子表明，预测结果不是由某一个“最佳”参数给出的，而是由所有可能的参数按它们的后验概率加权平均得到的。这样得到的不仅是一个点预测，还包含了预测不确定性的信息。与最大似然或最大后验相比，全贝叶斯方法的计算代价通常更高，但在小样本、模型不确定性较强或需要不确定性估计的场景下，更有解释力。

Note

MLE 给出一个点 θ^MLE​，MAP 给出一个点 θ^MAP​，全贝叶斯给出的是一整个分布 p(w∣D)。三者的关系可以简单记为：

  • MLE：只看数据，不看先验；
  • MAP：数据 + 先验，但输出还是一个点；
  • 全贝叶斯：数据 + 先验，保留完整的不确定性。 考试中如果让你比较三者，抓住“是否用先验”和“是否保留分布”这两个维度就够了。

8. 输出层、分布假设与损失函数的对应关系

神经网络里常用的目标函数之间，存在非常清晰的统计对应关系。如果目标值是连续变量，且观测噪声被建模为高斯分布，负对数似然就化为均方误差；如果目标值是二值变量，伯努利分布对应二分类交叉熵；如果目标值是互斥的多类别 one-hot 向量，多项分布与 Softmax 输出共同导出多类交叉熵；如果进一步在参数上引入高斯先验，最大后验估计就对应于带 L2​ 正则项的目标函数。

目标函数的选择因此服从同一个原则：先给出数据生成假设，再由似然或后验推出优化准则。损失函数并不是与模型结构割裂的外部部件，而是概率模型、输出层设计与参数估计方式在训练目标上的统一表达。

Summary

最后帮大家串一下考试最常考的推导链：

  1. 写出分布假设（高斯 / 伯努利 / 多项分布）；
  2. 写出 i.i.d. 下的似然函数（连乘）；
  3. 取对数，把乘积变求和；
  4. 把与参数无关的项丢掉；
  5. 最大化似然 ↔ 最小化负对数似然 = 损失函数。

正则化那一块记住：高斯先验 → L2​ 正则，拉普拉斯先验 → L1​ 正则（虽然课件没讲 L1​，但这是很好的延伸）。把这个框架记在脑子里，这讲的内容就彻底活了。
