概率极限理论

概率极限理论

Apr 23, 202621 min read

一、引言

概率极限理论研究随机变量序列在极限过程中的行为。概率的统计解释、随机变量对另一个随机变量的逼近、复杂分布对简单分布的近似，都依赖于极限思想。设 (Ω,F,P) 为概率空间，定义在其上的一组随机变量

X1​,X2​,…,Xn​,…

构成随机变量序列，记为 Xn​。

概率极限理论的两个基本问题是：其一，随机变量序列在何种意义下趋于某个随机变量；其二，这种收敛如何支撑概率与统计中的近似、估计与推断。

大数定律回答“平均行为是否稳定”，中心极限定理回答“累积波动呈现何种分布”。

二、随机变量序列的收敛性

1. 几乎处处收敛、依概率收敛与依分布收敛

设 Xn​ 和 X 都定义在概率空间 (Ω,F,P) 上。

若

P({ω∈Ω:n→∞lim​Xn​(ω)=X(ω)})=1

则称 Xn​ 概率为 1 地收敛于 X，或称 几乎处处收敛、几乎必然收敛，记作

Xn​a.e.​X

几乎处处收敛是逐点意义上的收敛，但允许在一个概率为 0 的集合上不收敛（类比几何概型）。

若对任意 ε>0，都有

n→∞lim​P(∣Xn​−X∣<ε)=1

则称 Xn​ 依概率收敛于 X，记作

Xn​P​X

这一定义强调的是“偏离超过任意固定误差界的概率趋于 0”。等价地，

∀ε>0,n→∞lim​P(∣Xn​−X∣≥ε)=0

依概率收敛不要求对每个样本点逐点收敛，只要求“大偏差”越来越少。

设 Xn​ 和 X 的分布函数分别为 Fn​(x) 与 F(x)。若对 F(x) 的每一个连续点 x，都有

n→∞lim​Fn​(x)=F(x)

则称 Xn​ 依分布收敛于 X，也称 弱收敛，记作

Xn​d​X

依分布收敛只涉及分布函数的极限，不要求 Xn​ 与 X 必须定义在同一个概率空间中，也不要求两者在样本点层面“靠近”。

2. 三种收敛方式的强弱关系

三种收敛之间存在如下蕴含链：

graph LR
    A["几乎处处收敛"] --> B["依概率收敛"]
    B --> C["依分布收敛"]

即

Xn​a.e.​X⟹Xn​P​X⟹Xn​d​X

从几乎处处收敛到依概率收敛

设

A={ω∈Ω:Xn​(ω)→X(ω)}

若 Xn​a.e.​X，则 P(A)=0。对任意固定的 ε>0，令

B={ω∈Ω:有无穷多个 n 使 ∣Xn​(ω)−X(ω)∣≥ε}

以及

Bm​={ω∈Ω:存在 n≥m 使 ∣Xn​(ω)−X(ω)∣≥ε}

则 Bm​↓B，并且 B⊆A。于是

P(Bm​)→P(B)≤P(A)=0

而事件 ∣Xm​−X∣≥ε 蕴含于 Bm​，因此

P(∣Xm​−X∣≥ε)≤P(Bm​)→0

从而得到

Xn​P​X

从依概率收敛到依分布收敛

设 x 是 F(x) 的连续点，任取 η>0。由事件包含关系可得

P(X≤x−η)−P(∣Xn​−X∣>η)≤P(Xn​≤x)≤P(X≤x+η)+P(∣Xn​−X∣>η)

当 n→∞ 时，由 Xn​P​X 可知

P(∣Xn​−X∣>η)→0

于是

F(x−η)≤n→∞liminf​Fn​(x)≤n→∞limsup​Fn​(x)≤F(x+η)

再令 η↓0，利用 x 是连续点，得到

n→∞lim​Fn​(x)=F(x)

因此

Xn​d​X

3. 三种收敛的典型例子

例 1：几乎处处收敛的例子

设 Ω=[0,1]，并取有限个点 a1​,a2​,…,ak​∈Ω。定义随机变量序列

Xn​(ω)={n,n1​,​ω∈a1​,a2​,…,ak​ω∈/a1​,a2​,…,ak​​

对任意 ω∈/a1​,…,ak​，有 Xn​(ω)=1/n→0。只有在有限个点上，Xn​(ω)=n 不收敛到 0。由于有限点集的概率为 0，故

P({ω:n→∞lim​Xn​(ω)=0})=1

因此

Xn​a.e.​0

几乎处处收敛的关键在于：允许在零概率集合上发生异常行为。

例 2：依概率收敛但不几乎处处收敛

设 Ω=(0,1]，对每个正整数 k 以及 i=1,2,…,k，定义

Yki​(ω)={1,0,​ω∈(ki−1​,ki​]其他​

再按字典顺序构造序列

X1​=Y11​,X2​=Y21​,X3​=Y22​,…

对任意 ε∈(0,1)，若 Xn​=Yki​，则

P(∣Xn​−0∣<ε)=P(Yki​=0)=1−k1​→1

因此

Xn​P​0

但对任意固定的 ω∈(0,1]，随着区间划分不断细化，ω 会落入无穷多个子区间中，因此序列 Xn​(ω) 中既有无穷多个 1，也有无穷多个 0，极限不存在。于是

Xn​a.e.​0

这说明依概率收敛不必然推出几乎处处收敛。

例 3：依分布收敛但不依概率收敛

设 X1​,X2​,… 与 X 相互独立，且都服从参数为 p=0.5 的 0-1 分布。由于每个 Xn​ 与 X 有相同分布，所以对一切 n 都有

FXn​​(x)=FX​(x)

因此

Xn​d​X

但若取 0<ε<1，则

P(∣Xn​−X∣<ε)=P(Xn​=X)

又因为 Xn​ 与 X 独立，故

P(Xn​=X)=P(Xn​=1,X=1)+P(Xn​=0,X=0)=0.5×0.5+0.5×0.5=0.5

这个概率不趋于 1，所以

Xn​P​X

该例表明依分布收敛只说明分布函数趋同，并不说明同一概率空间中的随机变量彼此越来越接近。

4. 依分布收敛的一个经典例子：泊松定理

若

Xn​∼B(n,pn​),npn​→λ>0

并且 X∼Poisson(λ)，则对任意 k=0,1,2,…，

P(Xn​=k)=(kn​)pnk​(1−pn​)n−k→e−λk!λk​=P(X=k)

因此

Xn​d​X

这一结果说明，当单次成功概率很小而试验次数很大、且均值 npn​ 稳定在常数 λ 附近时，二项分布可以用泊松分布近似。

三、大数定律

1. 大数定律的基本思想

大数定律研究“算术平均是否稳定”。它回答两个基本问题：一是事件发生的频率为何能够近似事件的概率；二是样本均值为何能够近似总体期望。其核心是：在适当条件下，大量独立或近独立的随机影响在平均后会变得稳定。

2. 重要不等式

马尔可夫不等式

设随机变量 X≥0，且 E(X) 存在。对任意 ε>0，

P(X≥ε)≤εE(X)​

Important

也可以通过类似于切比雪夫不等式那样的积分方式证明，那个会更简单一些。

Summary

证明可由指示函数完成。令

A=ω∈Ω:X(ω)≥ε

并定义其指示函数 IA​(ω)。则

εIA​(ω)≤X(ω)

对两边取期望可得

εE(IA​)≤E(X)

又因为 E(IA​)=P(A)，故

εP(A)≤E(X)

即得马尔可夫不等式。

若 E(∣X∣k) 存在，则将马尔可夫不等式应用于非负随机变量 ∣X∣k，得到

P(∣X∣≥ε)≤εkE(∣X∣k)​

这是高阶矩形式的马尔可夫不等式。

切比雪夫不等式

若 D(X) 存在，则对任意 ε>0，

P(∣X−E(X)∣≥ε)≤ε2D(X)​

等价地，

P(∣X−E(X)∣<ε)≥1−ε2D(X)​

切比雪夫不等式说明：方差越小，随机变量偏离其均值的概率越小。它是证明大数定律最直接的工具之一。

3. 弱大数定律的定义

设随机变量序列 Xk​ 的每个期望都存在，记算术平均为

Xˉn​=n1​k=1∑n​Xk​

若

Xˉn​−E(Xˉn​)P​0

即对任意 ε>0，

n→∞lim​P(∣Xˉn​−E(Xˉn​)∣<ε)=1

则称序列 Xk​ 服从大数定律。课程语境下，这里的“大数定律”指弱大数定律，其本质是样本平均在概率意义下稳定于其均值。

4. 伯努利大数定律

设事件 A 在 n 次独立重复试验中发生的次数为 nA​，单次试验中事件 A 发生的概率为 p。则对任意 ε>0，

n→∞lim​P(​nnA​​−p​≥ε)=0

等价地，

nnA​​P​p

这是频率稳定于概率的严格数学表述。

证明方法是引入指示变量。令

Xk​={1,0,​第 k 次试验中 A 发生第 k 次试验中 A 不发生​

则

nA​=k=1∑n​Xk​,Xˉn​=nnA​​

并且

E(Xk​)=p,D(Xk​)=p(1−p)=pq

于是

E(Xˉn​)=p,D(Xˉn​)=npq​

由切比雪夫不等式，

P(​nnA​​−p​≥ε)≤nε2pq​→0

故命题成立。

伯努利大数定律揭示了概率统计定义的合理性。频率“稳定于概率”并不意味着每次频率都等于概率，而是指频率与概率之间存在固定级别偏差的事件，其概率随着样本量增大而趋于 0。这种稳定称为依概率稳定。

5. 契比雪夫大数定律

设随机变量序列 X1​,X2​,… 相互独立，且具有相同的数学期望与方差：

E(Xk​)=μ,D(Xk​)=σ2<∞,k=1,2,…

则对任意 ε>0，

n→∞lim​P(​n1​k=1∑n​Xk​−μ​≥ε)=0

即

Xˉn​P​μ

证明仍基于切比雪夫不等式。由独立性可得

E(Xˉn​)=μ

以及

D(Xˉn​)=D(n1​k=1∑n​Xk​)=n21​k=1∑n​D(Xk​)=nσ2​

于是

P(∣Xˉn​−μ∣≥ε)≤nε2σ2​→0

契比雪夫大数定律说明：当样本量足够大时，样本均值几乎表现为一个常数，因此可以作为总体期望 μ 的近似。它给出了“以平均值估计期望”的理论基础。

6. 契比雪夫大数定律的拓展

弱大数定律并不局限于“独立同分布且同方差”的情形。

若 X1​,X2​,… 相互独立，但不要求具有相同的期望与方差，只要

E(Xk​)=μk​,D(Xk​)=σk2​≤σ2

则仍有

n1​k=1∑n​Xk​−n1​k=1∑n​μk​P​0

另一方面，独立性条件也可以放宽。若满足

n21​D(k=1∑n​Xk​)→0

则同样有

Xˉn​−E(Xˉn​)P​0

可见，大数定律的关键不是形式上的同分布，而是平均后的总波动必须相对于样本量足够小。

7. 辛钦大数定律

设随机变量序列 X1​,X2​,… 独立同分布，且数学期望存在，记

E(Xk​)=μ

则对任意 ε>0，

n→∞lim​P(​n1​k=1∑n​Xk​−μ​≥ε)=0

即

Xˉn​P​μ

辛钦大数定律比契比雪夫大数定律更强，因为它只要求期望存在，而不要求方差存在。伯努利大数定律是辛钦大数定律的特殊情形。

8. 强大数定律

若

Xˉn​−E(Xˉn​)a.e.​0

即

P({ω∈Ω:n→∞lim​[Xˉn​(ω)−E(Xˉn​)]=0})=1

则称 Xk​ 服从强大数定律。

强大数定律比弱大数定律更强，因为几乎处处收敛蕴含依概率收敛。它表明：除去一个零概率集合之外，样本均值对几乎每一条样本路径都真正收敛到期望。

9. 柯尔莫哥洛夫强大数定律与波雷尔强大数定律

柯尔莫哥洛夫强大数定律给出两个重要结论。

若 X1​,X2​,… 相互独立，期望与方差存在，并满足

n=1∑∞​n2D(Xn​)​<∞

则 Xn​ 服从强大数定律。

若 X1​,X2​,… 独立同分布，且期望存在，则 Xn​ 服从强大数定律。

对伯努利试验中的成功次数 nA​，由强大数定律可得

P({ω∈Ω:n→∞lim​nnA​​=p})=1

这称为波雷尔强大数定律。它表明频率不仅依概率趋于概率，而且几乎对每一条试验序列都真正收敛于概率。

四、中心极限定理

1. 中心极限定理的基本形式

大数定律处理的是平均值的稳定性，中心极限定理处理的是总和的波动形态。设

Sn​=k=1∑n​Xk​

若在适当条件下，标准化变量

D(Sn​)​Sn​−E(Sn​)​

依分布收敛于标准正态分布，则称 Xk​ 服从中心极限定理。

标准正态分布 N(0,1) 的密度函数与分布函数分别为

φ(x)=2π​1​e−x2/2−∞<x<+∞

以及

Φ(x)=∫−∞x​2π​1​e−t2/2dt−∞<x<+∞

因此中心极限定理可写为

D(Sn​)​Sn​−E(Sn​)​d​XX∼N(0,1)

中心极限定理的本质是：在大量独立或近独立的小影响共同作用下，总和经过中心化与标准化后呈现统一的极限分布——正态分布。

2. 独立同分布中心极限定理

设随机变量序列 X1​,X2​,… 独立同分布，且

E(Xk​)=μ,D(Xk​)=σ2>0

则对任意实数 x，

n→∞lim​P(σn​∑k=1n​Xk​−nμ​≤x)=Φ(x)

这就是独立同分布中心极限定理，又称 林德伯格-列维中心极限定理。

若记

Yn​=σn​∑k=1n​Xk​−nμ​

则有

Yn​d​N(0,1)

因此当 n 足够大时，可以作近似记号

Yn​≈N(0,1)

进一步地，由线性变换关系可得

k=1∑n​Xk​=σn​∗Yn​+nμ

从而

k=1∑n​Xk​≈N(nμ,nσ2)

这说明大样本下随机和近似服从正态分布。

3. 正态分布的封闭性

正态分布在概率论中占有中心地位，与其良好的代数封闭性密切相关。

若 X 与 Y 相互独立，且

X∼N(μ1​,σ12​),Y∼N(μ2​,σ22​)

则

X+Y∼N(μ1​+μ2​,σ12​+σ22​)

推广到独立正态变量 X1​,…,Xn​，若

Xi​∼N(μi​,σi2​),i=1,2,…,n

则

i=1∑n​Xi​∼N(i=1∑n​μi​,i=1∑n​σi2​)

此外，正态随机变量的线性变换仍服从正态分布；相互独立的正态随机变量的线性组合仍服从正态分布。这种封闭性使正态分布既是极限定理的极限分布，又是后续统计推断中的基础分布。

4. 德莫佛—拉普拉斯中心极限定理

设

Xn​∼B(n,p),0<p<1

则对任意实数 x，

n→∞lim​P(np(1−p)​Xn​−np​≤x)=Φ(x)

因此当 n 很大时，

Xn​≈N(np,np(1−p))

这就是德莫佛—拉普拉斯中心极限定理。它是独立同分布中心极限定理在伯努利试验中的具体形式，也是二项分布正态近似的理论基础。

5. 中心极限定理的意义

自然界中的许多随机现象可以看作许多彼此几乎没有依赖、且每个因素都不占支配地位的小随机因素共同作用的结果。若将这些小因素记为 X1​,X2​,…,Xn​，总体效应往往表现为它们的和

X1​+X2​+⋯+Xn​

中心极限定理表明，这类总和在适当标准化之后趋向正态分布。因此，大量现实中的总量型、误差型、聚合型随机变量都会服从或近似服从正态分布。

正态分布又称高斯分布。它最早出现在二项分布概率的近似计算与天文测量误差分析中。后来人们逐步发现，现实世界中的大量现象都具有近似正态性。中心极限定理之“中心”，指的正是正态分布在随机现象统计分析中的中心地位。

五、中心极限定理的典型应用

即

X>200

由德莫佛—拉普拉斯中心极限定理，

X≈N(np,np(1−p))

其中

np=10000×0.017=170

以及

np(1−p)=10000×0.017×0.983=167.11

故标准差近似为

167.11​≈12.93

于是

P(X>200)≈1−Φ(12.93200−170​)=1−Φ(2.32)≈0.01017

因此保险公司亏本的概率约为 0.01017，是一个较小的概率。

3. 标准正态随机数的近似生成

设 X1​,X2​,…,Xn​ 相互独立，且都服从 (0,1) 上的均匀分布，则

E(Xk​)=0.5,D(Xk​)=121​

由独立同分布中心极限定理，

Y=n/12​∑k=1n​Xk​−0.5n​≈N(0,1)

因此可以通过产生若干个独立均匀随机数并求和，再进行中心化与标准化，得到近似服从标准正态分布的随机数。实际计算中常取 n=12，于是

Y=k=1∑12​Xk​−6

这是利用中心极限定理进行随机模拟的经典方法之一。

六、小结

概率极限理论围绕三个层次展开。第一，随机变量序列的收敛性区分了几乎处处收敛、依概率收敛与依分布收敛，其中

Xn​a.e.​X⟹Xn​P​X⟹Xn​d​X

第二，大数定律研究平均值的稳定性。弱大数定律说明样本均值在概率意义下逼近期望，强大数定律说明这种逼近几乎对每条样本路径都成立。它们构成了以频率估计概率、以样本均值估计总体期望的理论基础。

第三，中心极限定理研究总和的波动分布。大量独立同分布随机变量之和，经中心化与标准化后趋向标准正态分布；二项分布因此可由正态分布近似；大量现实中的聚合型随机现象也因此表现出正态性。

大数定律解决“平均会不会稳定”的问题，中心极限定理解决“波动呈现何种形态”的问题。两者共同构成数理统计与大样本推断的核心基础。
