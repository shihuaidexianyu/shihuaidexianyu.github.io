概率论与数理统计

概率论与数理统计

Apr 15, 202615 min read

Summary

这篇笔记和你一起梳理机器学习里最常用的概率论工具：正态样本的均值分布，以及期望、方差、协方差的全部运算规则。读完之后，你应该能独立推导“为什么样本均值的方差会除以 n”，能准确判断期望的线性性何时成立、方差何时才能直接相加，并且不会被 E[X2] 和 (E[X])2 这类符号搞混。

正态分布中样本的均值分布

Intuition

想象你在工厂里测量零件长度。每一个零件的真实长度都围绕某个均值 μ 波动，波动幅度是 σ2。如果你随机抽取 n 个零件并计算它们的平均长度 ar X，这个平均值本身也是一个随机变量——它也会波动，但波动的范围一定比单个零件要小。这一节我们要回答：ar X 到底服从什么分布？它的期望和方差分别是多少？

背景设定

设我们有 n 个独立同分布（i.i.d.）的样本，每个都服从正态分布：

X1​,…,Xn​∼iidN(μ,σ2)

样本均值定义为

Xˉ=n1​i=1∑n​Xi​

我们要证明的核心结论是：

Xˉ∼N(μ,nσ2​)

Note

这个结论在统计学中无处不在。它告诉我们：** averaging（取平均）是一种降噪操作**——虽然每个样本都有噪声 σ2，但平均之后的噪声被压缩成了 σ2/n。样本越多，我们对均值的估计就越稳。

期望：为什么平均值的中心还是 μ

Intuition

直觉上，如果你把 n 个零件的长度加起来再除以 n，这个平均值既没有理由偏高，也没有理由偏低，它的中心当然还是真实均值 μ。

推导

这一步不需要正态分布的假设，只要期望存在即可。由期望的线性性：

E[Xˉ]=E[n1​i=1∑n​Xi​]=n1​i=1∑n​E[Xi​]

因为每个 Xi​ 的均值都是 μ，所以：

E[Xˉ]=n1​⋅nμ=μ

Example

假设零件真实长度是 10cm，你测了 3 个，读数可能是 9.8,10.2,10.0。平均值是 10.0。再换一组，可能是 10.1,9.9,10.0，平均值还是 10.0。长期看，所有可能样本的平均值的平均，一定收敛到 10cm。

方差：为什么平均值的波动变成了 σ2/n

Intuition

单个零件可能偏差很大（比如一个测出 12cm），但当你把多个零件平均后，偏高的样本和偏低的样本会互相抵消。所以平均值的波动范围一定比单个样本小，而且样本量 n 越大，抵消得越彻底。

推导

由于样本相互独立，常数 β=n1​ 提出来后要平方：

Var(Xˉ)=Var(n1​i=1∑n​Xi​)=n21​Var(i=1∑n​Xi​)

独立随机变量之和的方差等于方差之和：

Var(i=1∑n​Xi​)=i=1∑n​Var(Xi​)=nσ2

因此：

Var(Xˉ)=n21​⋅nσ2=nσ2​

Trap

很多同学在这里容易漏掉 n2。记住：βX 的方差是 β2Var(X)，不是 βVar(X)。因为方差是“平方的期望”，对尺度缩放的敏感度是平方级别的。

到这里，我们已经得到：

E[Xˉ]=μ,Var(Xˉ)=nσ2​

但这只说明了均值和方差，还没说明分布的形状一定是正态的。

为什么平均值仍然服从正态分布

Intuition

正态分布有一个非常漂亮的性质：几个正态随机变量的线性组合，结果仍然是正态随机变量。样本均值 χˉ​ 无非就是 X1​,…,Xn​ 各自乘上 n1​ 再相加，所以它当然还是正态的。

关键定理：

正态变量的线性组合仍然服从正态分布。

而 Xˉ 正是 X1​,…,Xn​ 的线性组合：

Xˉ=n1​X1​+n1​X2​+⋯+n1​Xn​

因为每个 Xi​ 都是正态分布，所以它们的这个线性组合也必然是正态分布。

结合刚才求出的期望 μ 和方差 nσ2​，就得到完整的分布结论：

Xˉ∼N(μ,nσ2​)

Example

假设某考试成绩服从 N(70,25)。如果你随机抽 1 个人，成绩可能在 60 到 80 之间大幅波动；但如果你抽 n=25 个人算平均分，这个平均分的分布就变成了 N(70,1)，基本稳定在 68 到 72 之间。这就是大样本估计的威力。

期望与方差的运算法则

Summary

下面这部分是“概率运算的速查手册”。我们会从定义出发，逐一推导期望和方差的核心性质。这些公式本身不复杂，但使用时的前提条件（比如是否要求独立）非常容易混淆，需要特别注意。

一、期望的运算法则

常数与常数倍

常数的期望就是它本身：

E[c]=c

常数倍可以提出期望：

E[aX]=aE[X]

更一般地，加上一个常数偏移：

E[aX+b]=aE[X]+b

Intuition

这很直观：如果你把所有测量值都放大 2 倍，平均数也会放大 2 倍；如果你给所有测量值都加上 3，平均数也会加上 3。

线性性（最关键）

对两个随机变量 X,Y：

E[X+Y]=E[X]+E[Y]

更一般地：

E[aX+bY+c]=aE[X]+bE[Y]+c

对于有限个随机变量 X1​,…,Xn​：

E[i=1∑n​Xi​]=i=1∑n​E[Xi​] E[i=1∑n​ai​Xi​]=i=1∑n​ai​E[Xi​]

Note

期望的线性性永远成立，不需要 X 和 Y 独立，甚至不需要它们不相关。 这是期望最强大的性质之一，也是很多同学在考场上容易遗忘的一点。

乘积的期望

乘积的期望一般不能拆：

E[XY]=E[X]E[Y]

只有当 X,Y 独立时，才有：

E[XY]=E[X]E[Y]

更一般地，若 X1​,…,Xn​ 相互独立：

E[i=1∏n​Xi​]=i=1∏n​E[Xi​]

Trap

线性性无条件成立，但乘积拆分必须有独立性。把两者搞混是初学者最常见的错误之一。

函数期望

若 g(X) 是随机变量 X 的函数，则函数的期望定义为：

离散情形：

E[g(X)]=x∑​g(x)p(x)

连续情形：

E[g(X)]=∫g(x)f(x)dx

Example

E[X2] 和 E[(X−μ)2] 都是函数期望的典型例子。前者是“平方的平均”，后者是“偏离均值的平方的平均”——后者其实就是方差。

二、方差的定义与展开公式

方差衡量的是随机变量偏离其期望的“平均平方距离”：

Var(X)=E[(X−E[X])2]

若记 μ=E[X]，也可写为：

Var(X)=E[(X−μ)2]

最常用的展开式

方差有一个非常实用的展开公式，在计算中几乎比定义更常用：

Var(X)=E[X2]−(E[X])2

推导

Var(X)​=E[(X−μ)2]=E[X2−2μX+μ2]=E[X2]−2μE[X]+μ2​

由于 μ=E[X]，所以：

Var(X)=E[X2]−μ2=E[X2]−(E[X])2

Intuition

这个公式在说：方差等于“平方的平均”减去“平均的平方”。如果 X 总是取同一个值（没有波动），那么 X2 的平均就等于平均的平方，方差为 0。波动越大，这两个量的差距就越大。

三、方差的运算法则

基本性质

常数没有波动：

Var(c)=0

随机变量平移不改变方差：

Var(X+c)=Var(X)

常数倍缩放使方差乘上该常数的平方：

Var(aX)=a2Var(X)

综合起来：

Var(aX+b)=a2Var(X)

Intuition

给所有数据加 3，数据的分散程度当然不变；但给所有数据乘 2，原本相距 1 的两个点现在相距 2，方差会放大到 4 倍。

两个随机变量之和

方差一般不能直接相加。通用公式是：

Var(X+Y)=Var(X)+Var(Y)+2Cov(X,Y)

同理：

Var(X−Y)=Var(X)+Var(Y)−2Cov(X,Y)

其中协方差定义为：

Cov(X,Y)=E[(X−E[X])(Y−E[Y])]

也可以展开为：

Cov(X,Y)=E[XY]−E[X]E[Y]

Intuition

协方差 Cov(X,Y) 刻画的是 X 和 Y 的“同向波动”程度。如果 X 偏高时 Y 也倾向于偏高，那么协方差为正，总方差会被放大；反之则被抵消。这就是为什么 X−Y 的方差里协方差项是负号——减法相当于引入反向波动。

对于一般的线性组合 aX+bY：

Var(aX+bY)=a2Var(X)+b2Var(Y)+2abCov(X,Y)

如果 X,Y 独立，则 Cov(X,Y)=0，上式简化为：

Var(aX+bY)=a2Var(X)+b2Var(Y)

同样，在独立条件下：

Var(X+Y)=Var(X)+Var(Y) Var(X−Y)=Var(X)+Var(Y)

Trap

注意：X−Y 在独立时方差也是相加，不是相减！因为减法只是改变了 Y 的方向，而方差对方向不敏感，只对波动幅度敏感。

推广到 n 个随机变量

若 X1​,…,Xn​ 相互独立，则：

Var(i=1∑n​Xi​)=i=1∑n​Var(Xi​)

以及：

Var(i=1∑n​ai​Xi​)=i=1∑n​ai2​Var(Xi​)

四、样本均值中最常用的公式

Intuition

这一节把前面所有的法则应用到样本均值 Xˉ 上。你会发现，样本均值的期望等于总体均值，样本均值的方差等于总体方差除以 n。这两个结论构成了统计推断的基石。

设样本均值为：

Xˉ=n1​i=1∑n​Xi​

若 X1​,…,Xn​ 同分布，且：

E[Xi​]=μ,Var(Xi​)=σ2

期望推导

由期望的线性性：

E[Xˉ]​=E[n1​i=1∑n​Xi​]=n1​i=1∑n​E[Xi​]=n1​⋅nμ=μ​

方差推导

若再假设 X1​,…,Xn​ 相互独立：

Var(Xˉ)​=Var(n1​i=1∑n​Xi​)=n21​Var(i=1∑n​Xi​)=n21​i=1∑n​Var(Xi​)=n21​⋅nσ2=nσ2​​

Note

这两个公式非常重要，务必牢记：

E[Xˉ]=μ,Var(Xˉ)=nσ2​

前者说明样本均值是总体均值的无偏估计；后者说明样本均值的波动随着样本量增加而以 1/n 的速度衰减。

五、几个容易混淆的点

Trap

这一节总结了概率论初学者最常踩的四个坑。每一个都值得反复看。

第一：期望的线性性总是成立

E[X+Y]=E[X]+E[Y]

始终成立，不需要独立性，甚至不需要不相关性。

第二：方差一般不能直接相加

通用公式是：

Var(X+Y)=Var(X)+Var(Y)+2Cov(X,Y)

只有在独立或至少不相关时，才有：

Var(X+Y)=Var(X)+Var(Y)

第三：E[X2] 一般不等于 (E[X])2

二者之差正是方差：

Var(X)=E[X2]−(E[X])2

只有当 X 是常数（方差为 0）时，两者才相等。

第四：E[XY] 一般不等于 E[X]E[Y]

只有在独立时才成立：

E[XY]=E[X]E[Y]

Example

设 X 是某地明天的降雨量，Y 是同一地明天的湿度。两者显然正相关：湿度高的时候降雨量也倾向于高。此时 E[XY] 会大于 E[X]E[Y]，因为高湿和高雨往往同时出现，乘积被“放大”了。
