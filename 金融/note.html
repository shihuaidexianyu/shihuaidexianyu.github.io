PE / PB / ROE

这三者最好不要孤立看：

  • ROE 决定公司 ” 赚钱能力/商业质量 ”
  • PB 主要在给 ” 净资产 ” 定价（尤其适合金融、重资产）
  • PE 主要在给 ” 盈利 ” 定价（适合盈利相对稳定的公司）

一句话：ROE 是根，PB/PE 是果。估值是否合理，取决于 ROE 是否撑得住。

1. 定义与直觉

1.1 ROE（净资产收益率）

  • 含义：公司用 ” 股东的钱 ” 一年能赚多少回报
  • 公式：ROE = 净利润 / 净资产
  • 直觉：
    • ROE 高且稳定 → 公司更 ” 值钱 “（可给溢价）
    • ROE 低或波动大 → 估值很难长期高

1.2 PB（市净率）

  • 含义：市场愿意用多少倍价格买公司账面净资产
  • 公式：PB = 市值 / 净资产 = 股价 / 每股净资产
  • 直觉：
    • PB=1：市场按 ” 账面净资产 ” 给价
    • PB>1：市场认为资产能 ” 更有效地赚钱 “（ROE 高、壁垒强、增长好）
    • PB<1：市场担心资产质量/坏账/盈利能力/周期风险

1.3 PE（市盈率）

  • 含义：市场愿意用多少倍价格买公司当期利润
  • 公式：PE = 市值 / 净利润 = 股价 / 每股收益
  • 直觉：
    • PE≈” 付出多少年利润买回公司 “（只是粗略直觉）
    • PE 高：增长预期高 / 确定性高 / 或短期利润被压低
    • PE 低：增长慢 / 风险高 / 或利润处于周期高点

2. 三者的关键关系：PB 与 ROE 的 ” 锚 ”

一个非常核心的经验：PB 本质上是 ROE 的函数（再叠加增长与风险）。

  • ROE 高且稳定 → PB 给得起溢价
  • ROE 低 → PB 很难长期高

2.1 粗略匹配（用于 ” 第一眼 ” 判断）

（不是铁律，但很好用）

  • ROE ~ 8%（一般）：PB 通常很难长期 > 1.5–2
  • ROE ~ 12–15%（不错）：PB 2–4 常见
  • ROE 20%+ 且稳定（很强）：PB 4–8 也可能被市场接受

逻辑：你愿意为 ” 能持续把净资产高效率变成利润 ” 的公司付更多倍数。

3. PE 如何和 ROE / 增长绑定

PE 是否 ” 贵 “，离不开两个问题：

  1. 利润是否可持续（有没有一次性利润、周期顶点利润、会计因素）
  2. 未来增长（增长越快、确定性越强，越能承受更高 PE）

3.1 粗略区间（只做筛查）

  • PE < 10：可能便宜，也可能是 ” 周期高点的假便宜 ” 或基本面有雷
  • PE 10–20：成熟稳态行业常见
  • PE 20–35：偏贵，需要增长或较强确定性
  • PE > 35–50+：明显昂贵，除非是高增长 + 壁垒强 + 利润质量硬

4. 一套实用判断流程（从 ” 质量 ” 到 ” 估值 “）

Step 1：先看 ROE（质量与定价权）

  • ROE 越高越好，但更重要的是稳定性
  • 追问：ROE 高是靠什么？
    • 靠品牌/渠道/技术壁垒（更可持续）
    • 还是靠加杠杆、涨价周期、会计处理（更脆）

Step 2：选对估值尺

  • 金融（银行/保险/券商）：优先 PB + ROE + 资产质量
  • 重资产（地产链、钢铁煤炭、航运等周期）：PB/周期位置/现金流更重要；PE 容易误导
  • 轻资产（消费、软件、服务）：PE 结合增长；PB 往往会很高（不必惊讶）

Step 3：看 PB/PE 是否被 ROE 和增长 ” 撑住 ”

  • 高 PB 必须有 高且稳 ROE 来解释
  • 高 PE 必须有 可持续增长 + 利润质量 来解释
  • 反过来：
    • 低 PB/低 PE 不一定便宜，可能是 ROE 要塌或风险很大

5. 最容易踩的坑（重点）

5.1 周期股 “PE 迷惑术 ”

  • 景气顶点：利润暴涨 → PE 看起来很低（你以为便宜，其实可能是顶部）
  • 景气底部：利润很差 → PE 看起来很高（你以为贵，其实可能是底部）
    ✅ 周期股更建议看：
  • PB + 资产质量 + 行业周期位置
  • 或 ” 归一化利润 ” 的 PE（用正常年份利润）

5.2 PB<1 不等于捡钱

PB<1 可能意味着：

  • 资产质量差（坏账、减值）
  • 盈利能力弱（ROE 常年低）
  • 行业长期衰退/政策风险

5.3 ROE 也会 ” 造 ”

ROE 高不一定真强，常见 ” 假强 ” 来源：

  • 高杠杆（金融/地产/部分制造扩张期）
  • 一次性收益（卖资产、政府补助）
  • 会计处理导致净资产偏小
    ✅ 看 ROE 时要追问：它是否可持续、是否靠杠杆硬抬？

6. 快速结论（可当作心智开关）

  • PB 高估的典型信号：PB 很高，但 ROE 撑不住（尤其 ROE 低或下滑）
  • PE 高估的典型信号：PE 很高，但增长兑现不确定、利润质量差、或周期在高位
  • ” 便宜 ” 的典型信号：估值不高 + ROE 稳定/可回升 + 风险可解释且可控

7. 一个小抄：如何一句话评价一只标的

  • ” 这公司 ROE 15% 且稳定，PB 2.5 倍不离谱；但如果增长放缓，PE 30 可能偏贵。”
  • ” 周期股 PE 低不代表便宜，我要看行业位置和归一化利润。”
  • “PB<1 我先查资产质量和 ROE，有没有长期赚不到钱的问题。“

波段 & 做 T

0. 核心概念一句话版

  • 波段：押 ” 方向 + 一段时间 “，吃一段趋势/区间移动的利润（几天～几周）。
  • 做 T：押 ” 波动 + 更短周期 “，围绕底仓做差价，目标是降低持仓成本/增厚收益（多为日内～数日节奏）。

1. 波段（Swing Trading）

1.1 定义

  • 在一个更大的结构里（趋势/箱体），尝试抓住价格从 ” 关键位置 ” 向另一个关键位置移动的一段。
  • 持仓周期常见：几天～几周（也可能更长，但核心是 ” 吃一段 ” 而不是长期持有）。

1.2 波段赚的是什么钱

  • 趋势钱：顺趋势低吸/回踩买入 → 趋势延续中卖出。
  • 区间钱：箱体下沿买入 → 箱体上沿卖出（震荡市常见）。

1.3 波段的关键要素

  • 结构：趋势 / 箱体区间是否清晰
  • 位置：支撑、压力、突破/回踩点
  • 节奏：放量/缩量、加速/衰竭、回踩确认
  • 风控：止损点位（结构被破坏就走）、仓位

1.4 优缺点

  • 优点：交易频率低、盯盘需求小、手续费影响相对低
  • 缺点：会经历回撤；买错容易被磨；需要判断趋势/区间有效性

2. 做 T（T Trading / Reduce Cost）

2.1 定义

  • 有底仓（中期看好或至少愿意持有），利用短周期波动做差价：
    • 目标一：降低底仓成本
    • 目标二：在不改变中期仓位的前提下增厚收益

2.2 做 T 赚的是什么钱

  • 赚的是来回震荡的波动，而不是趋势本身。
  • 所以做 T 天然更依赖 ” 箱体/震荡 ” 环境。

2.3 常见两种做法（围绕底仓）

  • 先卖后买（更常见）
    • 条件：你手里有底仓可以卖
    • 逻辑：冲高卖出部分 → 回落买回同样数量
    • 主要风险：卖飞（卖了继续涨，你买不回更低）
  • 先买后卖（更激进）
    • 逻辑：回落先加一部分 → 反弹后卖出同样数量（保持中期仓位不变）
    • 主要风险：加仓被套（买了继续跌，仓位被动变重）

注：A 股常见语境下 T+1，当天买的通常不能当天卖，所以 ” 日内做 T” 很多时候是用底仓完成当天卖出。

2.4 做 T 的 ” 有效区间 ”

做 T 要成立，通常需要：

  • 标的流动性好（点差小、成交活跃）
  • 波动够（有差价空间）
  • 价格在相对明确的区间内来回（震荡/箱体）
  • 你能执行纪律（到位就做，不到位不手痒）

2.5 做 T 的成本与门槛（非常关键）

  • 手续费 + 滑点 + 点差 是做 T 的 ” 隐形敌人 ”
  • 如果你抓的差价不够覆盖成本，做 T 就是 ” 忙着亏 ”
  • 做 T 次数越多，越要关注成本占比

3. 波段 Vs 做 T：对比速记表（文字版）

  • 时间尺度：
    • 波段：几天～几周
    • 做 T：日内～数日（围绕底仓的短波动）
  • 主要依赖：
    • 波段：结构（趋势/区间）+ 方向判断
    • 做 T：震荡波动 + 执行与成本控制
  • 主要风险：
    • 波段：趋势判断错、结构破坏导致止损
    • 做 T：卖飞/越 T 越亏、交易成本吞噬利润
  • 最适合的行情：
    • 波段：趋势明确或箱体边界清晰
    • 做 T：震荡市、箱体内反复来回

4. 什么时候更适合用哪个

4.1 更适合波段的环境

  • 单边趋势明显（上升/下降趋势）
  • 回踩有承接、突破后有确认
  • 你不想高频盯盘

4.2 更适合做 T 的环境

  • 明显箱体震荡（上沿压力、下沿支撑反复有效）
  • 日内波动足够，但又不至于直接单边飞走
  • 你有底仓，并愿意用纪律去优化成本

4.3 不太适合做 T 的环境（高风险）

  • 放量突破后的强单边（容易卖飞）
  • 跌破关键支撑后的弱单边（容易越补越亏）
  • 流动性差/点差大/成本高的标的

5. 常见 ” 失败模式 ” 清单（自检用）

  • 做 T 做成了 ” 追涨杀跌 “：看涨就加，看跌就砍，完全丢了区间逻辑
  • 没有底仓逻辑：本来不想拿，硬做 T 把自己做成长期套牢
  • 不算成本：差价太小，手续费把收益吃光
  • 没有规则：全靠感觉，越做越频繁，越频繁越容易犯错
  • 趋势行情硬做 T：强势股一冲就卖，结果一路踏空

6. 一个实用的组合思路（常见）

  • 波段负责方向与主利润：在关键位置建仓/减仓
  • 做 T 负责震荡期增厚：在箱体里小幅来回，降低成本
  • 当行情从震荡转为单边（突破/破位）：
    • 停止做 T，回归波段思路（避免卖飞/越补越套）
