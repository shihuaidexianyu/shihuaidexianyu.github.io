Lecture 5 随机变量的数字特征

随机变量的数字特征用于从若干方面概括其概率分布。数学期望描述平均水平，方差描述分散程度，偏度与峰度描述分布形态，分位数与中位数描述位置特征，协方差与相关系数描述多个随机变量之间的线性关系，矩则为这些数字特征提供了统一的表达框架。

引言

概率分布给出了随机变量取值的完整规律，但在许多理论分析与实际计算中，更常需要用少数几个数来概括其主要特征。数字特征的作用并不是替代分布本身，而是在保留核心信息的同时压缩描述复杂度。平均水平、离散程度、对称性、峰态以及变量之间的关联结构，都可以用相应的数字特征刻画。

数字特征之间存在清晰分工。数学期望回答“平均取值在何处”，方差回答“围绕平均值波动多大”，分位数回答“某一概率水平对应的阈值是多少”，协方差与相关系数回答“两个变量是否存在同向或反向的线性变化趋势”，矩则将这些概念纳入统一体系。

随机变量函数的期望与均值公式

设随机变量 X 取值为 xi​，对应概率为 pi​。若级数收敛，则函数 g(X) 的数学期望为

E[g(X)]=i∑​g(xi​)pi​

若 X 为连续型随机变量，概率密度为 f(x)，且积分存在，则

E[g(X)]=∫−∞+∞​g(x)f(x)dx

若 (X,Y) 为二维随机变量，联合密度为 f(x,y)，则

E[g(X,Y)]=∫−∞+∞​∫−∞+∞​g(x,y)f(x,y)dxdy

这些公式统称为均值公式。它们的意义在于：只要把需要考察的量写成随机变量的函数，就可以用同一套期望计算框架处理。后续的方差、协方差、矩、偏度、峰度，本质上都可以看成某种函数期望。数学期望本身就是一阶原点矩。

方差及其计算

设随机变量 X 的数学期望存在。若

E[(X−E(X))2]

存在，则称其为 X 的方差，记作 D(X) 或 Var(X)，即

D(X)=E[(X−E(X))2]

方差衡量的是随机变量取值偏离平均水平的平均平方程度，因此描述的是分散程度。由于平方会改变量纲，通常还引入标准差

σ=D(X)​

标准差与原变量量纲相同，便于解释与比较。

若 X 为离散型随机变量，P(X=xk​)=pk​，则

D(X)=k=1∑∞​(xk​−E(X))2pk​

若 X 为连续型随机变量，概率密度为 f(x)，则

D(X)=∫−∞+∞​(x−E(X))2f(x)dx

方差最常用的计算公式为

D(X)=E(X2)−[E(X)]2

其推导来自平方展开：

D(X)​=E[(X−E(X))2]=E[X2−2XE(X)+E(X)2]=E(X2)−2E(X)E(X)+E(X)2=E(X2)−[E(X)]2​

这个公式的作用在于把“围绕均值的平方偏差”转化为两个更容易计算的期望。它尤其适合常见分布与函数型随机变量的方差求解。

常见分布的方差

0-1 分布

设 X∼Bernoulli(p)，取值为 0,1，记 q=1−p。则

E(X)=p

又因为 X2=X，所以

E(X2)=p

因此

D(X)=E(X2)−[E(X)]2=p−p2=pq

0-1 分布的方差在 p=q=1/2 时达到最大值 1/4，说明最不确定的情形恰好发生在成功概率与失败概率相等时。

二项分布

设 X∼B(n,p)，记 q=1−p。由二项分布可以视作 n 个相互独立的 0-1 随机变量之和，也可由二阶矩直接计算。

先求

E(X)=np

再利用恒等式

X2=X(X−1)+X

得到

E(X2)=E[X(X−1)]+E(X)=n(n−1)p2+np

于是

D(X)​=E(X2)−[E(X)]2=n(n−1)p2+np−n2p2=np(1−p)=npq​

二项分布的方差与试验次数 n 成正比，也与单次成功概率的不确定性 pq 成正比。

泊松分布

设 X∼P(λ)。有

E(X)=λ

从二阶矩出发，

E(X2)=k=0∑∞​k2k!e−λλk​

将 k2 拆为 k(k−1)+k，得到

E(X2)=λ2+λ

于是

D(X)=E(X2)−[E(X)]2=λ2+λ−λ2=λ

泊松分布中，均值与方差相等。这是泊松分布最显著的数字特征之一。

几何分布

设 X 服从参数为 p 的几何分布，取值为 1,2,…，记 q=1−p，则

P(X=k)=qk−1p

有

E(X)=p1​

进一步可得

D(X)=p2q​

这说明当成功概率 p 很小时，等待首次成功所需的试验次数不仅均值较大，波动也较大。

均匀分布

设 X∼U(a,b)，则

f(x)=⎩⎨⎧​b−a1​0​a<x<b其他​

其数学期望为

E(X)=2a+b​

二阶矩为

E(X2)=b−a1​∫ab​x2dx=3a2+ab+b2​

于是

D(X)​=E(X2)−[E(X)]2=3a2+ab+b2​−4(a+b)2​=12(b−a)2​​

均匀分布的方差只取决于区间长度 b−a，与区间位置无关。

指数分布

设 X∼Exp(λ)，密度为

f(x)={λe−λx0​x≥0x<0​

其数学期望为

E(X)=λ1​

二阶矩为

E(X2)=∫0+∞​x2λe−λxdx=λ22​

因此

D(X)=λ22​−λ21​=λ21​

指数分布中，标准差与均值相等，都是 1/λ。

正态分布

设 X∼N(μ,σ2)，则

E(X)=μ

方差可由变量代换 u=(x−μ)/σ 计算，最终得到

D(X)=σ2

由

D(X)=E(X2)−[E(X)]2

还可反推出

E(X2)=μ2+σ2

正态分布的两个参数 μ,σ2 分别对应位置与尺度，是其最基本的数字特征。

Gamma 分布

设 X 服从参数为 α,β 的 Gamma 分布，密度为

f(x)=⎩⎨⎧​Γ(α)βα​xα−1e−βx0​x>0x≤0​

其数学期望为

E(X)=βα​

二阶矩为

E(X2)=β2α(α+1)​

于是

D(X)​=E(X2)−[E(X)]2=β2α(α+1)​−β2α2​=β2α​​

Gamma 分布的方差随着形状参数 α 增大而增大，随着率参数 β 增大而减小。

方差的性质与切比雪夫不等式

方差具有一组基础性质。

常数的方差为零：

D(C)=0

线性变换下，平移不改变方差，伸缩使方差按平方倍数变化：

D(aX)=a2D(X)

以及

D(aX+b)=a2D(X)

两个随机变量的和差满足

D(X±Y)=D(X)+D(Y)±2cov(X,Y)

若 X,Y 相互独立，则协方差为零，从而

D(X±Y)=D(X)+D(Y)

这一定理可推广到 n 个相互独立随机变量。若 X1​,…,Xn​ 相互独立，a1​,…,an​,b 为常数，则

D(i=1∑n​ai​Xi​+b)=i=1∑n​ai2​D(Xi​)

这说明独立随机扰动的总波动大小是各自方差按平方权重累加的结果。

方差还具有最小二乘意义。对任意常数 C，

D(X)≤E[(X−C)2]

且当且仅当 C=E(X) 时取等号。证明为

E[(X−C)2]−D(X)​=E[(X−C)2]−(E(X2)−[E(X)]2)=[E(X)−C]2≥0​

因此，在所有常数中，数学期望是使均方误差最小的那个常数。这一性质解释了平均值在估计论和回归分析中为何具有中心地位。

方差为零当且仅当随机变量几乎处处等于其数学期望，即

D(X)=0⟺P(X=E(X))=1

这意味着“没有波动”与“退化为常数”在概率意义下是同一回事。

切比雪夫不等式给出了偏离均值的概率上界。若 E(X) 与 D(X) 存在，则对任意 ε>0，

P(∣X−E(X)∣≥ε)≤ε2D(X)​

证明的核心是：在事件 {∣X−E(X)∣≥ε} 上，有

ε2(X−E(X))2​≥1

因此该事件的概率不超过对应期望，而该期望正好等于 D(X)/ε2。切比雪夫不等式的重要性在于，它不要求知道分布的具体形式，只需要均值和方差存在，就能控制偏离均值的概率。不过它给出的界通常较粗。

二项分布可以用方差可加性再次说明。若 X∼B(n,p)，可写成

X=X1​+⋯+Xn​

其中 Xi​ 相互独立，且每个 Xi​ 都服从参数为 p 的 0-1 分布。于是

D(X)=D(X1​)+⋯+D(Xn​)=npq

这个分解说明了“总计数的波动”来自独立单次试验波动的累积。

并非所有随机变量都具有数学期望和方差。即便数学期望与方差都存在，它们通常也不足以唯一确定分布。设随机变量 X 取值为 −1,0,1，概率分别为 0.1,0.8,0.1；随机变量 Y 取值为 −2,0,2，概率分别为 0.025,0.95,0.025。两者都满足

E(X)=E(Y)=0

以及

D(X)=D(Y)=0.2

但分布显然不同。这说明均值和方差只能刻画部分概率特性。

其他典型数字特征

偏度系数

若 E[(X−E(X))3] 存在，则称

α=[D(X)]3/2E[(X−E(X))3]​

为随机变量 X 的偏度系数。偏度刻画的是分布相对于数学期望的对称程度。若分布关于 E(X) 对称，则奇数阶中心矩为零，因此 α=0。正态分布满足

α=0

偏度为正通常意味着右尾较长，偏度为负通常意味着左尾较长，但偏度本身反映的是整体三阶不对称性，而不是某一个局部区间的形状。

峰度系数

若 E[(X−E(X))4] 存在，则称

γ=[D(X)]2E[(X−E(X))4]​

为随机变量 X 的峰度系数。峰度用于刻画在期望与方差固定时，分布峰部与尾部的综合形态。对连续型随机变量，峰度与密度曲线的陡峭程度有关。正态分布满足

γ=3

需要区分的是：峰度不是单纯的“尖不尖”，而是四阶中心矩与二阶中心矩平方的比值，因此同时受到峰部和尾部的影响。

变异系数

若 X 是非负随机变量，且 E(X)>0、D(X) 存在，则称

ν=E(X)D(X)​​

为 X 的变异系数。它也是分散程度指标，但与方差不同，变异系数是无量纲量，适合比较不同单位、不同量纲或不同平均水平下的离散程度。平均值相同的两个变量可以直接比较标准差；平均值不同且单位相同的两个变量，更适合比较变异系数。

分位数与中位数

设随机变量 X，实数 a 与概率水平 0<p<1。若满足

P(X≤a)≥p≥P(X<a)

则称 a 为 X 的 p 分位数，记作 xp​。其中 0.5 分位数称为中位数，记作 med(X)。

分位数与期望一样也描述位置，但中位数有两个显著特点：它总是存在，而且对极端值不敏感；相应地，它不具备期望那样的可加性。一般情形下分位数一定存在，但可能不唯一。为了唯一化，常定义

xp​=supa:P(X≤a)<p

或等价地定义为

xp​=infa:P(X≤a)≥p

对连续型随机变量，分位数通常唯一。若密度为 f(x)，则下侧 p 分位数 xp​ 满足

P(X≤xp​)=p

也即

∫−∞xp​​f(x)dx=p

若上侧分位数记为 xα​，定义为

P(X>xα​)=α

也即

∫xα​+∞​f(x)dx=α

连续型情形下，给定 p 或 α 求分位数，本质上是分布函数求逆；反过来，给定 x 求 F(x)，则是顺向积分。常见分布的分位数通常可查表得到。

设 X 为连续型随机变量，xα​ 为上侧分位数，则常用关系有：若 p+α=1，则 xp​=xα​；区间 (x1−α/2​,xα/2​) 的概率为 1−α；其补集概率为 α。当密度是偶函数时，分布关于 0 对称，满足

x1−α​=−xα​

并且

P(∣X∣>xα/2​)=α

这些关系在置信区间、显著性检验和标准正态表的使用中都非常常见。

协方差与相关系数

协方差

对二维随机变量 (X,Y)，若

E[(X−E(X))(Y−E(Y))]

存在，则称其为 (X,Y) 的协方差，记为

cov(X,Y)=E[(X−E(X))(Y−E(Y))]

协方差刻画两个变量之间的线性同变关系。若二者倾向于同时大于各自均值或同时小于各自均值，则协方差倾向为正；若一个大于均值而另一个小于均值，则协方差倾向为负；若正负抵消，则可能为零。协方差反映的是线性关系，而不是一般意义上的全部依赖关系。

协方差最常用的计算公式为

cov(X,Y)=E(XY)−E(X)E(Y)

它来自展开

(X−E(X))(Y−E(Y))

并对各项取期望。这个公式往往比直接按定义计算更方便。

协方差满足以下性质。交换性：

cov(X,Y)=cov(Y,X)

自协方差等于方差：

cov(X,X)=D(X)

线性变换下：

cov(a1​X+b1​,a2​Y+b2​)=a1​a2​cov(X,Y)

分配性：

cov(X+Y,Z)=cov(X,Z)+cov(Y,Z)

和差方差公式：

D(X±Y)=D(X)+D(Y)±2cov(X,Y)

若 X,Y 相互独立，则

cov(X,Y)=0

此外，由 Schwarz 不等式可得

[cov(X,Y)]2≤D(X)D(Y)

也即

∣cov(X,Y)∣≤D(X)​D(Y)​

最后一条给出了协方差绝对值的上界。只有当两个变量之间存在严格线性关系时，协方差才会达到这个界。

随机变量的标准化

若随机变量 X 的期望与方差都存在，且 D(X)=0，则称

X∗=D(X)​X−E(X)​

为 X 的标准化随机变量。标准化的结果是把原变量平移到均值为零，再缩放到方差为一，因此

E(X∗)=0

以及

D(X∗)=1

标准化的作用是消除位置与尺度差异，为比较不同变量、定义相关系数以及构造统计量提供统一尺度。

相关系数

若 cov(X,Y) 存在，且 D(X)>0、D(Y)>0，则称

ρXY​=D(X)​D(Y)​cov(X,Y)​

为 X,Y 的相关系数。利用标准化变量可以写成

ρXY​=E(X∗Y∗)

相关系数是无量纲量，它把协方差除以各自标准差，从而消除了单位影响。协方差受测量单位影响，而相关系数不受影响，因此更适合比较不同变量之间线性关系的强弱。若

ρXY​=0

则称 X,Y 不相关。

相关系数满足

∣ρXY​∣≤1

并且

∣ρXY​∣=1

当且仅当存在常数 a,b，其中 a=0，使得

P(Y=aX+b)=1

这意味着相关系数绝对值达到一时，两个变量之间存在几乎处处成立的严格线性关系。证明思路有两种。一种从协方差上界出发，若 P(Y=aX+b)=1，则 D(Y−aX)=0，再代入方差公式得到等号成立；另一种从标准化变量出发，构造 Z=X∗±Y∗，若 ∣ρXY​∣=1，则可使 D(Z)=0，从而得到线性关系。

不相关与独立

不相关与独立不是同一概念。由定义可知

ρXY​=0

当且仅当

cov(X,Y)=0

又等价于

E(XY)=E(X)E(Y)

在方差存在时，也等价于

D(X±Y)=D(X)+D(Y)

若 X,Y 相互独立，则一定不相关，因为独立推出 E(XY)=E(X)E(Y)。但逆命题一般不成立，不相关只能说明没有线性关系，不能说明没有更高阶或非线性的依赖结构。若 (X,Y) 服从二维正态分布，则不相关与独立等价，这也是二维正态分布的重要特征之一。

计算例子

在“2 红、2 黄、1 蓝”共 5 个球中任取 3 个，记红球数为 X，黄球数为 Y。由联合分布可算得

E(X)=E(Y)=1.2

以及

D(X)=D(Y)=0.36

又有

E(XY)=1.2

因此

cov(X,Y)=1.2−1.2×1.2=−0.24

从而

ρXY​=0.36​0.36​−0.24​=−32​

负相关表明在固定总抽取数下，红球数增大往往伴随黄球数减小。

对二维正态分布

(X,Y)∼N((μ1​μ2​​),(σ12​ρσ1​σ2​​ρσ1​σ2​σ22​​))

可直接计算得到

cov(X,Y)=ρσ1​σ2​

以及

ρXY​=ρ

因此二维正态分布中的参数 ρ 本身就是相关系数，这使五个参数 μ1​,μ2​,σ12​,σ22​,ρ 的含义完全明确。

设 (X,Y) 在单位圆盘上服从均匀分布，即

f(x,y)=⎩⎨⎧​π1​0​x2+y2≤1其他​

由对称性可得

E(X)=E(Y)=0

并且

cov(X,Y)=E(XY)=0

所以 X,Y 不相关。但其边缘密度为

fX​(x)=π21−x2​​∣x∣<1

以及

fY​(y)=π21−y2​​∣y∣<1

显然联合密度不等于边缘密度乘积，因此 X,Y 并不独立。这是“不相关不推出独立”的典型反例。

协方差矩阵

对二维随机变量 (X,Y)，由四个二阶中心矩排成的矩阵

Σ(X,Y)=(D(X)cov(Y,X)​cov(X,Y)D(Y)​)

称为其协方差矩阵。对二维情形，它同时记录了各自方差和相互协方差。

对 n 维随机变量 (X1​,…,Xn​)，记

σij​=cov(Xi​,Xj​)

若这些协方差都存在，则称

Σ=​σ11​σ21​⋮σn1​​σ12​σ22​⋮σn2​​⋯⋯⋱⋯​σ1n​σ2n​⋮σnn​​​

为该随机向量的协方差矩阵。它具有三个基本性质。第一，Σ 对称，因为 σij​=σji​。第二，对角元就是各分量方差：

σii​=D(Xi​)

第三，满足

σij2​≤σii​σjj​

这实际上是协方差 Schwarz 不等式在矩阵分量上的表达。

协方差矩阵还是非负定矩阵。任取向量

α=(α1​,α2​,…,αn​)T

并记 μi​=E(Xi​)，则

αTΣα=E​(i=1∑n​αi​(Xi​−μi​))2​≥0

因为平方的期望不可能为负。这一性质保证了协方差矩阵可以作为多维波动结构的合法描述。

把随机向量写成列向量

X=​X1​X2​⋮Xn​​​

其期望向量为

E(X)=​E(X1​)E(X2​)⋮E(Xn​)​​

则协方差矩阵可以紧凑地写成

Σ(X)=E[(X−E(X))(X−E(X))T]

这个向量形式把分量间的所有二阶中心矩统一写在一个表达式中，是多元统计中最常用的记号。

对 n 维正态随机变量，

(X1​,X2​,…,Xn​)∼N(μ,Σ)

其密度函数可写为

f(x)=(2π)n/2∣Σ∣1/21​exp(−21​(x−μ)TΣ−1(x−μ))

其中 μ 是期望向量，Σ 是协方差矩阵。由此可见，多元正态分布的几何形状完全由均值向量与协方差矩阵决定。

若随机向量在线性变换下满足 Y=AX，其中 A 为矩阵，则

Σ(Y)=AΣ(X)AT

推导为

Σ(Y)​=E[(Y−E(Y))(Y−E(Y))T]=E[(AX−E(AX))(AX−E(AX))T]=E[A(X−E(X))(X−E(X))TAT]=AE[(X−E(X))(X−E(X))T]AT=AΣ(X)AT​

这条公式是多元线性模型、主成分分析和误差传播公式的基础。

随机变量的矩

矩是把数字特征统一起来的工具。设 k 为正整数，则

E(Xk)

称为 X 的 k 阶原点矩，

E(∣X∣k)

称为 X 的 k 阶绝对原点矩，

E[(X−E(X))k]

称为 X 的 k 阶中心矩。其中特别地，

E(X)

是一阶原点矩，而

E[(X−E(X))2]=D(X)

是二阶中心矩。

对二维随机变量 (X,Y)，还可以定义混合矩。原点混合矩为

E(XkYl)

中心混合矩为

E[(X−E(X))k(Y−E(Y))l]

其中

E(XY)

是二阶混合原点矩，而

E[(X−E(X))(Y−E(Y))]

是二阶混合中心矩，也就是协方差。相应地，相关系数可视为标准化变量 X∗,Y∗ 的二阶混合矩：

ρXY​=E(X∗Y∗)

因此，协方差与相关系数都可以纳入矩的统一框架。

偏度系数与峰度系数也都可以解释为标准化中心矩。偏度是三阶标准化中心矩，峰度是四阶标准化中心矩。由此可见，位置、离散、形态和相关结构这些看似不同的数字特征，实质上都可以理解为不同阶、不同形式的矩。

从统一角度看，可以作如下对应。数学期望是一阶原点矩，方差是二阶中心矩，协方差是二阶混合中心矩，若 E(X)=0，则中心矩与相应原点矩相同；相关系数是标准化变量的二阶混合矩；二维协方差矩阵由四个二阶中心矩组成，n 维协方差矩阵则由 n2 个二阶中心矩组成。这样，矩不仅提供了统一记号，也揭示了各种数字特征之间的内在联系。

小结

随机变量的数字特征形成了从一维到多维、从位置到形态、从单变量到多变量的完整描述体系。数学期望与分位数刻画位置，方差与变异系数刻画分散，偏度与峰度刻画形态，协方差与相关系数刻画线性关系，矩则提供统一表达。理解这些概念时，需要始终区分三个边界：第一，均值与方差不能代替完整分布；第二，不相关不等于独立；第三，协方差反映的是带量纲的线性同变，而相关系数反映的是无量纲的线性强度。
