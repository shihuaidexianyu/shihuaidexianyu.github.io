五、时间片轮转与 CPU 利用率

  1. [原题] 对某系统进行检测后发现，在阻塞 I/O 之前，平均每个进程运行时间为 T。一次进程切换需要的时间为 S，这里 S 实际上就是开销。对于采用时间片长度为 Q 的轮转调度，请给出以下各种情况的 CPU 利用率的计算公式：

    • (a) Q = ∞
    • (b) Q > T
    • (c) S < Q < T
    • (d) Q = S
    • (e) Q 趋近于 0

  6. [扩展] 当 Q > T 时，进程通常在时间片用完前发生什么？

  7. [扩展] 当 S < Q < T 时，一个周期内有效运行时间和切换开销分别是多少？

  8. [扩展] 当 Q = S 时，CPU 有多少比例时间花在上下文切换上？

  9. [扩展] 当 Q 趋近于 0 时，CPU 利用率为什么趋近于 0？
