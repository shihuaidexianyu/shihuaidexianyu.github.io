滞后回归：基本概念

滞后回归（lagged regression）是一类把 ” 过去若干时刻的信息 ” 作为预测变量的回归分析。它的核心思想是：对于当前时刻 t 的行为或状态，不只考察当前输入，而是系统地考察 t−1,t−2,…,t−i 等更早时刻的事件分别产生多大影响。这样得到的一组回归系数，按滞后步数 i 排列后，就形成了一条 ” 时间影响曲线 ” 或 ” 历史权重谱 “。这条曲线刻画的是系统对过去信息的依赖范围与依赖形状，因此常被用来分析时间积分范围、记忆时程和多时间尺度加工。

在这篇论文【Hybrid neural–cognitive models reveal how memory shapes human reward learning.】中，滞后回归有两个用途。第一，它被用于行为层面，分析过去动作和奖励怎样影响未来选择；第二，它被用于模型内部表征层面，分析当前奖励状态 st(r)​ 的各个主成分究竟携带了多远的奖励历史信息。前者回答 ” 人和模型怎样利用过去做选择 “，后者回答 ” 模型内部状态如何表征过去 “。

一、行为层面的滞后回归

1. 研究目的

论文在方法部分明确指出，这一分析的目标是考察过去任务事件如何影响参与者之后的行为，并比较不同认知模型在这种时间依赖上的表现。作者采用的是一种相对模型无关的做法：不用先假定某个 RL 参数，而是直接用逻辑回归量化过去动作与奖励对当前选择的影响，并比较这些影响随滞后步数变化的时间轮廓。

2. 因变量

对于四臂 bandit 任务中的每一个 bandit，作者各建了一个回归模型。某一模型的因变量是该 bandit 在整个 trial 序列中的被选时间序列

a1:n​,

其中 (n=150)。若某 trial 选择了当前这个 bandit，则记为 1；否则记为 0。因为四个 bandit 在任务结构上没有本质差别，而且四个回归模型的结果几乎一致，作者最后把四个模型的结果取平均后可视化。

3. 自变量构造

作者为每个 bandit 的选择序列构造了两组滞后自变量。第一组是 bandit-reward regressors，表示 ” 过去在选择当前 bandit 时获得的奖励 “；第二组是 other-reward regressors，表示 ” 过去在选择其他 bandit 时获得的奖励 “。方法部分把它们分别写成：

ri:n+i​×a1:n​,ri:n+i​×(1−a1:n​).

直观地说，前一组变量回答 ” 若过去选的是当前目标 bandit，它之前拿到多少奖励 “；后一组变量回答 ” 若过去选的是其他 bandit，它们之前拿到多少奖励 “。作者向前最多追溯 20 个 trial，因此总共放入 40 个回归量：20 个当前 bandit 的历史奖励量，20 个其他 bandit 的历史奖励量。

xt,iBR​=rt−i​1[at−i​=b],xt,iOR​=rt−i​1[at−i​=b].

是指示函数。这个写法不是论文逐字公式，而是对其方法描述的标准化整理。它表达的正是论文方法部分所说的那两类回归量。

4. 回归系数的解释

这一回归的关键不在单个 trial 的预测精度，而在整组系数的形状。若 βiBR​ 在某个小滞后上较大，说明 ” 当前目标 bandit 在最近几次获得的奖励 ” 会显著提高之后再次选择它的概率；若 βiOR​ 为负或与 βiBR​ 方向相反，则说明 ” 其他 bandit 过去获得的奖励 ” 会把选择概率从当前 bandit 拉走。于是，系数随 i 的衰减速度可以被理解为系统整合奖励历史的时间范围。

5. 论文中的发现

主文把这一分析画在 Fig. 4g，并将其称作 integration timescale。结果显示，人类行为表现出一种较浅、并且非单调的历史依赖；Memory-ANN 能较好复现这种模式，而更简单的模型做不到。作者因此认为，仅靠经典 RL 那种把过去压缩成少数标量决策变量的方式，不足以解释人类选择中这种分布在多步滞后上的复杂时间依赖。

二、奖励状态解码中的滞后回归

1. 研究目的

在解释 reward module 时，论文不仅关心 ” 状态 st(r)​ 会怎样调节当前 reward→Q 的映射 “，还进一步关心 ” 这个状态本身到底记住了什么历史 “。为此，作者先从训练好的 Memory-ANN 中提取 trial-by-trial 的隐藏状态 s(r)，对其做 PCA，然后用滞后回归去检测每个主成分与过去奖励之间的关系。主文指出，这一步是为了 assess the contents of the representations，也就是评估状态表征的内容。

2. 分析对象

这里的因变量与自变量不再是 ” 是否选择某个 bandit”，而是 ” 当前 reward state 的某个主成分 ” 和 ” 过去某个滞后上的奖励 “。论文方法部分写道：对每一个 delay (i)，作者都计算一个单独的回归模型，用当前状态 st(r)​ 的某个 PC 去预测 trial t−i 上观察到的奖励；主文正文则把它表述为 ” 对每一个 delay (i)，用 reward rt−i​ 去预测每个 state PC”。两种说法在形式上方向相反，但它们共同指向的是同一件事：为每个滞后 i 估计当前状态 PC 与过去奖励之间的耦合强度。论文真正关心的是系数随滞后变化的谱，而不是单次回归的预测方向。

因此，在教材式表述中，可以把这类分析概念化为

PCk,t​αk,0​i=1∑L​wk,i​,rt−i​εt​,

或者等价地理解为：对每个固定的 (i)，估计当前 PCk,t​ 与 rt−i​ 的线性关系强度 (w_{k,i})。论文的重点正是这些 wk,i​ 随 i 的分布形状。

3. 系数谱的含义

若某个主成分 PCk​ 只在 i=1,2,3 等极短滞后上有较大系数，而对更早的奖励几乎不敏感，则说明这一维状态主要编码最近几次奖励，可理解为一种偏短时程的奖励记忆。反过来，若某个主成分在较长滞后范围内都保持明显系数，则说明这一维状态携带了更长期的奖励背景信息，反映的可能不是单个 recent outcome，而是 ” 环境总体是否富足 "" 最近是在变好还是变坏 ” 等更慢的统计量。

4. 论文中的发现

主文明确报告，有一部分 PC 只对最近少数几次奖励显示较大的回归权重，这与 ” 这些 PC 跟踪最近个体结果、而对更早事件不敏感 ” 的解释一致；另一些 PC 则对几乎整个奖励历史都敏感，可能为环境总体奖励丰富度以及其变化方向提供基线。作者据此得出结论：(s_t^{(r)}) 不是一个单一时间尺度的缓存，而是一种多时间尺度的分布式奖励历史表征。这一发现与前面 probing 分析中看到的 ” 不同 PC 对奖励呈现不同斜率、不同持续时间的积分更新 ” 是相互一致的。

三、为什么滞后回归在这篇论文里特别关键

这篇论文的核心争论并不是 ” 某个固定 RL 更新公式参数取多少更好 “，而是 ” 过去经验是否被压缩成少数标量，还是被保存在更丰富的内部记忆状态中 “。滞后回归在这里的价值，就在于它把这个抽象问题转成了一个可直接观察的时间结构问题：系统究竟对过去多远的信息敏感，敏感的形状是单调衰减还是多峰、是短时还是长时、是单一尺度还是多尺度。

在行为层面，滞后回归显示人类选择的历史依赖是浅的、非单调的，Memory-ANN 能复现，而经典 RL 类模型不能。
在隐藏状态层面，滞后回归进一步显示 reward state 的不同主成分分别覆盖不同长度的奖励历史，说明模型内部确实形成了多时间尺度记忆。

因此，滞后回归在本文中既是一个行为诊断工具，也是一个表征解码工具。前者说明 ” 人类行为对过去的依赖形状 “；后者说明 ” 模型内部状态如何承载这种依赖 “。二者合在一起，构成了论文论证链中的关键一环：Memory-ANN 之所以优于经典 RL，不只是因为它更灵活，而是因为它拥有能够跨多时间尺度表征历史的内部状态。

四、可直接记忆的总结

滞后回归是一种把过去多个时刻的信息显式展开为预测变量的回归分析，其核心产物不是单个系数，而是一整条 ” 随滞后变化的权重曲线 “。在这篇论文中，它有两种用途。其一，用逻辑回归分析过去动作和奖励对未来选择的影响，从而刻画人和模型的行为整合时间尺度；其二，用当前 reward state 的主成分与过去奖励之间的滞后关联，刻画隐藏状态所编码的奖励历史范围。前者说明选择如何依赖过去，后者说明内部记忆如何表征过去。两者共同支持了论文的核心结论：奖励学习并不是单纯由固定规则更新单值 (Q)，而是由多时间尺度的内部记忆状态持续调节。
