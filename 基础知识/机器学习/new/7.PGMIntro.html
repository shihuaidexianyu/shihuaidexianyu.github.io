概率图模型

概率图模型以图结构表示高维随机变量之间的联合分布，通过图中的节点、边与局部因子，把“概率分布的表示”“条件独立关系的编码”“推断与学习的计算过程”统一到同一框架之中。它既是对概率分布结构的压缩表达，也是对复杂概率计算可行化的重要方法。

一、概率分布的基本规则与图模型的出发点

概率图模型建立在若干基本概率规则之上。对随机变量的联合分布、边缘分布与条件分布，常用的基本关系包括边缘化规则、乘法公式、链式法则与贝叶斯公式。

离散情形下，边缘化规则可写为

p(x1​)=x2​∑​p(x1​,x2​)

连续情形下，相应地写为

p(x1​)=∫p(x1​,x2​),dx2​

乘法公式写为

p(x1​,x2​)=p(x1​∣x2​)p(x2​)=p(x2​∣x1​)p(x1​)

对任意多个随机变量，链式法则写为

p(x1​,x2​,…,xp​)=i=1∏p​p(xi​∣x1​,…,xi−1​)

贝叶斯公式写为

p(x1​∣x2​)=p(x2​)p(x2​∣x1​)p(x1​)​

这些公式在理论上始终成立，但当变量维度很高时，直接表示和计算联合分布会迅速变得困难。若不引入额外结构，联合分布需要描述全部变量组合的概率，参数规模与计算复杂度都会随维度急剧增长。概率图模型的核心思想正是在这一点上展开：通过对变量间依赖关系作结构化约束，把原本庞大的联合分布分解成若干局部项，并把这些局部项之间的关系用图表示出来。

这种约束通常体现为条件独立性。若变量集合被划分为 A、B、C 三部分，且在给定 XC​ 的条件下，XA​ 与 XB​ 相互独立，则记为

XA​⊥XB​∣XC​

条件独立性并不是对联合分布信息的简单删减，而是对依赖结构的明确说明。概率图模型的作用就在于用图把这种结构编码出来，并据此得到更紧凑的表示、更高效的推断以及更清晰的建模假设。

二、概率图模型的基本组成

概率图模型通常包含三个相互关联的层面：表示、推断与学习。

表示问题讨论如何用图结构编码概率分布及条件独立关系。常见的表示形式包括有向图模型、无向图模型以及更一般的因子图。若变量为离散型或依赖方向本身具有明确意义，常使用有向图；若更关心对称依赖关系与整体能量形式，常使用无向图；若需要对分解结构作更细致的显式表示，则可使用因子图。

推断问题讨论如何由模型计算感兴趣的概率量，例如边缘概率、条件概率以及最大后验估计。高维分布的关键困难并不在于写下联合分布，而在于如何对某些变量求和、积分或最大化。概率图模型正是借助局部分解，把这些原本难以处理的操作转化为结构化计算。

学习问题讨论如何从数据中估计模型参数与图结构。参数学习在完备数据情形下较为直接，而在存在隐变量时，通常需要使用迭代方法，例如 EM 算法。若连图结构本身也未知，则进一步构成结构学习问题。

三、有向图模型：贝叶斯网络

1. 基本定义与因子分解

贝叶斯网络是一个有向无环图。图中的每个节点对应一个随机变量，每条有向边表示一种直接依赖关系。若记节点 i 的父节点集合为 pa(i)，则整个联合分布可以分解为

p(x1​,x2​,…,xp​)=i=1∏p​p(xi​∣xpa(i)​)

这一分解说明，图不仅表达“谁依赖谁”，也同时给出了联合分布的构造方式。只要图是有向无环图，就总能按照某个拓扑顺序写出这样的分解。

贝叶斯网络中的每个局部条件分布都是联合分布的一部分。通过把高维联合分布拆为若干局部条件分布，模型得以显著降低表示复杂度。真正起作用的关键不只是分解本身，而是分解背后的局部独立性假设：每个变量在给定其父节点后，与其非后代变量条件独立。

2. 三种基本局部结构

贝叶斯网络中的条件独立关系可先从三个最基本的局部结构理解。

链式结构

graph LR
    A["A"] --> B["B"]
    B --> C["C"]

其联合分布为

p(A,B,C)=p(A)p(B∣A)p(C∣B)

在给定中间节点 B 后，A 与 C 条件独立，即

A⊥C∣B

这意味着 B 截断了从 A 到 C 的信息传递。若 B 未被观测，A 与 C 一般并不独立；若 B 已知，A 对 C 的额外解释能力消失。

分叉结构

graph LR
    B["B"] --> A["A"]
    B --> C["C"]

其联合分布为

p(A,B,C)=p(B)p(A∣B)p(C∣B)

在给定共同原因 B 后，两个结果变量相互条件独立，即

A⊥C∣B

这表示由共同原因引起的相关性，在原因已知后会被解释掉。该结构与链式结构在条件独立性质上相同，区别只在于依赖关系的因果方向不同。

汇聚结构

graph LR
    A["A"] --> B["B"]
    C["C"] --> B["B"]

其联合分布为

p(A,B,C)=p(A)p(C)p(B∣A,C)

在未给定 B 时，A 与 C 彼此独立，即

A⊥C

但一旦对 B 进行条件化，A 与 C 一般不再独立，即通常有

A⊥C∣B

这是贝叶斯网络中最重要的特殊结构。两个本来互不相关的原因变量，在已知共同结果后会产生统计相关。若进一步观测到 B 的后代，这条路径同样会被“打开”。因此，汇聚结构与链式、分叉结构在条件独立行为上根本不同。

3. D-分离与全局条件独立

局部结构说明了最基本的独立关系，但实际模型通常远比三个节点复杂。贝叶斯网络中的全局条件独立由 D-分离刻画。

对图中的一条路径，若其中存在一个非汇聚节点落在条件集合 C 中，则该路径被阻断；若其中存在一个汇聚节点，而该节点及其任何后代都不在 C 中，则该路径也被阻断。只有当路径上所有非汇聚节点都不在 C 中，且所有汇聚节点本身或其某个后代在 C 中时，该路径才是激活的。

若从集合 A 到集合 B 的所有路径都被 C 阻断，则称 A 与 B 在给定 C 下 D-分离，并由此推出

XA​⊥XB​∣XC​

D-分离把局部结构推广到整个有向图，使得“从图上读出条件独立关系”成为可能。它是贝叶斯网络表示能力的核心，也是推断算法能够只关注局部相关部分的重要依据。

4. 马尔可夫毯

贝叶斯网络中，单个节点的条件分布不需要依赖全图所有变量。若记除 xi​ 之外的所有变量为 x−i​，则有

p(xi​∣x−i​)=p(xi​∣xMB(i)​)

其中，MB(i) 表示节点 i 的马尔可夫毯，它由三部分组成：该节点的父节点、子节点以及这些子节点的其他父节点。马尔可夫毯之外的变量，在给定马尔可夫毯后不再为 xi​ 提供额外信息。

从因子分解角度看，xi​ 的局部条件分布只与包含 xi​ 的那些因子有关，因此有

p(xi​∣x−i​)∝p(xi​∣xpa(i)​)j∈ch(i)∏​p(xj​∣xpa(j)​)

其中 ch(i) 表示 i 的子节点集合。这个公式说明，对单个变量做局部更新时，只需考虑与之直接相关的局部项，而无须扫描整个联合分布。

5. 典型有向图模型

贝叶斯网络是一类统一框架，许多经典模型都可视为其特殊情形。

朴素贝叶斯模型假设在给定类别变量 y 后，各个观测特征条件独立，其联合分布写为

p(y,x1​,…,xp​)=p(y)i=1∏p​p(xi​∣y)

其核心假设是“特征之间的依赖主要由类别变量解释”。这一假设虽然强，但在分类任务中往往能以较低的模型复杂度获得有效结果。

高斯混合模型引入离散隐变量 z 表示样本属于哪个成分。若共有 K 个混合成分，则

p(x)=k=1∑K​πk​N(x∣μk​,Σk​)

其中 πk​ 为第 k 个成分的混合权重，μk​ 与 Σk​ 分别为该成分的均值和协方差矩阵。隐变量 z 的引入把“复杂分布”分解成“简单分布的加权组合”。

马尔可夫链刻画时间序列中相邻时刻的局部依赖，满足一阶马尔可夫性质时，有

p(x1:T​)=p(x1​)t=2∏T​p(xt​∣xt−1​)

隐马尔可夫模型在此基础上再引入观测层。记隐状态为 z1:T​，观测为 x1:T​，则

p(z1:T​,x1:T​)=p(z1​)t=2∏T​p(zt​∣zt−1​)t=1∏T​p(xt​∣zt​)

这类模型把“系统状态随时间演化”与“状态如何生成观测”分开表示，因此特别适合处理时序数据中的隐结构问题。

线性动态系统、卡尔曼滤波与粒子滤波也都可视为动态贝叶斯网络的具体形式。差异主要在于状态转移和观测模型是线性还是非线性、分布是高斯还是非高斯。高斯过程则可以理解为以连续索引集合为定义域的无限维高斯分布，其本质仍然是对随机变量集合联合分布的结构化刻画。

四、无向图模型：马尔可夫网络

1. 基本定义

马尔可夫网络又称马尔可夫随机场，用无向图表示随机变量之间的对称依赖关系。与有向图不同，它不强调“谁生成谁”或“依赖方向”，而强调“哪些变量在结构上彼此相邻”。边的存在表示直接相互作用，边的缺失则与某种条件独立结构相关。

无向图中的条件独立关系通常从三个层面刻画。

全局马尔可夫性质指：若节点集合 C 在图上把 A 与 B 分隔开，则有

XA​⊥XB​∣XC​

局部马尔可夫性质指：对任一节点 i，在给定其邻居集合 N(i) 后，该节点与其余非邻居节点条件独立，即

Xi​⊥XV∖(i∪N(i))​∣XN(i)​

成对马尔可夫性质指：若节点 i 与节点 j 之间没有边，则在给定其余所有节点后，有

Xi​⊥Xj​∣XV∖i,j​

在分布满足严格正性条件时，这三种表述是等价的。它们共同说明：无向图中的独立关系来自图上的分隔结构，而不是来自有向路径上的传递与阻断规则。

2. 团、势函数与因子分解

无向图要真正构成概率模型，还需要给出联合分布的因子分解。为此引入团的概念。团是图中一组两两相连的节点集合；若该集合已经不能再加入更多节点而保持两两相连，则称为最大团。

设图中的团集合为 C，则联合分布可写为

p(x)=Z1​c∈C∏​ϕc​(xc​)

其中，ϕc​(xc​) 是定义在团上的势函数，要求取正值；Z 是配分函数，用于保证概率归一化。离散情形下，

Z=x∈X∑​c∈C∏​ϕc​(xc​)

连续情形下，上式中的求和应改为对全空间的积分。

势函数并不一定本身就是概率分布。它只需要刻画某一局部配置的相对“相容程度”或“偏好强度”。多个势函数相乘之后，经由 Z 归一化，才得到合法的联合分布。正因为势函数不受局部归一约束，无向图模型在描述对称相互作用时具有更大的表达灵活性。

3. Gibbs 分布与能量形式

若把势函数写成指数形式

ϕc​(xc​)=exp(−Ec​(xc​))

则整个分布可写为

p(x)=Z1​exp(−c∈C∑​Ec​(xc​))

其中 Ec​(xc​) 称为局部能量函数。局部能量越低，对应的配置越容易出现；局部能量越高，对应的配置越不受偏好。这种写法称为 Gibbs 分布或玻尔兹曼分布形式。

这种表达与指数族分布在形式上高度一致，也使无向图模型能够自然连接到统计物理中的能量最小化思想与机器学习中的正则化、最大熵建模等观点。对正分布而言，Hammersley-Clifford 定理表明：满足图上的马尔可夫性质与能够按图的团分解为势函数乘积，这两件事是等价的。

五、有向图与无向图之间的转换

1. 道德图

在很多情况下，需要把有向图转换为无向图，以便应用无向图上的统一推断框架。贝叶斯网络转化为无向图时，首先把每个节点的父节点两两相连，然后去掉所有边的方向。得到的无向图称为道德图。

这一操作的关键在于补上汇聚结构中父节点之间缺失的边。对结构

graph LR
    A["A"] --> B["B"]
    C["C"] --> B["B"]

若直接去掉箭头，会丢失由共同子节点诱发的依赖关系。因此需要先把 A 与 C 相连，再忽略边方向，得到一个三角形结构。所谓“道德化”，正是指让共同指向同一子节点的父节点“成婚”。

道德图的作用不是保留原有向图中全部精细语义，而是为后续的无向图因子分解与精确推断提供一个可操作的结构基础。

2. 因子图

因子图是一种更细粒度的表示方式。它把变量节点与因子节点显式地区分开来，用二部图表示联合分布的乘积分解。若

p(x)∝s∏​fs​(xs​)

则因子图中每个变量对应一个变量节点，每个局部因子 fs​ 对应一个因子节点，因子节点与其所依赖的变量节点相连。

因子图的优点在于：它直接表达“哪个局部函数依赖哪些变量”，因此比单纯的有向图或无向图更接近实际计算结构。消息传递算法在因子图上具有特别清晰的表达。因子图并不必然无环，但它能够把原模型中的局部因式分解关系完整保留下来，因此在推断与实现上都非常重要。

六、推断问题

1. 推断的目标

推断的核心任务是在给定模型后计算感兴趣的概率量。常见目标包括边缘概率、条件概率以及最大后验估计。边缘概率关心某个变量或变量子集本身的分布；条件概率关心在给定观测后未观测变量的后验分布；最大后验估计则关心最可能的隐变量配置或参数取值。

若记观测证据为 E，则最大后验问题可写为

x^=argxmax​p(x∣E)

这类问题在序列标注、图像分割、聚类与结构预测中都非常常见。

2. 精确推断与近似推断

精确推断方法要求得到严格正确的概率结果。典型方法包括变量消除、信念传播以及 Junction Tree。它们的共同思想是利用图结构把原本在高维空间中的求和、积分或最大化变成局部计算，但计算代价仍然会受到图复杂度，特别是树宽的显著影响。

近似推断方法在计算成本过高或图中存在复杂环路时使用。常见方法包括有环图上的 Loopy Belief Propagation、重要性采样、马尔可夫链蒙特卡罗方法以及变分推断。它们通过随机采样、分布近似或迭代消息传递来换取可计算性，但精度、收敛性与结果解释方式各不相同。

七、变量消除

变量消除是最基本的精确推断方法。它的思想是：只要某个变量在最终目标中不需要保留，就可通过局部求和或积分把该变量“消去”，并把中间结果保存为新的局部因子。

以链式结构 a→b→c→d 为例，若要求 p(d)，则从联合分布出发有

p(d)=a,b,c∑​p(a,b,c,d)

利用链式分解可写为

p(d)=a,b,c∑​p(a)p(b∣a)p(c∣b)p(d∣c)

进一步按消除顺序嵌套整理，可得

p(d)=c∑​p(d∣c)b∑​p(c∣b)a∑​p(b∣a)p(a)

这个过程表明，变量消除并不是对整个联合分布一次性求和，而是把高维求和重写为一系列低维局部计算。每次消去一个变量，都会产生一个新的中间因子，供后续计算继续使用。

变量消除的效率高度依赖于消除顺序。不同顺序会导致中间因子的维数差异很大，从而显著影响时间与空间复杂度。寻找最优消除顺序一般是困难问题，其本质与图的树宽控制有关。

八、信念传播

1. 基本思想

变量消除虽然能重用部分中间计算，但若对多个查询重复执行，仍可能产生大量重复工作。信念传播进一步把局部计算组织成沿边传递的消息，使得树形结构上的边缘概率计算可以高效复用。

在成对势函数形式的无向树上，若节点势为 ϕi​(xi​)，边势为 ϕij​(xi​,xj​)，则从节点 j 传向节点 i 的消息定义为

mj→i​(xi​)=xj​∑​ϕj​(xj​)ϕij​(xi​,xj​)k∈N(j)∖i∏​mk→j​(xj​)

其中 N(j) 表示节点 j 的邻居集合。该式表示：节点 j 把自己收到的所有其他邻居的信息先整合，再通过与 i 相连的边势投射给 i。

当所有相关消息都已收集后，节点 i 的边缘信念可写为

bi​(xi​)∝ϕi​(xi​)j∈N(i)∏​mj→i​(xi​)

在树结构上，若消息从叶节点向内收集，再从根向外分发，则每条边只需在两个方向上传递一次消息，就可得到所有节点的精确边缘分布。

2. 与变量消除的关系

信念传播并不是与变量消除无关的另一套算法，而是对变量消除中“局部中间结果可复用”这一思想的系统化表达。消息本质上就是某段子图对边界变量形成的局部摘要。变量消除是按某个查询一步步局部求和；信念传播则把这些局部结果沿图边保存和复用，因此在多查询场景下更高效。

3. 适用条件与局限

标准信念传播在树图上给出精确结果。若图中存在环，则直接使用同样的消息更新得到的是 Loopy Belief Propagation。此时算法不一定收敛，即使收敛也不一定得到精确边缘分布，但在许多实际问题中仍可能表现良好。图中环路越复杂、依赖越强，近似误差与收敛问题通常越突出。

九、Max-Product 算法

当推断目标不是求边缘分布，而是求最可能的变量配置时，可将信念传播中的求和替换为最大化，得到 Max-Product 算法。对边 j→i，其消息写为

mj→i​(xi​)=xj​max​ϕj​(xj​)ϕij​(xi​,xj​)k∈N(j)∖i∏​mk→j​(xj​)

该算法沿图传递的不是“所有可能配置的总权重”，而是“在给定边界变量取值时，内部子图所能达到的最大权重”。在树结构上，它可用于高效求解 MAP 配置。若在序列模型中使用这一思想，就得到经典的 Viterbi 算法。

实际求解最大后验配置时，除了保留最大值本身，通常还需要记录使最大值达到的自变量取值，即回溯指针。仅有最大值而没有对应的回溯信息，无法在最后恢复出完整的最优配置。

在数值实现中，常把势函数取对数，将乘积转化为求和，从而得到 Max-Sum 形式。这可以减轻下溢风险，并使优化过程更接近动态规划的标准写法。

十、概率图模型的整体理解

概率图模型的统一性体现在三个方面。第一，它把高维联合分布转化为图上的局部分解，使原本不可处理的概率对象获得结构化表达。第二，它把条件独立关系显式编码在图中，使“哪些信息相关、哪些信息可屏蔽”成为可读、可证明的结构性质。第三，它把推断与学习建立在局部计算之上，使边缘化、后验计算、最大后验估计以及参数学习都能够围绕相同的图结构展开。

有向图模型适合表达生成顺序、因果方向与逐层条件化结构；无向图模型适合表达对称相互作用、整体兼容性与能量约束；因子图则进一步把局部因子的计算结构显式化。无论采用哪种表示形式，其核心目标都不是图本身，而是利用图来控制高维概率问题的复杂度，并在表示、推断与学习之间建立统一的数学联系。
